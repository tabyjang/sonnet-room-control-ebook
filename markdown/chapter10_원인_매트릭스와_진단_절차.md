# Chapter 10. 원인 매트릭스와 진단 절차

---

## 한눈에 보기 (핵심 5줄)

1. 같은 증상이라도 원인이 전원·배선·통신·부품 중 어느 계층인지 먼저 분류해야 한다.
2. 원인 매트릭스는 증상과 원인의 교차점에서 어느 장치를 먼저 볼지 우선순위를 알려준다.
3. 제어기·구동기·온도조절기 각각을 분리 진단하면 교체 비용을 최소화할 수 있다.
4. 지역난방은 압력·유량·밸브 계통이 추가되므로 1차측/2차측 구분 진단이 필수다.
5. 단계별 복구 절차를 순서대로 따르면 실수 없이 정상 상태로 복원할 수 있다.

---

## 왜 중요한가? — 생활 속 예시

**예시 1 — "세 장치를 다 교체했는데 여전히 고장이다"**

원인 계층을 분류하지 않고 부품부터 교체하면 헛돈을 씁니다.
매트릭스로 원인 계층을 먼저 좁히면 꼭 필요한 부품만 교체할 수 있습니다.

**예시 2 — "지역난방인데 제어기 문제라고 교체했지만 여전히 차갑다"**

지역난방 공급 압력이 낮아서 생긴 문제였는데, 세대 내 장치만 점검했습니다.
1차측(열공급)과 2차측(세대 제어)을 구분하지 않으면 이런 오진이 생깁니다.

## 10.1 전원·배선·통신 원인 분류

### 원인 계층 구조

```
┌──────────────────────────────────────────────────────────┐
│              고장 원인 계층 (위에서부터 순서대로 확인)    │
├──────────────────────────────────────────────────────────┤
│                                                          │
│  Layer 1. 전원 계층                                      │
│  ┌───────────────────────────────────────────────┐       │
│  │ 차단기 트립 / 전원 미인가 / 전압 불안정        │       │
│  └───────────────────────────────────────────────┘       │
│                   ↓ 전원 정상이면                        │
│  Layer 2. 배선 계층                                      │
│  ┌───────────────────────────────────────────────┐       │
│  │ 단선 / 단락 / 접촉 불량 / 오결선 / 노이즈     │       │
│  └───────────────────────────────────────────────┘       │
│                   ↓ 배선 정상이면                        │
│  Layer 3. 통신·신호 계층                                 │
│  ┌───────────────────────────────────────────────┐       │
│  │ 신호선 불량 / 보일러 연동 불량 / 앱 연동 오류  │       │
│  └───────────────────────────────────────────────┘       │
│                   ↓ 신호 정상이면                        │
│  Layer 4. 부품 계층                                      │
│  ┌───────────────────────────────────────────────┐       │
│  │ 온도조절기 불량 / 구동기 불량 / 제어기 불량   │       │
│  └───────────────────────────────────────────────┘       │
│                   ↓ 부품 정상이면                        │
│  Layer 5. 설비 계층                                      │
│  ┌───────────────────────────────────────────────┐       │
│  │ 보일러 불량 / HIU 불량 / 배관 막힘 / 밸런싱   │       │
│  └───────────────────────────────────────────────┘       │
└──────────────────────────────────────────────────────────┘
```

### 계층별 점검 항목표

| 계층 | 점검 항목 | 측정 도구 | 정상 기준 |
|------|-----------|-----------|-----------|
| **전원** | 차단기 상태 | 육안 | 차단기 ON 상태 |
| **전원** | 제어기 입력 전압 | 멀티미터 AC | 210~230V |
| **전원** | 구동기 출력 전압 | 멀티미터 AC | 제품 정격 전압 ±10% |
| **배선** | 각 전선 도통 | 멀티미터 저항 | 수 Ω 이하 |
| **배선** | 단락 여부 | 멀티미터 저항 | L-N 간 ∞ |
| **배선** | 접촉 저항 | 멀티미터 저항 | 단자당 1Ω 이하 |
| **신호** | 온도조절기 신호선 | 멀티미터 저항 | 도통 정상 |
| **신호** | BLR 출력 | 멀티미터 저항/전압 | 난방 요청 시 도통 |
| **부품** | 구동기 코일 저항 | 멀티미터 저항 | 50~500Ω (제품별) |
| **부품** | 온도조절기 접점 | 멀티미터 저항 | 요청 시 0Ω / 해제 시 ∞ |
| **설비** | 분배기 유량계 | 육안 | 해당 루프 유량 표시 |
| **설비** | 보일러 공급 온도 | 온도계 또는 보일러 표시 | 설계 온도 범위 |

## 10.2 제어기·구동기·온도조절기 불량 분리 진단

### 장치별 불량 분리 진단 매트릭스

```
┌──────────────────────────────────────────────────────────────┐
│         장치 불량 분리 진단 매트릭스                          │
├────────────┬─────────────┬─────────────┬─────────────────────┤
│ 확인 항목  │ 온도조절기  │ 제어기      │ 구동기              │
│            │ 불량 시     │ 불량 시     │ 불량 시             │
├────────────┼─────────────┼─────────────┼─────────────────────┤
│ 제어기 LED │ 해당 존 LED │ LED 자체    │ LED 정상            │
│ 상태       │ 미점등      │ 이상        │ (제어기는 신호 출력)│
├────────────┼─────────────┼─────────────┼─────────────────────┤
│ 구동기     │ 구동기 단자 │ 구동기 단자 │ 구동기 단자         │
│ 단자 전압  │ 전압 없음   │ 전압 없음   │ 전압 정상           │
│            │             │             │ (구동기만 미반응)   │
├────────────┼─────────────┼─────────────┼─────────────────────┤
│ 신호선     │ 신호선      │ 신호선      │ 신호선 정상         │
│ 도통       │ 도통 정상   │ 도통 정상   │                     │
│            │ (제어기 LED │             │                     │
│            │  안 켜짐)   │             │                     │
├────────────┼─────────────┼─────────────┼─────────────────────┤
│ 다른 채널  │ 다른 채널   │ 다른 채널도 │ 다른 구동기는       │
│ 동작       │ 정상        │ 동일 이상   │ 정상                │
└────────────┴─────────────┴─────────────┴─────────────────────┘
```

### 온도조절기 불량 단독 진단

```
온도조절기 불량 여부 판별 절차
          │
          ▼
  ① 해당 존 신호선을 제어기에서 분리
          │
          ▼
  ② 신호선 양단을 멀티미터로 단락(0Ω) 연결
     (온도조절기가 난방 요청 상태 시뮬레이션)
          │
          ▼
  ③ 제어기 해당 존 LED 점등 확인
     └─ LED 켜짐 → 온도조절기 불량 (또는 신호선 불량)
     └─ LED 꺼짐 → 제어기 해당 채널 불량
```

### 구동기 불량 단독 진단

```
구동기 불량 여부 판별 절차
          │
          ▼
  ① 구동기 단자 전압 측정
     └─ 0V → 제어기 출력 또는 배선 불량
     └─ 정상 전압 → ②로
          │
          ▼
  ② 구동기 코일 저항 측정 (전원 OFF 후)
     └─ ∞ (무한대) → 코일 단선 → 교체
     └─ 0Ω → 코일 단락 → 교체
     └─ 정상 저항 → ③으로
          │
          ▼
  ③ 구동기 수동 동작 테스트
     └─ 정상 전압 인가 후 2~5분 내 동작 없음
        → 구동기 기계적 불량 → 교체
```

### 제어기 불량 단독 진단

```
제어기 불량 여부 판별 절차
          │
          ▼
  ① 전원 입력 전압 측정 (정상: AC 220V)
     └─ 0V → 배선 불량 (제어기 자체는 정상일 수 있음)
     └─ 정상 → ②로
          │
          ▼
  ② 모든 채널 LED 상태 확인
     └─ 특정 채널만 이상 → 해당 채널 불량
     └─ 전체 이상 → 제어기 전체 불량
          │
          ▼
  ③ 정상 채널에 불량 존 신호선 교차 연결
     └─ 정상 채널에서도 동작 안 함 → 신호선/온도조절기 불량
     └─ 정상 채널에서 동작함 → 원래 채널 불량 → 제어기 교체
```

## 10.3 지역난방 계통 (압력·유량·밸브) 연계 진단

### 지역난방 연계 진단 구조

```
┌──────────────────────────────────────────────────────────┐
│  지역난방 각방제어 진단 계층                              │
│                                                          │
│  [1차측: 열공급 계통] ← 이 구간은 열공급사/관리사무소    │
│  ┌─────────────────────────────────────────┐            │
│  │ 열공급소 공급 압력 / 온도 / 단지 밸브   │            │
│  └─────────────────────────────────────────┘            │
│                      ↓                                   │
│  [HIU: 세대 열교환기]                                    │
│  ┌─────────────────────────────────────────┐            │
│  │ 스케일 / 순환펌프 / 유량조절밸브        │            │
│  └─────────────────────────────────────────┘            │
│                      ↓                                   │
│  [2차측: 세대 각방제어] ← 이 전자책 적용 범위            │
│  ┌─────────────────────────────────────────┐            │
│  │ 제어기 / 구동기 / 온도조절기 / 분배기   │            │
│  └─────────────────────────────────────────┘            │
└──────────────────────────────────────────────────────────┘
```

### 지역난방 압력 진단표

| 증상 | 확인 항목 | 정상 기준 | 이상 시 조치 |
|------|-----------|-----------|-------------|
| 세대 전체 미온 | HIU 2차측 공급 압력 | 1.0~2.0 bar | 관리사무소 1차측 압력 확인 |
| 순환 느림·소음 | HIU 순환펌프 동작음 | 정상 작동음 | HIU 전문 A/S |
| 압력 급등 경보 | 팽창탱크 압력 | 0.5~1.5 bar | 팽창탱크 점검 의뢰 |
| 온수 공급 불량 | 1차측 공급 온도 | 60°C 이상 | 관리사무소 문의 |

### 지역난방 유량 진단표

| 증상 | 확인 항목 | 정상 기준 | 이상 시 조치 |
|------|-----------|-----------|-------------|
| 특정 루프 유량 0 | 분배기 유량계 | 설계 유량 범위 내 표시 | 해당 구동기 동작·결선 확인 |
| 루프 간 유량 불균형 | 유량계 눈금 비교 | 루프별 편차 20% 이내 | 밸런싱 밸브 재조정 |
| 전체 유량 낮음 | 분배기 전체 유량계 | 설계 유량의 80% 이상 | HIU 유량조절밸브 확인 |
| 특정 루프 유량 과다 | 유량계 눈금 | 다른 루프 대비 2배 이상 | 해당 루프 밸런싱 밸브 조임 |

### 지역난방 밸브 계통 진단표

```
밸브 계통 진단 흐름도

  [세대 난방 전체 불량]
          │
          ▼
  HIU 공급측 차단 밸브 열림 확인
     └─ 닫힘 → 밸브 열기 (관리사무소 확인 후)
     └─ 열림 → ↓
          │
          ▼
  분배기 공급 밸브 열림 확인
     └─ 닫힘 → 밸브 열기
     └─ 열림 → ↓
          │
          ▼
  분배기 각 루프 밸런싱 밸브 확인
     └─ 전체 잠김 → 밸브 열기
     └─ 일부 잠김 → 해당 루프 열기
          │
          ▼
  구동기 전체 수동 테스트
     └─ 전체 닫힘 유지 → 제어기·신호 계층 점검
```

## 10.4 단계별 복구 절차와 확인 항목

### 복구 절차 전체 흐름

```
┌──────────────────────────────────────────────────────────┐
│  이상 발생 → 복구 표준 절차                              │
└──────────────────────────────────────────────────────────┘
          │
          ▼
  STEP 1. 안전 확보
  ┌──────────────────────────────────────┐
  │ • 차단기 OFF                         │
  │ • 검전기로 무전압 확인               │
  │ • 배관 이상 시 분배기 밸브 잠금      │
  └──────────────────────────────────────┘
          │
          ▼
  STEP 2. 원인 계층 분류
  ┌──────────────────────────────────────┐
  │ • Layer 1~5 순서로 점검              │
  │ • 매트릭스에서 원인 범위 좁히기      │
  └──────────────────────────────────────┘
          │
          ▼
  STEP 3. 원인 부품/배선 특정
  ┌──────────────────────────────────────┐
  │ • 분리 진단으로 불량 장치 확정       │
  │ • 교체 부품 준비                     │
  └──────────────────────────────────────┘
          │
          ▼
  STEP 4. 조치 실행
  ┌──────────────────────────────────────┐
  │ • 결선 교정 또는 부품 교체           │
  │ • 한 번에 하나씩만 수정             │
  └──────────────────────────────────────┘
          │
          ▼
  STEP 5. 복구 확인 시험
  ┌──────────────────────────────────────┐
  │ • 차단기 ON → 기초 동작 확인         │
  │ • 존별 동작 시험 (Chapter 8 참조)    │
  │ • 보일러 연동 확인                   │
  └──────────────────────────────────────┘
          │
          ▼
  STEP 6. 정상 복구 확인 후 사용자 인계
```

### 복구 단계별 확인 항목표

| 단계 | 확인 항목 | 기준 |
|------|-----------|------|
| 1. 안전 확보 | 차단기 OFF 상태 | 검전기 미표시 |
| 2. 원인 분류 | Layer 1부터 순서 점검 | 각 계층 측정값 정상 기준 충족 |
| 3. 원인 특정 | 불량 장치 1개로 좁히기 | 교차 테스트로 최종 확정 |
| 4. 조치 실행 | 단일 수정 후 시험 반복 | 수정 후 재시험 필수 |
| 5. 복구 시험 | 존별 동작·보일러 연동 | 모든 존 정상 동작 |
| 6. 사용자 인계 | 원인·조치 내용 설명 | 사용자 이해 및 서명 |

### 복구 판정 기준표

```
┌───────────────────────────────────────────────────────┐
│  복구 완료 판정 기준                                   │
├──────────────────────────┬────────────────────────────┤
│  항목                    │  기준                      │
├──────────────────────────┼────────────────────────────┤
│  전원 LED                │  정상 점등                 │
│  존별 밸브 동작          │  전 존 열림·닫힘 정상      │
│  온도 응답               │  설정온도 방향으로 변화    │
│  보일러 연동             │  기동·정지 정상            │
│  누수                    │  없음                      │
│  소음                    │  비정상 소음 없음          │
│  온도 표시               │  안정적 표시               │
├──────────────────────────┼────────────────────────────┤
│  전 항목 충족            │  → 복구 완료, 사용자 인계  │
│  미충족 항목 있음        │  → STEP 2부터 재진단       │
└──────────────────────────┴────────────────────────────┘
```

## 통합 원인 매트릭스 (전체 요약)


## 초보자 Q&A

**Q1. 멀티미터로 어떤 항목을 측정해야 하나요?**
A. 전압(AC 220V 확인), 저항(코일·신호선 도통), 이 두 가지만 알면 대부분의 진단이 가능합니다. 멀티미터 기본 사용법을 한 번 익혀두면 큰 도움이 됩니다.

**Q2. 교체 전 원래 부품을 보관해야 하나요?**
A. 네, 교체 후 1~2주는 보관하세요. 교체 후에도 증상이 계속되면 다른 원인일 수 있으므로, 이전 부품의 재점검이 필요할 수 있습니다.

**Q3. 제어기와 구동기 제조사가 달라도 함께 쓸 수 있나요?**
A. 동일한 전선 방식(2선식/3선식)이고 전압 규격이 맞으면 사용 가능합니다. 단, 제조사 혼용 시 A/S 경계가 불분명해질 수 있습니다. 가능하면 동일 제조사 시스템으로 구성을 권장합니다.

**Q4. 복구 후 얼마나 지나야 정상이라고 확인할 수 있나요?**
A. 복구 직후 존별 동작 시험으로 즉시 확인 가능합니다. 단, 온도 안정화는 바닥 열용량 때문에 1~2시간 운전 후 판단하세요.

**Q5. 보일러 연동 신호선 규격을 모를 때 어떻게 하나요?**
A. 보일러 매뉴얼에서 "외부 실내온도조절기", "외부 입력", "CH 입력" 단자를 찾으세요. 무전압 건접점 입력이 가장 일반적입니다. 모를 경우 보일러 제조사 A/S 센터에 연락하세요.
