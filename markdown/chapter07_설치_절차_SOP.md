# Chapter 7. 설치 절차 SOP

---

## 한눈에 보기 (핵심 5줄)

1. 설치는 반드시 **전원 차단 → 기기 설치 → 배선 → 결선 → 점검 → 시운전** 순서로 진행한다.
2. 순서를 바꾸면 감전·오결선·기기 손상이 발생하므로 절대 건너뛰지 않는다.
3. 각 장치(제어기·구동기·온도조절기)의 설치 위치·높이·방향 기준을 지켜야 정확한 제어가 된다.
4. 전기 작업 전 검전기로 무전압 확인, 배관 작업 전 누수 확인은 매 단계 필수 절차다.
5. 설치 완료 후 검수 기준을 모두 통과해야 Chapter 8 시운전 단계로 넘어간다.

---

## 왜 중요한가? — 생활 속 예시

**예시 1 — "구동기를 먼저 달고 나중에 배선하려니 공간이 너무 좁다"**

분배기 위에 구동기를 모두 장착하고 나면 전선 연결 공간이 좁아집니다.
SOP 순서대로 배선 경로를 먼저 계획하고, 구동기 장착과 배선을 동시에 진행해야 합니다.

**예시 2 — "온도조절기를 벽에 달았는데 수평이 안 맞아 보기 싫다"**

수평계 없이 눈으로만 맞추면 비뚤어지기 쉽습니다.
설치 기준(수평계·높이·위치)을 지키면 보기 좋고 정확한 측정이 가능합니다.

## 7.1 공정 순서 (기기 설치 → 배선 → 결선 점검)

### 전체 설치 공정 흐름도

```
┌─────────────────────────────────────────────────────┐
│              각방제어 설치 공정 순서                  │
└─────────────────────────────────────────────────────┘
         │
         ▼
  [STEP 0] 안전 확인
   • 분전반 차단기 OFF
   • 검전기로 무전압 확인
   • 작업 구역 정리
         │
         ▼
  [STEP 1] 제어기 설치
   • 분배기 근처 벽면 위치 선정
   • 브래킷 또는 DIN 레일 고정
   • 전원선 경로 확인 (미결선)
         │
         ▼
  [STEP 2] 구동기 장착
   • 분배기 기존 수동 핸들 제거
   • 구동기 어댑터 체결 (규격 확인)
   • 구동기 본체 장착 · 방향 확인
         │
         ▼
  [STEP 3] 온도조절기 베이스 설치
   • 각 방 설치 위치 결정 (내벽·높이)
   • 매립박스 또는 표면 브래킷 고정
   • 배선 홀 통과 확인
         │
         ▼
  [STEP 4] 배선 작업
   • 제어기 ↔ 구동기 전선 포설
   • 제어기 ↔ 온도조절기 신호선 포설
   • 제어기 ↔ 보일러/HIU 연동선 포설
   • 전선관·몰딩 고정 및 정리
         │
         ▼
  [STEP 5] 결선 작업
   • 단자 결선 (Chapter 6 결선도 준수)
   • 전선 마킹 태그 부착
   • 접지선 연결
         │
         ▼
  [STEP 6] 결선 점검
   • 멀티미터 도통 점검
   • 단락·단선 확인
   • 결선도와 실제 배선 대조
         │
         ▼
  [STEP 7] 전원 인가 및 기초 동작 확인
   • 차단기 ON
   • 제어기 전원 LED 확인
   • 각 존 온도조절기 표시 확인
         │
         ▼
  [STEP 8] 설치 완료 검수
   • 검수 기준 체크리스트 완료
   • 이상 없으면 Chapter 8 시운전으로
```

## 7.2 장치별 설치 기준 (높이·위치·간격)

### 제어기 설치 기준

```
┌─────────────────────────────────────────────────────┐
│  제어기 설치 위치 기준                               │
│                                                     │
│  [분배기]                                           │
│     │                                               │
│     │ ← 가능하면 1m 이내                            │
│     ▼                                               │
│  [제어기 설치 위치]                                 │
│     • 바닥에서 1.2~1.5m 높이 (작업·확인 편의)       │
│     • 습기·직사광선 없는 곳                         │
│     • 환기 공간 확보 (제어기 상하 5cm 이상)         │
│     • 분전반과의 전선 경로 최단 거리                 │
└─────────────────────────────────────────────────────┘
```

| 항목 | 기준 | 이유 |
|------|------|------|
| 설치 높이 | 바닥에서 1.2~1.5m | 단자 작업·LED 확인 편의 |
| 분배기와의 거리 | 1m 이내 권장 | 구동기 배선 길이 최소화 |
| 습기 | 건조한 환경 | 기판 부식 방지 |
| 환기 | 상하 5cm 이상 여유 | 내부 발열 배출 |
| 고정 방식 | DIN 레일 또는 벽면 나사 | 진동·이탈 방지 |

### 구동기 설치 기준

```
분배기 단면도
┌─────────────────────────────────────────────────────┐
│                                                     │
│   공급 헤더                                         │
│   ┌──┬──┬──┬──┐                                    │
│   │  │  │  │  │  ← 여기에 구동기 장착               │
│   └──┴──┴──┴──┘                                    │
│                                                     │
│  구동기 장착 방향:                                   │
│  ① 수직 상향 (가장 일반적, 권장)                    │
│  ② 수평 (제품 허용 시)                              │
│  ③ 수직 하향 → 대부분 금지 (왁스 편마모)            │
│                                                     │
│  장착 순서:                                         │
│  수동 핸들 제거 → 어댑터 체결 → 구동기 삽입 →      │
│  고정 링 또는 클립으로 잠금                         │
└─────────────────────────────────────────────────────┘
```

| 항목 | 기준 | 이유 |
|------|------|------|
| 장착 방향 | 수직 상향 또는 수평 | 왁스 소자 균일 팽창 |
| 어댑터 규격 | 분배기 나사 규격 일치 (M28/M30) | 기밀 유지, 누수 방지 |
| 고정 상태 | 손으로 흔들어도 움직이지 않음 | 진동으로 인한 결선 이탈 방지 |
| 전선 여유 | 구동기 ~ 제어기 간 10cm 여유 | 유지보수 시 탈착 편의 |

### 온도조절기 설치 기준

```
방 내부 설치 위치 기준
┌─────────────────────────────────────────────────────┐
│                                                     │
│  천장 ─────────────────────────────────────────    │
│                                                     │
│              ← 에어컨 취출구 방향 (설치 금지)       │
│                                                     │
│  ████ ← 창문 (이 옆 설치 금지)                     │
│  ████                                               │
│                                                     │
│          [온도조절기 권장 위치]                     │
│     →    내벽 중앙부, 바닥에서 1.2~1.5m 높이       │
│                                                     │
│  ████                         ████                  │
│  ████ ← TV·냉장고 (1m 이상 이격)  ← 가전 발열     │
│                                                     │
│  바닥 ─────────────────────────────────────────    │
└─────────────────────────────────────────────────────┘
```

| 항목 | 기준 |
|------|------|
| 설치 높이 | 바닥에서 1.2~1.5m |
| 벽면 | 외벽 금지, 내벽 설치 |
| 햇빛 | 직사광선 닿지 않는 위치 |
| 에어컨 | 취출구 방향 1m 이상 이격 |
| 가전 발열 | TV·냉장고 등에서 1m 이상 이격 |
| 수평 | 수평계로 확인 후 고정 |
| 배선 통과 | 벽 뒤 배선 홀 밀봉 (외풍 차단) |

## 7.3 안전 수칙 (전기·화재·누수)

### 전기 안전 수칙

```
┌──────────────────────────────────────────────────────┐
│  전기 작업 필수 안전 수칙                             │
│                                                      │
│  ① 작업 전: 분전반 차단기 OFF                        │
│     └─ 잠금 장치 또는 경고 태그 부착                 │
│                                                      │
│  ② 작업 전: 검전기로 무전압 확인                     │
│     └─ 표시 등 없음 확인 후 작업 시작                │
│                                                      │
│  ③ 작업 중: 절연 장갑·절연 드라이버 착용             │
│                                                      │
│  ④ 작업 중: 한 손 작업 원칙                          │
│     └─ 양손으로 통전부 동시 접촉 금지                │
│                                                      │
│  ⑤ 작업 중: 물기 있는 손·환경에서 작업 금지          │
│                                                      │
│  ⑥ 작업 후: 도통 점검 완료 후에만 차단기 ON          │
└──────────────────────────────────────────────────────┘
```

### 화재 예방 수칙

| 수칙 | 내용 |
|------|------|
| 전선 허용 전류 준수 | 전선 굵기에 맞는 전류 이하로 사용 |
| 단자 나사 조임 확인 | 느슨한 단자 → 아크(불꽃) 발생 → 화재 위험 |
| 전선 피복 손상 확인 | 작업 전후 피복 상태 육안 점검 |
| 가연성 재료 근처 작업 금지 | 전선 작업 구역에 종이·천 제거 |
| 차단기 용량 준수 | 과부하 차단기 교체 금지 |

### 누수 예방 수칙

| 수칙 | 내용 |
|------|------|
| 구동기 장착 전 밸브 잠금 | 분배기 공급 밸브 잠금 후 구동기 교체 |
| 어댑터 체결 토크 확인 | 너무 강하면 나사 손상, 너무 약하면 누수 |
| 장착 후 물 채우고 누수 점검 | 구동기 장착 후 밸브 열어 5분 이상 관찰 |
| 배관 연결부 테이프 점검 | 실테이프(테프론) 감김 상태 확인 |

### 작업 즉시 중단 기준

```
아래 상황 발생 시 즉시 작업 중단 → 안전 확보 → 전문가 호출

  ❌ 전선에서 연기나 냄새 발생
  ❌ 차단기가 반복적으로 트립됨
  ❌ 배관 연결부에서 물이 새거나 뚝뚝 떨어짐
  ❌ 가스 냄새가 느껴짐
  ❌ 감전 느낌 (찌릿함) 발생
  ❌ 구조물(벽·바닥)에 예상치 못한 배선·배관 발견
```

## 7.4 설치 완료 검수 기준

### 설치 완료 검수 체크리스트

#### A. 제어기 검수

```
[ ] 제어기가 수평으로 단단히 고정되어 있다
[ ] 전원 LED가 정상 점등된다
[ ] 모든 단자 나사가 규정 토크로 조여져 있다
[ ] 전원선·신호선·구동기선이 올바른 단자에 결선되어 있다
[ ] 접지선이 연결되어 있다
[ ] 제어기 주변 환기 공간(상하 5cm)이 확보되어 있다
```

#### B. 구동기 검수

```
[ ] 모든 구동기가 수직 상향 또는 허용 방향으로 장착되어 있다
[ ] 어댑터 체결 후 손으로 흔들어도 흔들리지 않는다
[ ] 장착 부위에서 누수가 없다 (5분 이상 관찰)
[ ] 구동기 전선이 올바른 단자에 결선되어 있다
[ ] 전선이 케이블 타이로 정리되어 이동·마찰이 없다
[ ] 구동기 LED(있는 경우) 또는 상태 표시가 정상이다
```

#### C. 온도조절기 검수

```
[ ] 모든 온도조절기가 지정 높이(1.2~1.5m)에 수평으로 설치되어 있다
[ ] 외벽·직사광선·에어컨·가전 발열 영향 없는 위치에 있다
[ ] 배선 통과 홀이 밀봉되어 있다 (외풍 차단)
[ ] 전원 공급 또는 배터리 장착 상태가 정상이다
[ ] 온도 표시가 정상적으로 나타난다
[ ] 올바른 존 번호에 대응하는 단자에 신호선이 연결되어 있다
```

#### D. 배선·결선 검수

```
[ ] 전선 마킹 태그가 모든 전선에 부착되어 있다
[ ] 전원선과 신호선이 분리 배선되어 있다
[ ] 전선관·몰딩이 단단히 고정되어 있다
[ ] 멀티미터 도통 점검을 모든 선에 완료했다
[ ] 결선도와 실제 배선이 일치한다
[ ] 남은 전선 끝이 절연 처리되어 있다
```

#### E. 안전 검수

```
[ ] 모든 전선 피복에 손상이 없다
[ ] 단자 커버 및 제어기 케이스가 닫혀 있다
[ ] 누수 부위가 없다
[ ] 차단기 용량이 제어기 정격 전류 이상이다
[ ] 접지 저항이 정상 범위(100Ω 이하)이다 (가능한 경우)
```

### 검수 결과 판정 기준

```
┌─────────────────────────────────────────────────────┐
│  검수 판정                                          │
├─────────────┬───────────────────────────────────────┤
│  모든 항목  │  → 시운전 단계(Chapter 8) 진행         │
│  체크 완료  │                                       │
├─────────────┼───────────────────────────────────────┤
│  미완료 항목│  → 해당 항목 재작업 후 재검수          │
│  1개 이상   │     (시운전 진행 금지)                 │
├─────────────┼───────────────────────────────────────┤
│  안전 항목  │  → 즉시 작업 중단                     │
│  미완료     │     전문가 확인 후 재개                │
└─────────────┴───────────────────────────────────────┘
```

## 자주 하는 실수 TOP 5

| # | 실수 | 올바른 방법 |
|---|------|------------|

| 2 | 구동기 방향 확인 없이 장착 | 제품 화살표 또는 매뉴얼 방향 먼저 확인 |

| 4 | 도통 점검 전 전원 인가 | 반드시 멀티미터 도통 점검 완료 후 차단기 ON |
| 5 | 검수 체크리스트 없이 시운전 | 검수 기준 모든 항목 완료 확인 후 시운전 |

## 문제 발생 시 즉시 대응

| 상황 | 즉시 조치 |
|------|-----------|
| 구동기 장착 후 누수 발생 | 분배기 공급 밸브 잠금 → 어댑터 재체결 또는 교체 |
| 차단기 ON 즉시 트립 | 차단기 OFF → 멀티미터 단락 점검 → 원인 해소 후 재투입 |
| 온도조절기 화면 미표시 | 전원 배선 확인 → 배터리 교체 → 제어기 전원 출력 확인 |
| 제어기 특정 LED 이상 | 제조사 매뉴얼 오류 코드 확인 → 해당 존 결선 재확인 |

## 초보자 Q&A

**Q1. 구동기 장착 시 기존 수동 핸들이 잘 안 빠집니다. 어떻게 하나요?**
A. 수동 핸들은 대부분 시계 방향으로 돌리면 잠기고, 반시계 방향으로 돌리면 풀립니다. 오래된 밸브는 고착되어 있을 수 있으므로 윤활제(WD-40 등)를 살짝 분사 후 부드럽게 돌려보세요. 강제로 힘을 주면 밸브 스템이 손상됩니다.

**Q2. 전선관(CD관) 없이 전선을 그냥 벽에 붙여도 되나요?**
A. 가정용 내장 공간(천장·벽 속)은 CD관 또는 PE관을 사용하는 것이 원칙입니다. 노출 배선 시에는 PVC 몰딩을 사용해 전선을 보호하고 미관을 정리하세요.

**Q3. 온도조절기 베이스를 붙일 때 드릴이 꼭 필요한가요?**
A. 석고보드 벽면은 드릴 없이 나사를 돌려 고정할 수 있습니다. 콘크리트·벽돌 벽면은 드릴로 앵커(다월)를 박아야 합니다. 접착 방식은 장기적으로 탈락 위험이 있어 권장하지 않습니다.

**Q4. 설치 중 부품이 파손됐을 때 어떻게 하나요?**
A. 작업을 중단하고 파손된 부품을 교체한 후 재시작합니다. 특히 구동기 어댑터 나사가 파손된 경우 분배기 밸브 스템 손상 여부도 함께 확인하세요.

**Q5. 설치 완료 사진은 왜 찍어야 하나요?**
A. 벽 마감 후 배선 경로를 알 수 없게 됩니다. 나중에 유지보수·추가 공사 시 배선 위치를 파악하기 위해 반드시 배선 경로와 결선 상태 사진을 남겨두세요.
