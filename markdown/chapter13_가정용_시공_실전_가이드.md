## 13-2. 자가 시공 가능 범위 — "혼자 해도 되나요?"

### 법적 기준 (전기공사업법 기준)

```
┌────────────────────────────────────────────────────────────────────┐
│                    작업 범위 판단 기준                                 │
│                                                                      │
│    220V 교류 전원 배선 건드림?                                         │
│             │                                                        │
│      ┌──────┴──────┐                                                 │
│     YES             NO                                               │
│      │               │                                               │
│  전기면허 필요      저전압 신호선만 다룸 → 자가 작업 가능              │
│  (전기공사업자)     (5~24V DC 온도조절기 신호선 등)                    │
└────────────────────────────────────────────────────────────────────┘
```

### 작업별 자가 시공 가능 여부

| 작업 항목 | 자가 가능 여부 | 난이도 | 비고 |
|-----------|:---:|:---:|------|
| 온도조절기(실내기) 교체 | ✔ 가능 | ★☆☆ | 신호선 2~4가닥 연결만 |
| 구동기(액추에이터) 교체 | ✔ 가능 | ★★☆ | 분배기 밸브에 끼우기만 |
| 제어기(컨트롤러) 교체 | △ 주의 | ★★★ | 220V 전원 단자 있음 → 전원 차단 후 진행 |
| 분배기 교체 | ✘ 불가 | — | 배관 공사 → 전문 업체 필수 |
| 지역난방 HIU 설정 변경 | ✘ 불가 | — | 지역난방 공사 관할 |
| 신규 배선 공사 | ✘ 불가 | — | 전기공사업 면허 필요 |
| 기존 배선 재활용 연결 | △ 주의 | ★★☆ | 전원선 아닌 신호선만 |
| 앱·스마트홈 설정 | ✔ 가능 | ★☆☆ | 제품 매뉴얼 참고 |

> **핵심 원칙**: 220V 전원선(굵은 검정·흰색 선)은 절대 자가 작업 금지.
> 가는 신호선(보통 0.75mm² 이하, 여러 색상)은 전원 차단 후 자가 교체 가능.

### 자가 교체 시 안전 수칙

```
[STEP 1] 두꺼비집(분전반)에서 해당 회로 차단기 OFF
    ↓
[STEP 2] 온도조절기 또는 제어기 전면부 탈거
    ↓
[STEP 3] 기존 배선 스마트폰으로 사진 촬영 (원상복구 대비)
    ↓
[STEP 4] 동일한 위치에 동일한 색상의 선 연결
    ↓
[STEP 5] 차단기 ON → 정상 작동 확인
    ↓
[STEP 6] 각 방 온도 설정 후 구동기 작동 소리 확인
```

### 온라인 구매 시 호환성 체크리스트

```
인터넷에서 부품 구매 전 반드시 확인할 것

□ 기존 제어기 브랜드·모델명 확인 (제어기 전면 스티커)
□ 2선식 / 3선식 구분 확인 (현재 구동기 전선 수 세기)
□ 온도조절기 통신 방식 확인 (유선 / 무선 / RS-485)
□ 구동기 전압 규격 확인 (AC 220V / DC 24V 구분)
□ 분배기 포트 수 확인 (현재 루프 수 = 구동기 개수)
□ 월패드 연동 필요 여부 (아파트 홈넷 시스템 확인)
□ AS 가능 지역 확인 (브랜드 AS 센터 위치)
```

> **주의**: 브랜드가 다른 제어기 + 온도조절기 혼용 시 통신 불가 경우 많음.
> 반드시 **같은 브랜드, 같은 세대(Generation) 제품** 조합으로 구매.

## 13-6. 난방비 절감 실전 계산법

### 각방제어 절감 원리

```
[기존 — 전체 가동 방식]
 보일러 가동 → 분배기 → 모든 방 동시 유입
 └── 빈 방도 가열 → 불필요한 에너지 낭비

[각방제어 도입 후]
 보일러 가동 → 분배기 → 필요한 방만 선택적 유입
 └── 빈 방 구동기 닫힘 → 해당 루프 순환 차단
 └── 에너지를 사용 중인 방에만 집중
```

### 절감 효과 추정 공식

| 항목 | 계산식 | 예시 (3룸 기준) |
|------|--------|----------------|
| 평균 사용 방 수 | 실사용 방 ÷ 전체 방 수 | 2 ÷ 4 = 50% |
| 이론적 절감률 | (1 - 사용 비율) × 60% | 50% × 60% = 30% |
| 월 난방비 | 기존 요금 × (1 - 절감률) | 10만원 × 70% = 7만원 |
| 월 절감액 | 기존 요금 × 절감률 | 10만원 × 30% = 3만원 |
| 투자 회수 기간 | 총 설치비 ÷ 월 절감액 | 120만원 ÷ 3만원 = 40개월 |

> **실제 절감률**: 가정마다 다르지만 국내 실측 사례 기준 **겨울철 난방비 15~35% 절감** 보고됨.
> 빈 방 비율이 높을수록, 방 간 온도 편차가 클수록 절감 효과 증가.

### 온도 설정별 절감 전략

| 구역 | 추천 설정 온도 | 이유 |
|------|:---:|------|
| 거실 (주생활 공간) | 21~23°C | 활동 공간, 체감 온도 중요 |
| 안방 (취침) | 18~20°C | 수면 시 낮은 온도가 숙면에 유리 |
| 아이방 (학습/수면) | 20~22°C | 집중력·수면 모두 고려 |
| 서재·작업실 | 20~21°C | 사용 시간에만 가동 |
| 빈 방·창고 | 10~13°C | 동파 방지 최저 온도 유지 |
| 욕실 (온수 바닥) | 25~28°C | 짧은 시간 고온 설정 |

## 13-7. 세입자·임차인 주의사항

### 임차인이 가장 많이 겪는 상황

```
상황 A: "방이 너무 추워서 온도조절기 교체하고 싶은데…"
  → 온도조절기(실내기) 교체는 원상복구 가능 → 집주인 동의 없이 가능
  → 단, 퇴거 시 원상복구(기존 제품 재설치) 의무

상황 B: "각방제어 시스템 자체가 없어서 새로 설치하고 싶다"
  → 신규 배선 공사 포함 → 반드시 집주인 서면 동의 필요
  → 공사비 부담 및 퇴거 시 처리 방법 계약서에 명시 권장

상황 C: "구동기가 고장났는데 집주인이 안 고쳐준다"
  → 임대인의 수선 의무 (민법 제623조) 해당
  → 내용증명 발송 후 자가 수리 → 비용 청구 가능
  → 단, 임차인 고의·과실로 인한 고장은 임차인 부담
```

### 퇴거 시 체크리스트

| 항목 | 처리 방법 |
|------|-----------|
| 임차인이 설치한 온도조절기 | 원상복구 (기존 제품 재설치) 또는 집주인과 협의 잔류 |
| 임차인이 설치한 구동기 | 원상복구 기본, 협의 시 잔류 가능 |
| 임차인 비용으로 신규 설치한 전체 시스템 | 집주인 동의 하에 비용 협의 또는 철거 |
| 공사 중 발생한 벽면 훼손 | 임차인 원상복구 의무 |

### 계약서에 미리 넣을 문구 (예시)

```
"임차인은 임대인의 서면 동의를 받아 각방제어 온도조절 장치
(제어기 1대, 온도조절기 4개, 구동기 4개)를 설치하며,
설치 비용은 임차인이 부담한다. 임대차 계약 종료 시
설치물 처리(원상복구 또는 잔류)에 관하여는 상호 협의하여 결정한다."
```

## 13-8. 업체 선정 & 견적 검증 가이드

### 견적 요청 전 준비물

```
□ 아파트 평수 및 방 구조 (방 개수, 분배기 위치)
□ 현재 시스템 사진 (분배기, 제어기, 기존 온도조절기)
□ 희망 브랜드 또는 기능 (스마트홈 연동 여부 등)
□ 예산 범위 (부품비+시공비 총액 기준)
□ 공사 희망 일정 (평일/주말, 오전/오후)
```

### 정상 견적서 vs 의심 견적서 비교

| 항목 | 정상 견적서 | 의심 견적서 |
|------|------------|------------|
| 부품 명세 | 브랜드·모델명·수량·단가 명시 | "각방제어 일식" 등 뭉뚱그림 |
| 시공비 | 항목별 분리 (철거비, 설치비, 시험비) | 총액만 표기 |
| 부가세 | 별도 또는 포함 명시 | 언급 없음 |
| AS 조건 | 기간·범위·연락처 명시 | 구두로만 약속 |
| 업체 정보 | 사업자등록번호, 주소 | 개인 연락처만 |

### 시세 대비 견적 판단 기준

```

│              30평 3룸 기준 적정 견적 범위 (2026년 기준)           │

│  총액                        │  80~120만원                       │
├──────────────────────────────┼─────────────────────────────────┤
│  ⚠ 50만원 미만 총액 견적     │  저가 부품 / 비공식 루트 의심      │
```

### 사기성 업체 식별 패턴

| 패턴 | 실제 사례 | 대처법 |
|------|----------|--------|
| "오늘만 특가" 압박 | 전화로 당일 결정 유도 | 전화 끊고 재검토 |
| 점검 후 갑자기 비용 추가 | 와서 보니 추가 공사 필요 주장 | 사전 서면 견적 고집 |
| 현금 결제 유도 | 카드 불가, 현금 영수증 없음 | 카드·계좌이체만 사용 |
| 브랜드 불명 부품 사용 | 국산 명품이라 주장하는 무브랜드 부품 | 모델명 현장 확인 |
| AS 연락 두절 | 시공 후 전화 안 받음 | 계약 전 AS센터 전화 테스트 |

### 계약서 필수 체크포인트 5가지

```
✔ 1. 부품 브랜드·모델명·수량 명시
✔ 2. 하자보수 기간 및 무상 AS 범위 명시 (최소 1년)
✔ 3. 총 공사금액 및 부가세 포함 여부 명시
✔ 4. 공사 일정 및 완료 기준 명시
✔ 5. 추가 공사 발생 시 사전 서면 동의 조항 삽입
```

## 13-9. 긴급 상황 자가 대처법

### 상황 1 — 한겨울 난방 갑자기 전혀 안 됨

```
STEP 1. 두꺼비집 확인
   → 각방제어 회로 차단기 트립 여부 확인
   → 트립 시: 원인 제거 후 복구
         ↓
STEP 2. 제어기 전원 LED 확인
   → 꺼져 있음: 전원 플러그 또는 차단기 확인
   → 켜져 있음: 다음 단계
         ↓
STEP 3. 보일러 작동 여부 확인
   → 보일러 자체 오류 코드 있음: 보일러 AS
   → 보일러 정상: 다음 단계
         ↓
STEP 4. 분배기 수동 밸브 확인
   → 분배기 옆 빨간/파란 수동 밸브 모두 열려 있는지 확인
   → 닫혀 있으면: 반시계 방향으로 돌려 열기
         ↓
STEP 5. 온도조절기에서 수동 가동 시도
   → 설정 온도를 현재 온도보다 5°C 이상 높게 설정
   → 구동기 작동음(칙칙 소리) 들리면 정상 신호
         ↓
STEP 6. 위 모두 해결 안 됨 → 업체 AS 접수
```

### 상황 2 — 특정 방만 안 따뜻함

```
점검 순서:

온도조절기 설정 온도 확인
        │
  설정 낮음? → 온도 올리기
        │
  설정 정상?
        ↓
해당 방 구동기 확인 (분배기에서)
        │
  구동기 LED 없거나 소리 없음?
        ↓
구동기 전선 접촉 불량 확인
   → 커넥터 재결합 시도
        │
  여전히 이상?
        ↓
구동기 교체 (자가 가능, 1~2만원대 부품)
```

### 상황 3 — 동파 방지 긴급 조치

```
장기 공실·여행 중 동파 예방 설정

✔ 온도조절기 전체 → 최저 온도 10°C 이상으로 설정
✔ 보일러 동파 방지 모드 활성화 (보일러 조작부에서)
✔ 분배기 수동 밸브 — 모두 열린 상태 유지 (순환 보장)
✔ 수도 계량기함 보온재 확인
✔ 외출 전 모든 온도조절기 앱으로 원격 확인
```

### 상황 4 — AS 전화 전 빠른 셀프 체크

| 증상 | 자가 점검 순서 | 예상 소요 시간 |
|------|--------------|:---:|
| 전체 난방 안됨 | 차단기 → 보일러 → 분배기 밸브 | 5분 |
| 특정 방만 안됨 | 온도 설정 → 구동기 → 신호선 접촉 | 10분 |
| 온도조절기 화면 꺼짐 | 전원 어댑터 → 배터리 교체 | 3분 |
| 앱 연결 안됨 | Wi-Fi 재연결 → 라우터 재시작 | 5분 |
| 구동기 소음 심함 | 구동기 고정 나사 조임 → 교체 검토 | 5분 |

> **불필요한 출장비 방지**: 대부분의 경우 위 자가 점검으로 해결됩니다.
> 출장비는 보통 2~5만원이므로 셀프 점검 먼저, 해결 안 되면 AS 접수.

## 13-10. 실수 TOP 5 — 가정용 시공에서 자주 하는 실수

| 순위 | 실수 내용 | 결과 | 예방법 |
|:---:|-----------|------|--------|
| 1 | 브랜드 혼용 구매 (제어기 A + 온도조절기 B) | 통신 불가, 작동 안 됨 | 같은 브랜드 세트로 구매 |
| 2 | 5GHz Wi-Fi에 스마트 온도조절기 연결 시도 | 연결 계속 실패 | 2.4GHz 공유기 SSID에 연결 |
| 3 | 전원 차단 없이 제어기 선 건드림 | 쇼트 → 제어기 고장 | 반드시 차단기 OFF 후 작업 |
| 4 | 아파트 관리사무소 미신고 후 공사 | 경고·원상복구 요구 | 공사 전 신고 필수 |
| 5 | 구동기 방향 확인 없이 장착 | 밸브 열림/닫힘 반대로 작동 | 장착 후 작동 방향 반드시 확인 |

## 13-11. Q&A — 가정용 자주 묻는 질문

**Q1. 각방제어 설치하면 겨울 난방비가 얼마나 줄어요?**
> 가정마다 다르지만 15~35% 절감이 일반적입니다. 특히 낮에 비어 있는 방이 많은 맞벌이 가정일수록 효과가 큽니다.

**Q2. 온도조절기만 스마트형으로 교체할 수 있나요?**
> 기존 제어기와 통신 방식이 맞아야 합니다. 제어기 브랜드에서 출시한 스마트 온도조절기로만 교체 가능하며, 타 브랜드 단독 스마트 온도조절기는 제어기 없이 독립 작동하는 방식(무선 보일러 차단형)으로만 가능합니다.

**Q3. 구동기(분배기에 붙은 작은 모터)가 고장나면 직접 교체할 수 있나요?**
> 네. 구동기는 분배기 밸브에 클립으로 끼우는 방식이라 전기 지식 없어도 교체 가능합니다. 단, 동일 전압(AC220V 또는 DC24V) 규격의 제품으로 교체해야 합니다.

**Q4. 온도조절기 화면에 "E" 오류가 뜨는데요?**
> 제품마다 오류 코드가 다릅니다. 제어기 또는 온도조절기 케이스 안쪽 스티커나 제품 설명서의 오류 코드 표를 확인하세요. 주로 센서 단선(E1), 통신 오류(E2~E5), 전원 이상(E9) 등입니다.

**Q5. 여름에도 각방제어 시스템을 켜둬야 하나요?**
> 냉방 기능이 없는 순수 난방 시스템이라면 여름에는 꺼두어도 됩니다. 단, 제어기 전원은 유지해야 Wi-Fi 연결이 끊기지 않습니다. 냉난방 겸용 제품(팬코일 유닛 연동형)은 여름에도 냉방 모드로 활용 가능합니다.
