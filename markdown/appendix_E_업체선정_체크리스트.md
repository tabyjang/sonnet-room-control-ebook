## E-2. 업체 기본 신뢰도 확인 체크리스트

> 견적 전화 또는 방문 시 확인할 사항입니다.

```
[ 사업자 정보 ]
□ 사업자등록번호를 알려주나?
□ 상호명·대표자명·주소가 명확한가?
□ 전기공사업 또는 설비업 등록 업체인가?
□ 네이버·카카오 지도 등에 실제 사업장이 등록되어 있는가?
□ 리뷰(구글·네이버 플레이스)가 실재하는가? (리뷰 0개 주의)

[ 상담 태도 ]
□ 견적 전 현장 실사 또는 사진 요청을 하는가? (정상)
□ 전화 한 통에 바로 가격을 확정하려 하지 않는가?
□ "오늘만 할인" 압박이 없는가?
□ 질문에 구체적으로 답변하는가?
□ 부품 브랜드·모델명을 먼저 말해주는가?
```

## E-3. 현장 실사 체크리스트 (업체 방문 시)

```
[ 업체 담당자가 확인해야 할 것 ]
□ 분배기 위치 및 현재 구동기 상태 직접 확인
□ 제어기 위치 및 기존 배선 상태 점검
□ 각 방 온도조절기 위치·배선 수 확인
□ 220V 전원 공급 경로 파악
□ 아파트 세대 내 배관 구조 파악 (지역난방 여부)

[ 내가 확인할 것 ]
□ 담당자가 분배기·배선을 직접 보고 판단하는가?
□ 사진 촬영 후 이를 견적 근거로 활용하는가?
□ "추가 공사 필요할 수도 있다"는 말을 솔직하게 하는가?
□ 현장 실사 없이 전화로만 최종 가격을 제시하면 신뢰도 낮음
```

## E-7. 계약서 필수 체크리스트

```
계약 전 아래 항목이 계약서에 모두 포함되었는지 확인

□ 갑(발주자) 성명·주소·연락처
□ 을(업체) 상호·사업자등록번호·대표자·주소·연락처
□ 공사 장소 (아파트명·동·호수)
□ 공사 내용 (부품 브랜드·모델명·수량 명시)
□ 공사 총금액 및 부가세 포함 여부
□ 대금 지급 조건 (계약금·잔금 비율 및 시기)
□ 공사 착수일 및 완료 예정일
□ 하자보수 책임 기간 및 범위

□ 분쟁 발생 시 처리 방법 (관할 법원 또는 소비자분쟁조정위원회)
□ 갑·을 서명 또는 날인
```

## E-8. 계약금·잔금 지급 가이드

| 단계 | 지급 시점 | 권장 비율 | 주의사항 |
|------|----------|:---:|----------|
| 계약금 | 계약서 작성 시 | 30% | 현금영수증 또는 세금계산서 수령 |
| 중도금 | 공사 착수일 (선택) | 0~30% | 대규모 공사에서만 적용 |
| 잔금 | 시운전 완료·인수 후 | 40~70% | 모든 방 정상 작동 확인 후 지급 |

> **핵심 원칙**: 잔금은 반드시 시운전 완료 후 지급. 공사 전 100% 현금 요구하면 계약 거부.

## E-9. 공사 완료 후 확인 체크리스트

```
시공 업체가 철수하기 전 반드시 확인할 것

[ 기능 확인 ]
□ 전체 방 온도조절기 전원 켜짐 확인
□ 각 방 온도 설정 → 구동기 작동음 확인 (방별로 1개씩)
□ 보일러 연동 — 난방 요청 시 보일러 가동 여부 확인
□ 스마트폰 앱 연결 및 원격 제어 테스트 (스마트 제품의 경우)

[ 마감 확인 ]
□ 온도조절기 벽면 고정 상태 (흔들림 없음)
□ 제어기 벽면 고정 및 커버 체결 상태
□ 구동기 분배기 고정 상태 (탈거 없음)
□ 배선 정리 상태 (노출 배선 케이블타이 정리)
□ 공사 중 발생한 벽면·바닥 오염 청소 여부

[ 인수인계 ]
□ 제품 보증서·설명서 수령
□ AS 연락처 메모 또는 명함 수령
□ 앱 계정 설정 및 로그인 정보 확인
□ 계약서 사본·영수증 보관
□ 잔금 지급 및 영수증 수령
```

## E-10. AS 요청 전 자가 점검 체크리스트

> 불필요한 출장비(회당 2~5만원) 방지를 위해 먼저 확인하세요.

| 증상 | 먼저 확인할 것 | 자가 해결 가능? |
|------|--------------|:---:|
| 특정 방 안 따뜻함 | 온도 설정값 확인 → 현재 온도보다 높게 설정되어 있는지 | ✔ |
| 온도조절기 화면 꺼짐 | 전원 연결·배터리 확인 | ✔ |
| 앱 연결 안됨 | Wi-Fi 재연결, 라우터 재시작 | ✔ |
| 전체 난방 안됨 | 두꺼비집 차단기 확인 → 보일러 오류 코드 확인 | △ |
| 구동기 소음 | 고정 나사 조임 | ✔ |
| 오류 코드 표시 | 제품 설명서 오류 코드 표 확인 | △ |
| 보일러 연동 안됨 | 보일러 자체 오류 여부 먼저 확인 | △ |
