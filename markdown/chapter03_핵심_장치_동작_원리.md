# Chapter 3. 핵심 장치 동작 원리

---

## 한눈에 보기 (핵심 5줄)

1. 각방제어의 핵심 장치는 **제어기(Controller)**, **구동기(Actuator)**, **온도조절기(Thermostat)** 세 가지다.
2. 온도조절기는 방 온도를 측정하고 명령을 내리는 "두뇌", 제어기는 명령을 처리하는 "중앙 처리 장치", 구동기는 실제 밸브를 움직이는 "손발"이다.
3. 세 장치가 정확히 연동되어야 원하는 온도가 유지된다.
4. 센서 위치가 잘못되면 온도 오차가 생겨 난방이 과잉 또는 부족해진다.
5. 각 장치의 역할과 고장 증상을 알면 문제 발생 시 빠르게 원인을 찾을 수 있다.

---

## 왜 중요한가? — 생활 속 예시

**예시 1 — "설정은 22°C인데 방은 27°C다"**

온도조절기 센서가 에어컨 바람이나 창문 근처 냉기에 노출되어 있으면,
실제 방 온도와 다른 값을 읽어 제어기가 계속 밸브를 열어둡니다.
장치 역할을 이해하면 "센서 위치 문제"임을 빠르게 진단할 수 있습니다.

**예시 2 — "구동기를 교체했는데도 밸브가 안 열린다"**

구동기는 정상이어도 제어기 출력 단자가 불량이거나, 온도조절기 신호선이 단선이면 동작하지 않습니다.
세 장치의 역할 분담을 알아야 어디서 문제가 생겼는지 순서대로 점검할 수 있습니다.

## 3.1 제어기(Controller) 기능과 제어 로직

### 제어기란?

제어기는 온도조절기로부터 신호를 받아 구동기에 명령을 전달하는 **중간 처리 장치**입니다.
분배기 근처(분전반 또는 별도 박스)에 설치되며, 여러 존을 동시에 관리합니다.

```
[온도조절기 1] ──신호──┐
[온도조절기 2] ──신호──┤
[온도조절기 3] ──신호──┤  →  [제어기]  →  [구동기 1,2,3...]
[온도조절기 N] ──신호──┘          │
                                  └→ [보일러 연동 신호]
```

### 주요 기능

| 기능 | 설명 |
|------|------|
| 존별 신호 수신 | 각 온도조절기의 ON/OFF 신호 입력 처리 |
| 구동기 출력 제어 | 해당 존의 구동기에 전압 출력 (열림/닫힘) |
| 보일러 연동 | 1개 이상 존에서 난방 요청 시 보일러 기동 신호 출력 |
| 전원 공급 | 일부 제품은 온도조절기에 DC 전원 공급 |
| 상태 표시 | LED 또는 디스플레이로 각 존 동작 상태 표시 |

### 제어 로직 흐름

```
① 온도조절기 → 제어기: "난방 필요" 신호 입력
② 제어기 → 구동기: 해당 존 밸브 열기 출력
③ 제어기 → 보일러: 콜-포-히트(Call for Heat) 신호 출력
④ 보일러 기동 → 난방수 순환 시작
⑤ 온도 설정값 도달 → 온도조절기 신호 해제
⑥ 제어기 → 구동기: 밸브 닫기
⑦ 모든 존 요청 없을 시 → 보일러 신호 해제 → 보일러 정지
```

### 제어기 단자 구성 (일반적)

| 단자 | 기호 | 역할 |
|------|------|------|
| 전원 입력 | L, N | AC 220V 전원 |
| 보일러 연동 | BLR 또는 CH | 보일러 콜-포-히트 출력 |
| 존 입력 | Z1~ZN | 온도조절기 신호 수신 |
| 존 출력 | A1~AN | 구동기 전압 출력 |
| 접지 | GND | 안전 접지 |

## 3.2 구동기(Actuator) 종류와 밸브 연동 방식

### 구동기란?

구동기는 분배기 밸브 위에 장착해 **전기 신호에 따라 밸브를 열고 닫는 장치**입니다.
"전동 밸브 헤드" 또는 "액추에이터"라고도 불립니다.

### 구동기 종류

#### 1) 열동형 (Thermal/Wax) 구동기 — 가장 일반적

| 항목 | 내용 |
|------|------|
| 동작 원리 | 내부 왁스(Wax) 소자에 전기 가열 → 팽창 → 핀 돌출 → 밸브 개방 |
| 전선 방식 | 2선식 (가열 ON/OFF) |
| 동작 시간 | 열림: 2~3분, 닫힘(냉각 복귀): 3~5분 |
| 장점 | 구조 단순, 저렴, 소음 없음 |
| 단점 | 응답 느림, 중간 개도 불가 |
| 적용 | 일반 주거용 각방제어 대다수 |

#### 2) 모터형 구동기

| 항목 | 내용 |
|------|------|
| 동작 원리 | 소형 전동 모터로 밸브 축 회전 |
| 전선 방식 | 3선식 (COM/OPEN/CLOSE) |
| 동작 시간 | 2~5분 (제품별) |
| 장점 | 응답 빠름, 위치 제어 가능 |
| 단점 | 가격 높음, 모터 고장 가능성 |
| 적용 | 상업용, 고급 주거용 |

### 분배기 밸브 연동 방식

```
분배기
 ├─ 유로(Flow path) ── 밸브 스템 ── 구동기 핀 접촉
 │
 │   [구동기 OFF / 냉각 상태]    [구동기 ON / 가열 상태]
 │     핀 수축 → 스프링 닫힘       핀 팽창 → 밸브 열림
 │
 └─ 난방수 흐름: 차단              개통
```

### 구동기 설치 방향

- 구동기는 **반드시 수직(상향) 또는 수평** 장착 (제품 지정 방향 준수)
- 역방향 설치 시 왁스 소자 편마모 → 조기 고장 원인
- 분배기 밸브 규격(M28, M30 나사 등)과 구동기 어댑터 일치 확인 필수

## 3.3 온도조절기(Thermostat) 센싱 및 제어 특성

### 온도조절기란?

온도조절기는 사용자가 목표 온도를 설정하고, 실내 온도를 측정해 제어기에 신호를 보내는 **방별 제어 인터페이스**입니다.

### 온도 측정 방식

| 방식 | 원리 | 특징 |
|------|------|------|
| NTC 서미스터 | 온도 상승 시 저항 감소 | 소형, 저렴, 일반적 |
| PTC 서미스터 | 온도 상승 시 저항 증가 | 일부 고급 제품 |
| PT100 (백금) | 정밀 저항 변화 | 산업용, 고정밀 |

> 주거용 각방제어는 대부분 **NTC 서미스터** 방식입니다.

### 제어 방식

#### ON/OFF 제어 (가장 일반적)

```
설정 온도: 22°C
히스테리시스(Hysteresis): ±0.5°C

동작:
  실내 온도 < 21.5°C  →  "난방 켜기" 신호 출력
  실내 온도 > 22.5°C  →  "난방 끄기" 신호 출력
```

- **히스테리시스**: 온도조절기가 너무 자주 ON/OFF 되는 것을 방지하는 온도 폭
- 히스테리시스가 없으면 설정온도 부근에서 밸브가 수백 번 개폐 → 구동기 수명 단축

#### PID 제어 (고급형)

- 현재 온도·변화율·누적 오차를 계산해 밸브 개도를 비례적으로 제어
- 온도 오버슈트(설정값 초과) 최소화
- 주거용보다 상업용·정밀 제어 환경에서 주로 사용

### 온도조절기 출력 신호 방식

| 방식 | 설명 | 연결 대상 |
|------|------|-----------|
| 무전압 접점 (건접점) | 내부 릴레이 스위치만 제공, 전원 없음 | 제어기의 존 입력 단자 |
| 전압 출력 | DC 3V/5V/12V 등 직접 출력 | 제어기 또는 구동기 직결 |

> 대부분의 주거용 시스템은 **무전압 접점** 방식입니다.

## 3.4 센서 오차, 위치 선정, 보정 기준

### 센서 오차 원인

| 원인 | 증상 | 대책 |
|------|------|------|
| 직사광선 | 실제보다 높은 온도 측정 | 햇빛 닿지 않는 내벽에 설치 |
| 에어컨·환기구 근처 | 차가운 공기로 낮은 온도 측정 | 기류 영향 없는 위치로 이동 |
| 가전제품 발열 | TV·냉장고 열로 높은 온도 측정 | 발열체에서 1m 이상 이격 |
| 외벽 설치 | 벽체 냉기 전도로 낮은 온도 측정 | 내벽(단열 측)에 설치 |
| 바닥에 너무 낮게 설치 | 바닥 복사열 영향 | 바닥에서 1.2~1.5m 높이 권장 |

### 설치 위치 기준

```
[올바른 위치]
- 내벽 기준
- 바닥에서 1.2~1.5m 높이
- 출입구에서 먼 쪽 (문 개폐 바람 영향 제거)
- 햇빛, 에어컨, 가전 발열 없는 곳

[피해야 할 위치]
- 외벽 (냉기 전도)
- 창문 옆 (외풍)
- TV/냉장고 옆 (발열)
- 에어컨 취출구 정면
- 욕실/주방 근처 (습기, 증기)
```

### 온도 보정(Offset) 방법

온도조절기 설정에서 ±수°C 범위로 보정값을 입력합니다.

**보정 절차**:
1. 정밀 온도계(기준 온도계)를 온도조절기 옆 동일 높이에 설치
2. 30분 이상 안정화 대기
3. 기준 온도계 측정값과 온도조절기 표시값 비교
4. 차이만큼 오프셋 설정 (예: 기준 21°C, 표시 22°C → 오프셋 -1°C)

> **보정 범위**: 일반적으로 ±3°C 이내. 범위 초과 시 센서 불량 의심.

## 문제 발생 시 즉시 대응

| 증상 | 추정 원인 | 1차 조치 |
|------|-----------|----------|
| 설정온도와 실제온도 차이 큼 | 센서 위치 불량 또는 오차 | 보정값 조정, 위치 변경 검토 |
| 특정 존 구동기 무반응 | 제어기 출력 불량 또는 결선 불량 | 해당 존 단자 전압 측정 |
| 보일러가 켜지지 않음 | 연동선 단선 또는 접촉 불량 | BLR 단자 연결 상태 확인 |
| 밸브 열림 후 닫히지 않음 | 온도조절기 접점 고착 | 온도조절기 교체 검토 |

> 멀티미터로 단자 전압을 측정할 때도 반드시 **한 손 작업 원칙**을 지키세요.
> (감전 시 심장 통과 방지를 위해 한 손은 주머니에 넣고 측정)

## 초보자 Q&A

**Q1. 온도조절기를 거실에만 달면 안 되나요?**
A. 거실 온도조절기 하나로 전체를 제어하면 각방제어가 아닙니다. 각 방마다 독립적인 온도조절기가 있어야 방별로 다른 온도를 설정할 수 있습니다.

**Q2. 제어기 없이 온도조절기와 구동기를 직접 연결할 수 있나요?**
A. 일부 단순 2선식 시스템에서는 가능하지만, 보일러 연동·다중 존 관리·전원 배분을 위해 제어기 사용을 권장합니다. 직결 시 보일러 연동이 안 되어 에너지 낭비가 생깁니다.

**Q3. 온도조절기 배터리가 없는데 전원은 어떻게 공급되나요?**
A. 유선형 온도조절기는 제어기에서 신호선과 함께 DC 전원(3V~12V)을 공급받습니다. 배터리형은 무선 또는 독립 전원 방식입니다.

**Q4. 구동기가 동작하는 "딸깍" 소리가 나는 게 정상인가요?**
A. 열동형 구동기는 소음이 거의 없습니다. 반복적인 딸깍 소리는 밸브 완전 개폐 불량이나 히스테리시스 미설정으로 인한 잦은 ON/OFF일 수 있습니다. 히스테리시스 설정을 확인하세요.

**Q5. 제어기를 분배기 근처가 아닌 다른 곳에 설치할 수 있나요?**
A. 가능하지만 구동기까지의 배선 거리가 길어져 전압강하·노이즈 문제가 생길 수 있습니다. 가급적 분배기(구동기) 근처 또는 분전반 내 설치를 권장합니다.
