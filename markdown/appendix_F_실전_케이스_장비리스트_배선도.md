## CASE F-1. 신축 아파트 33평 — 거실 5구 분배기, 3룸 각방제어 신규 설치

### 상황 요약

```
┌─────────────────────────────────────────────────────────────────┐
│  주거 유형   : 신축 아파트 (입주 1년 이내)                         │
│  평형        : 33평 (전용 84㎡)                                   │
│  난방 방식   : 지역난방 (HIU 설치됨)                              │
│  방 구성     : 거실 + 안방 + 방2 + 방3                            │
│  분배기      : 거실 벽면 매립 5구 분배기                           │
│              (루프 5개 중 — 거실 2루프, 안방 1루프, 방2 1루프, 방3 1루프)│
│  기존 시스템 : 없음 (신규 설치)                                   │
│  배선 상태   : 신축이므로 제어기 → 각 방 배선 타공 가능            │
│  목표        : 3개 방 독립 온도 제어, 앱 연동 없음 (기본형)        │
└─────────────────────────────────────────────────────────────────┘
```

|:---:|------|-----------|:---:|------|
| 1 | 각방제어기 (컨트롤러) | 4채널 이상, AC 220V 입력, 3선식 출력 | 1대 | 안방·방2·방3·예비 1채널 |
| 2 | 온도조절기 (실내기) | 유선형, RS-485 또는 전용 신호 방식 | 3개 | 안방·방2·방3 각 1개 |
| 3 | 전동 구동기 (액추에이터) | 3선식, AC 220V, NC형(상시닫힘) | 3개 | 분배기 루프 3개에 장착 |

### F-1-4. 거실 루프 처리 방법

```
5구 분배기에서 거실 루프(루프1·루프2)는 각방제어 없이 상시 개방 운영

방법 A — 구동기 미설치 (권장)
  루프1, 루프2에 구동기 장착 안 함
  → 밸브 핸들을 '열림' 위치로 고정
  → 거실은 보일러 가동 시 항상 난방됨

방법 B — 별도 온도조절기 추가
  루프1+루프2를 묶어 구동기 1개 + 온도조절기 1개 추가
  → 제어기 채널 4개 모두 사용 (안방·방2·방3·거실)
  → 거실도 독립 온도 제어 가능
```

### F-1-5. 설정 순서

```
STEP 1. 제어기 설치 (거실 벽면 또는 분배기 옆)
  → 분전반 전용 차단기 OFF 후 전원선 연결
  → 차단기 ON → 제어기 LED 점등 확인

STEP 2. 구동기 장착 (분배기 루프3·4·5)
  → 구동기를 밸브 어댑터에 끼우고 클립 고정
  → 구동기 전선(3심) → 제어기 CH1·CH2·CH3 단자에 연결
  → 전선 색상: 보통 갈색=L, 파랑=N, 주황/회색=제어신호

STEP 3. 온도조절기 설치 (안방·방2·방3)
  → 벽면 신호선 인출구 위치 확인
  → 신호선 4심 → 온도조절기 단자 연결
  → 온도조절기 베이스 벽면 고정 → 본체 결합

STEP 4. 제어기 채널 매핑 설정
  → 제어기 설정 메뉴 진입 (보통 리셋+설정 동시 5초)
  → CH1 = 안방 온도조절기 주소 01
  → CH2 = 방2 온도조절기 주소 02
  → CH3 = 방3 온도조절기 주소 03

STEP 5. 시운전
  → 각 방 온도조절기 온도를 현재보다 5°C 높게 설정
  → 해당 구동기 열림 동작 확인 (2~3분 후 '칙' 소리)
  → 보일러 기동 여부 확인
  → 목표 온도 도달 후 구동기 닫힘 동작 확인

STEP 6. 히스테리시스 설정 (선택)
  → 제어기 설정에서 히스테리시스 1.0°C 설정 권장
  → 너무 작으면 구동기 반복 동작 → 소음·수명 단축
```

---

### F-1-6. 체크리스트 (시공 완료 후)

```
□ 제어기 전원 LED 정상 점등
□ 온도조절기 3개 전원 및 현재 온도 표시 정상
□ 각 방별 온도 올리면 해당 구동기만 열리는지 확인
□ 다른 방 구동기는 닫혀 있는지 확인
□ 거실 루프1·2 수동 개방 상태 고정 확인
□ 분배기 누수 없음 확인
□ 배선 정리 (몰딩 또는 케이블 타이)
□ 제어기 채널 매핑 설정값 사진 촬영 보관
```

## CASE F-2. 구축 아파트 40평 — 기존 2선식 → 3선식 교체, 4룸

### 상황 요약

```
┌─────────────────────────────────────────────────────────────────┐
│  주거 유형   : 구축 아파트 (준공 15년)                            │
│  평형        : 40평 (전용 115㎡)                                  │
│  난방 방식   : 개별난방 (가스보일러)                              │
│  방 구성     : 거실 + 안방 + 방2 + 방3 + 서재                    │
│  분배기      : 보일러실 6구 분배기                                │
│  기존 시스템 : 2선식 각방제어 (설치 15년, 구동기 2개 고장)         │
│  배선 상태   : 기존 2심 배선 전 구간 포설됨                       │
│  목표        : 3선식으로 전체 교체, 서재 추가, 보일러 연동 유지    │
└─────────────────────────────────────────────────────────────────┘
```

---

### F-2-1. 필요 장비 리스트

| 번호 | 품목 | 규격·사양 | 수량 | 비고 |
|:---:|------|-----------|:---:|------|
| 1 | 각방제어기 | 6채널, 3선식 출력, 보일러 연동 단자 포함 | 1대 | 기존 제어기 철거 |
| 2 | 온도조절기 | 유선형 3선식 대응 | 5개 | 거실 포함 5개 방 |
| 3 | 3선식 전동 구동기 | AC 220V, NC형, 기존 밸브 어댑터 호환 확인 | 5개 | 기존 2선식 구동기 전수 교체 |
| 4 | 신호선 | 0.75mm² × 4심 | 60m | 기존 2심선 교체 필요 구간만 |
| 5 | 구동기 전원선 | 0.75mm² × 3심 | 20m | 분배기 구간 |
| 6 | 보일러 연동선 | 0.75mm² × 2심 | 5m | 제어기 → 보일러 |
| 7 | 전선관 (CD관) | Φ16 | 5m | 추가 배선 구간 |
| 8 | 커넥터 / 와이어너트 | 전선 접속용 | 1세트 | |

**핵심 확인 사항**

```
2선식 → 3선식 교체 전 반드시 확인할 것:

□ 기존 분배기 밸브 어댑터 규격 확인
   → 3선식 구동기와 호환 여부 (나사 피치·어댑터 크기)
   → 호환 불가 시: 밸브 어댑터도 교체 필요 (+추가 비용)

□ 기존 배선관(배선 피복 속 전선 수) 확인
   → 2선식 배선(2심)은 3선식 구동기에 사용 불가
   → 제어기 → 분배기 구간은 3심 신규 포설 필요

□ 보일러 연동 방식 확인
   → 기존 제어기의 보일러 ON/OFF 신호 방식 (무전압 접점 / 12V 등)
   → 신규 제어기와 동일 방식인지 확인
```

```

### F-2-4. 2선식 → 3선식 교체 순서

```
STEP 1. 분전반 차단기 OFF (각방제어 전용 회로)

STEP 2. 기존 2선식 구동기 전수 탈거
  → 밸브 어댑터 클립 해제 → 구동기 위로 분리
  → 전선 커넥터 분리

STEP 3. 기존 2선식 제어기 철거
  → 단자대 전선 모두 분리 → 제어기 벽면 분리

STEP 4. 신규 배선 포설
  → 제어기 위치 → 분배기 구간 : 3심 전선 신규 포설
  → 기존 2심 전선 활용 가능한 신호선 구간은 재활용
  → 추가된 서재 구간 : 신규 4심 신호선 포설

STEP 5. 신규 3선식 제어기 설치 및 배선 연결

STEP 6. 신규 3선식 구동기 장착
  → 밸브 어댑터 호환 확인 후 장착

### F-5-1. 단종 제품 대체 확인 절차

```
STEP 1. 기존 온도조절기 모델명 확인
  → 기기 뒷면 스티커 또는 제어기 단자대 라벨 확인

STEP 2. 제조사 AS 센터 전화
  → "모델 Y 단종 여부, 대체 모델 문의"
  → 대부분 동일 브랜드 현행 모델로 대체 가능

STEP 3. 대체 모델 호환성 확인
  → 통신 방식 동일 여부 (RS-485 주소 방식 등)
  → 전원 전압 동일 여부 (DC 12V / DC 24V 등)
  → 단자 수 동일 여부 (2심 / 4심)

STEP 4. 자가 교체
  → 차단기 OFF
  → 기존 온도조절기 베이스에서 본체만 분리
  → 기존 배선 그대로 신규 모델에 동일하게 연결
  → 주소 설정 (신규 모델에서 기존 주소값 동일하게 설정)
  → 차단기 ON → 작동 확인
```
