# Chapter 5. 설계 및 시공 전 준비

---

## 한눈에 보기 (핵심 5줄)

1. 시공 전 현장 조사를 충분히 하면 자재 낭비·재공사를 막을 수 있다.
2. 방(존) 개수와 루프 길이를 파악해야 제어기·구동기 수량을 정확히 산출한다.
3. 자재는 제어기·구동기·온도조절기·전선·분배기 순으로 목록을 작성한다.
4. 전기 도면과 설비 도면을 함께 검토해야 배선 경로와 간섭 요소를 사전에 파악한다.
5. 전문가 의뢰 구간(배전반 내부, 구조 배관)을 먼저 구분해 작업 범위를 명확히 한다.

---

## 왜 중요한가? — 생활 속 예시

**예시 1 — "구동기를 샀는데 분배기 나사 규격이 안 맞는다"**

현장 분배기 밸브 나사 규격(M28·M30 등)을 미리 확인하지 않으면
구동기를 사 와도 장착이 안 됩니다. 반품·재구매로 며칠을 허비하게 됩니다.

**예시 2 — "제어기를 설치했는데 전선 길이가 짧아 온도조절기까지 닿지 않는다"**

각 방까지의 거리를 미리 측정하지 않으면
전선이 부족하거나, 반대로 남아서 정리가 어려워집니다.

## 5.1 현장 조사 체크리스트

### 현장 조사 흐름도

```
현장 도착
    │
    ▼
① 열원 장치 확인
  (보일러 / HIU 모델·위치·연동 방식)
    │
    ▼
② 분배기 확인
  (위치·루프 수·밸브 규격·현재 구동기 방식)
    │
    ▼
③ 배관·배선 경로 확인
  (전선관 여유·배관 간섭·통과 구간)
    │
    ▼
④ 각 방 확인
  (방 크기·루프 길이·온도조절기 설치 가능 위치)
    │
    ▼
⑤ 분전반 확인
  (빈 차단기 슬롯·접지 단자·제어기까지 거리)
    │
    ▼
⑥ 현장 메모·사진 기록
    │
    ▼
자재 목록 작성
```

## 5.2 부하 추정과 존 구성 원칙

### 방별 부하 추정

```
난방 부하 간이 산출식

  필요 열량(W) = 방 면적(㎡) × 단위 부하(W/㎡)

  단위 부하 기준:
  ┌─────────────────────┬──────────────┐
  │ 건물 유형           │ 단위 부하    │
  ├─────────────────────┼──────────────┤
  │ 고단열 신축 아파트  │ 30~40 W/㎡   │
  │ 일반 아파트         │ 40~60 W/㎡   │
  │ 노후 아파트         │ 60~80 W/㎡   │
  │ 단독주택            │ 70~100 W/㎡  │
  └─────────────────────┴──────────────┘
```

### 존 구성 원칙

| 원칙 | 설명 | 예시 |
|------|------|------|
| 1존 1온도조절기 | 존마다 독립 온도 제어 | 안방 1개 = Zone 1 |
| 넓은 공간 분할 | 40㎡ 이상 거실은 2존 분할 검토 | 거실·주방 분리 |
| 유사 사용 패턴 묶기 | 항상 같이 사용하는 공간은 1존 | 주방+식당 |
| 루프 길이 균형 | 한 루프 최대 100m 이내 | 긴 루프는 2개 분할 |
| 욕실·화장실 | 독립 존 또는 인근 방과 묶기 | 소면적으로 독립 존 비효율 |

### 존 구성 예시 (3베드룸 아파트)

```
┌─────────────────────────────────┐
│  평면도 예시 (84㎡ 아파트)       │
│                                 │
│  ┌────────┐  ┌──────────┐       │
│  │        │  │          │       │
│  │ 안방   │  │  거실    │       │
│  │ Zone 2 │  │  Zone 1  │       │
│  │ (12㎡) │  │  (25㎡)  │       │
│  └────────┘  └──────────┘       │
│  ┌────────┐  ┌──────────┐       │
│  │아이방  │  │  주방    │       │
│  │ Zone 3 │  │  Zone 1  │       │
│  │ (10㎡) │  │  (포함)  │       │
│  └────────┘  └──────────┘       │
│  ┌────────┐                     │
│  │  서재  │                     │
│  │ Zone 4 │                     │
│  │ (8㎡)  │                     │
│  └────────┘                     │
└─────────────────────────────────┘
  → 총 4존 / 온도조절기 4개 / 구동기 4개
```

## 5.3 자재 선정 기준 (제어기·구동기·온도조절기)

### 자재 선정 흐름도

```
현장 조사 완료
      │
      ▼
  존 수 확정
      │
      ├─ 제어기: 존 수 이상 채널 제품 선택
      ├─ 구동기: 분배기 루프 수 = 구동기 수
      │           + 밸브 나사 규격 맞춤 어댑터 확인
      ├─ 온도조절기: 존 수만큼
      │              + 유선/무선 선택
      └─ 부자재: 전선·전선관·커넥터·나사 등
```

### 제어기 선정 기준

| 항목 | 확인 내용 |
|------|-----------|
| 채널 수 | 존 수 이상 (여유 1~2채널 권장) |
| 구동기 방식 | 2선식 전용 / 3선식 전용 / 겸용 |
| 보일러 연동 | 콜-포-히트 출력 단자 유무 |
| 전원 | AC 220V 입력 |
| 온도조절기 전원 공급 | DC 전원 공급 가능 여부 |
| 설치 형태 | DIN 레일 / 벽면 박스 |
| 제조사 A/S | 국내 A/S 가능 여부 |

### 구동기 선정 기준

| 항목 | 확인 내용 |
|------|-----------|
| 방식 | 2선식 열동형 / 3선식 모터형 |
| 밸브 나사 규격 | M28×1.5 / M30×1.5 (분배기 규격 일치) |
| 작동 전압 | AC 220V / AC 24V (제어기 출력 전압 일치) |
| 작동 온도 범위 | 설치 환경 기준 |
| 동작 시간 | 열림·닫힘 시간 확인 |
| 방수 등급 | 보일러실 습기 환경 고려 |

### 온도조절기 선정 기준

| 항목 | 확인 내용 |
|------|-----------|
| 연결 방식 | 유선(2선/4선) / 무선(RF·Wi-Fi) |
| 전원 방식 | 제어기에서 DC 공급 / 자체 배터리 |
| 표시 방식 | LCD / LED / 터치스크린 |
| 프로그램 기능 | 시간별·요일별 스케줄 설정 가능 여부 |
| 히스테리시스 설정 | 조정 가능 여부 |
| 앱 연동 | 스마트홈·앱 제어 지원 여부 |
| 설치 방식 | 매립형 / 표면 부착형 |

### 자재 수량 산출표 (예시: 4존)

| 자재 | 수량 산출 기준 | 예시 수량 |
|------|--------------|-----------|
| 제어기 | 1대 (4채널 이상) | 1대 |
| 구동기 | 분배기 루프 수 | 4개 |
| 온도조절기 | 존 수 | 4개 |
| 신호선 (2심) | 제어기~각 온도조절기 거리 합계 + 여유 20% | 실측 후 산출 |
| 구동기 전선 (2심 or 3심) | 제어기~분배기 구동기 수×거리 | 실측 후 산출 |
| 전선 커넥터 | 접속 개소 × 2 | 20개 이상 |
| 구동기 어댑터 | 분배기 규격 맞는 것 | 구동기 수만큼 |
| 차단기 | 제어기용 전용 | 1개 |
| 전선관 (CD관) | 배선 경로 합계 | 실측 후 산출 |

## 5.4 도면 검토 포인트 (전기/설비 협업)

### 전기 도면 검토 항목

```
전기 도면에서 확인해야 할 것
┌────────────────────────────────────────┐
│ ① 분전반 위치 및 회로 구성             │
│    └─ 제어기용 여유 슬롯 확인          │
│                                        │
│ ② 각 방 전기 배선 경로                 │
│    └─ 온도조절기 신호선 경로 계획       │
│                                        │
│ ③ 접지 계통                           │
│    └─ 제어기 접지 단자 연결 가능 여부  │
│                                        │
│ ④ 기존 전선관 경로 및 여유 공간        │
│    └─ 신규 전선 추가 삽입 가능 여부    │
└────────────────────────────────────────┘
```

### 설비 도면 검토 항목

```
설비 도면에서 확인해야 할 것
┌────────────────────────────────────────┐
│ ① 분배기 위치 및 배관 경로             │
│    └─ 구동기 설치 공간 확인            │
│                                        │
│ ② 각 방 루프 배관 경로 및 길이         │
│    └─ 루프 길이 기반 밸런싱 계획       │
│                                        │
│ ③ 공급·환수 배관 방향                 │
│    └─ 구동기 설치 방향 확인            │
│                                        │
│ ④ 열원(보일러/HIU) 연동 배관           │
│    └─ 제어기 연동 신호선 경로 계획     │
└────────────────────────────────────────┘
```

### 전기·설비 협업 주요 확인 사항

| 구분 | 확인 항목 | 협의 내용 |
|------|-----------|-----------|
| 전기 | 제어기 전용 회로 | 분전반에 전용 차단기 설치 계획 |
| 전기 | 온도조절기 배선 경로 | 도배·마감 전 선행 작업 일정 |
| 전기 | 접지 연결 | 접지선 공유 또는 독립 접지 |
| 설비 | 분배기 구동기 교체 | 기존 배관 연결 상태 확인 |
| 설비 | 보일러/HIU 연동 | 외부 제어 입력 단자 확인 |
| 설비 | 루프 밸런싱 | 설치 완료 후 유량 조정 일정 |

### 전문가 의뢰 구간 정리

```
┌──────────────────────────────────────────────────┐
│          작업 범위 구분                            │
├───────────────────────┬──────────────────────────┤
│  직접 가능 (이 전자책) │  전문가 의뢰 필수         │
├───────────────────────┼──────────────────────────┤
│ 온도조절기 교체         │ 분전반 내부 회로 작업     │
│ 구동기 교체             │ 접지 공사                │
│ 제어기 단자 결선        │ 구조 변경 배관 공사       │
│ 신호선 연결             │ 가스·열공급 배관 작업     │
│ 기본 점검 및 시운전     │ HIU 1차측 작업           │
│                         │ 법정 자격 필요 전기 공사 │
└───────────────────────┴──────────────────────────┘
```

| 상황 | 즉시 조치 |
|------|-----------|
| 구동기 규격 불일치 발견 | 작업 중단, 올바른 어댑터 또는 구동기 재선택 |
| 전선 길이 부족 | 커넥터로 연장, 또는 경로 변경 |
| 분전반 여유 슬롯 없음 | 전기 전문가 의뢰로 분전반 증설 또는 회로 재배치 |
| 기존 배선관 여유 없음 | 표면 배선(몰딩) 또는 별도 전선관 추가 계획 |

## 초보자 Q&A

**Q1. 도면이 없는 구형 아파트는 어떻게 하나요?**
A. 도면이 없으면 현장 실측이 더 중요합니다. 분배기·분전반·각 방을 직접 눈으로 확인하고 줄자로 거리를 재며, 기존 배선 경로를 추적해 메모합니다.

**Q2. 존 수가 많을수록 무조건 좋은가요?**
A. 꼭 그렇지 않습니다. 존이 많을수록 장치 수·배선·비용이 늘어납니다. 실제 사용 패턴에 맞게 필요한 방만 독립 존으로 구성하는 것이 효율적입니다.

**Q3. 온도조절기를 무선으로 선택하면 배선 작업이 없나요?**
A. 온도조절기 신호선은 없어지지만, 구동기 전선과 제어기 전원은 여전히 유선으로 필요합니다. 무선 온도조절기는 배터리 교체 관리가 필요합니다.

**Q4. 자재를 온라인으로 사도 괜찮나요?**
A. 가능하지만 분배기 규격(나사 사이즈)과 제어기-구동기 호환성을 반드시 확인 후 구매하세요. 규격 불일치로 반품·재구매가 발생하는 경우가 많습니다.

**Q5. 시공 전 관리사무소나 건물주에게 신고해야 하나요?**
A. 아파트의 경우 공용 설비(배관·배전)에 영향이 없는 세대 내 자재 교체는 대부분 신고 불필요합니다. 단, 배관 공사·분전반 작업이 포함되면 관리사무소에 사전 신고를 권장합니다.
