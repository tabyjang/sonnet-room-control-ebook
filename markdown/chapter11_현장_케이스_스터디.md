# Chapter 11. 현장 케이스 스터디

---

## 한눈에 보기 (핵심 5줄)

1. 실제 현장 사례는 교과서에 없는 복합 원인과 현실적 해결 과정을 보여준다.
2. 신축·리모델링·반복 고장·실패 사례는 각각 다른 패턴의 교훈을 담고 있다.
3. 케이스마다 "증상 → 진단 과정 → 원인 → 조치 → 결과 → 교훈" 흐름으로 정리했다.
4. 비슷한 상황이 발생했을 때 이 케이스를 참고하면 진단 시간을 크게 단축할 수 있다.
5. 실패 사례에서 얻은 교훈이 성공 사례만큼 중요하다.

## 11.1 신축 현장 적용 사례

### 케이스 1. 신축 84㎡ 아파트 4존 각방제어 초기 설치

#### 현장 조건

```
┌─────────────────────────────────────────────────────┐
│  현장 정보                                          │
│  • 건물 유형: 신축 아파트 (입주 전)                 │
│  • 세대 면적: 84㎡ (방 3개 + 거실/주방)             │
│  • 열원: 개별 보일러 (콘덴싱 가스 보일러)           │
│  • 분배기: 6루프 (거실·안방·아이방·서재·주방·욕실)  │
│  • 적용 방식: 3선식 / 존 수: 4존                    │
│  • 스마트홈: 없음 (일반 유선 온도조절기)            │
└─────────────────────────────────────────────────────┘
```

#### 진행 과정

| 단계 | 작업 내용 | 소요 시간 | 특이 사항 |
|------|-----------|-----------|-----------|
| 현장 조사 | 분배기·분전반·각 방 실측 | 1시간 | 신축이라 도면 정확함 |
| 자재 선정 | 제어기(4ch)·구동기 6개·온도조절기 4개 | - | 주방·욕실은 인접 존에 통합 |
| 제어기 설치 | 보일러실 벽면 고정 | 30분 | - |
| 구동기 장착 | 분배기 6루프 전체 | 1시간 | M30 어댑터 사용 |
| 온도조절기 설치 | 4개 방 내벽 | 1시간 | 수평계 사용 |
| 배선·결선 | 신호선·구동기선·보일러 연동 | 2시간 | CD관 경로 사전 계획 |
| 시운전 | 존별 동작·보일러 연동 | 1시간 | - |

#### 문제 발생 및 해결

```
[문제] 시운전 중 거실 온도가 설정값 초과 (과열)

원인 분석:
  거실 루프(18m) < 안방 루프(35m)
  → 짧은 거실 루프에 유량 집중
  → 밸런싱 밸브 미조정 상태

해결:
  분배기 환수 헤더 밸런싱 밸브로
  거실 루프 유량 25% 감소 조정
  → 전 루프 유량 균등화 후 정상
```

#### 결과 및 교훈

| 항목 | 내용 |
|------|------|
| 설치 총 소요 시간 | 약 6.5시간 (1인 기준) |
| 최종 동작 | 전 존 정상 |
| 난방비 절감 (추정) | 이전 대비 약 20% |
| **핵심 교훈** | **신축이라도 루프 길이 차이가 있으면 밸런싱 밸브 조정 필수** |

### 케이스 2. 신축 현장 스마트홈 연동 문제

#### 현장 조건 및 증상

```
┌─────────────────────────────────────────────────────┐
│  현장 정보                                          │
│  • 건물 유형: 신축 스마트홈 아파트                  │
│  • 각방제어: Wi-Fi 온도조절기 + 스마트홈 허브 연동  │
│  • 증상: 앱에서 설정 변경 시 30~60초 지연           │
│          가끔 설정이 적용 안 되고 원래 값으로 복귀  │
└─────────────────────────────────────────────────────┘
```

#### 진단 과정

```
증상: 앱 설정 지연·미적용
          │
          ▼
  ① Wi-Fi 신호 강도 측정
     → 온도조절기 위치 수신 강도: -80dBm (약함)
          │
          ▼
  ② 스마트홈 허브와 온도조절기 통신 프로토콜 확인
     → 온도조절기: Zigbee 프로토콜
     → 스마트홈 허브: Wi-Fi 전용 (Zigbee 지원 안 함)
     → 프로토콜 불일치 확인
          │
          ▼
  원인: 프로토콜 불일치로 인한 간헐적 통신 오류
```

#### 해결 및 교훈

| 조치 | 내용 |
|------|------|
| 즉시 조치 | Zigbee 지원 스마트홈 허브로 교체 |
| 보완 조치 | 중간 Zigbee 리피터 1개 추가 |
| 결과 | 앱 응답 2초 이내, 안정적 동작 |
| **핵심 교훈** | **자재 선정 시 온도조절기·제어기·스마트홈 허브의 통신 프로토콜 사전 호환성 확인 필수** |

## 11.2 노후 아파트 교체 사례

### 케이스 3. 15년 된 아파트 2선식→3선식 전환 교체

#### 현장 조건

```
┌─────────────────────────────────────────────────────┐
│  현장 정보                                          │
│  • 건물 유형: 준공 15년 아파트 (리모델링)           │
│  • 기존 시스템: 2선식 열동형 구동기 (노후)          │
│  • 요청 사항: 3선식 모터형으로 교체                 │
│  • 기존 배선: 2심 전선 (전선관 내)                  │
└─────────────────────────────────────────────────────┘
```

#### 진단 과정 및 문제

```
[사전 조사 결과]

  기존 배선 상태 점검
  ├─ 전선관 내부: 2심 전선 (L, N)
  ├─ 전선관 여유 공간: 거의 없음
  └─ 전선 피복 상태: 일부 노후화 확인

  문제 발생:
  3선식 구동기는 3심 전선 필요
  → 기존 2심 전선 재활용 불가
  → 전선관 내 추가 전선 삽입 불가 (여유 없음)
```

#### 해결 방안 비교 검토

```
┌───────────────┬────────────────────┬──────────────────┐
│ 방안          │ 내용               │ 선택 결과        │
├───────────────┼────────────────────┼──────────────────┤
│ 안 A          │ 기존 전선관 교체   │ 마감 공사 필요   │
│ (전면 재배선) │ + 3심 전선 재포설  │ 비용·기간 과다   │
│               │                    │ ✕ 기각           │
├───────────────┼────────────────────┼──────────────────┤
│ 안 B          │ 3선식 포기,        │ 비용 저렴        │
│ (2선식 유지)  │ 2선식 신제품 교체  │ 기존 배선 재활용 │
│               │                    │ ✓ 채택           │
├───────────────┼────────────────────┼──────────────────┤
│ 안 C          │ 표면 몰딩으로      │ 미관 저하        │
│ (표면 배선)   │ 3심 신규 포설      │ 거주자 반대      │
│               │                    │ ✕ 기각           │
└───────────────┴────────────────────┴──────────────────┘
```

#### 결과 및 교훈

| 항목 | 내용 |
|------|------|
| 최종 선택 | 2선식 신형 구동기 + 신형 제어기로 교체 |
| 추가 작업 | 노후 전선 피복 손상 구간 절연 테이프 처리 |
| 결과 | 전 존 정상 동작, 배선 추가 공사 없음 |
| **핵심 교훈** | **교체 전 기존 배선 방식·상태·전선관 여유를 반드시 먼저 조사해야 불필요한 재공사를 막는다** |

### 케이스 4. 리모델링 중 분배기 밸브 규격 불일치

#### 현장 조건 및 증상

```
┌─────────────────────────────────────────────────────┐
│  현장 정보                                          │
│  • 건물 유형: 20년 된 아파트 전면 리모델링          │
│  • 문제: 구매한 구동기가 분배기에 장착 안 됨        │
│  • 분배기 밸브 규격: M28×1.5                        │
│  • 구매한 구동기 어댑터 규격: M30×1.5               │
└─────────────────────────────────────────────────────┘
```

#### 진행 과정 및 해결

```
[사건 경위]

  ① 온라인에서 일반 구동기 구매 (M30 어댑터 포함)
  ② 현장 도착 후 장착 시도
  ③ 나사 규격 불일치 → 장착 불가 발견
  ④ 분배기 규격 실측: M28×1.5 확인
  ⑤ M28 어댑터 별도 주문 (3일 소요)

[비용 손실]
  • 공사 지연: 3일
  • 추가 배송비 발생
  • 거주자 불편 발생
```

#### 결과 및 교훈

| 항목 | 내용 |
|------|------|
| 해결 방법 | M28 어댑터 추가 구매 후 정상 장착 |
| 지연 비용 | 3일 공사 지연 + 추가 배송비 |
| **핵심 교훈** | **현장 방문 시 분배기 밸브 나사 규격 실측 (버니어캘리퍼스 또는 규격 게이지)이 자재 구매 전 필수 절차** |

## 11.3 반복 고장 현장 개선 사례

### 케이스 5. 구동기 3개월마다 반복 교체

#### 현장 조건 및 증상

```
┌─────────────────────────────────────────────────────┐
│  현장 정보                                          │
│  • 증상: 특정 존(Zone 3) 구동기가 3개월마다 고장   │
│  • 동일 구동기 이미 4회 교체 (1년 6개월)           │
│  • 증상: 밸브가 완전히 닫히지 않다가 코일 과열      │
└─────────────────────────────────────────────────────┘
```

#### 근본 원인 탐색 과정

```
반복 고장 원인 탐색
          │
          ▼
  ① 구동기 제품 불량 가능성
     → 다른 제조사 제품으로도 동일 증상 재현
     → 제품 불량 아님
          │
          ▼
  ② 설치 방향 점검
     → 수직 상향 정상 장착 확인
     → 방향 문제 아님
          │
          ▼
  ③ 분배기 밸브 스템 점검
     → 밸브 스템에 석회질(스케일) 두꺼운 층 발견
     → 구동기 핀이 완전 수축해도 밸브 닫힘 불완전
     → 구동기 핀이 항상 스트레스 상태 → 코일 과열 → 소손
          │
          ▼
  근본 원인: 밸브 스템 스케일 축적
  (구동기 교체만으로는 해결 불가)
```

#### 해결 과정

| 조치 순서 | 작업 내용 |
|-----------|-----------|
| 1 | 분배기 공급 밸브 잠금 후 해당 루프 물 빼기 |
| 2 | 분배기 밸브 스템 분리 |
| 3 | 스케일 제거제로 밸브 스템 세척 |
| 4 | 밸브 스템 재조립 후 작동 테스트 |
| 5 | 신규 구동기 장착 |
| 6 | 연수기 설치 검토 (재발 방지) |

#### 결과 및 교훈

| 항목 | 내용 |
|------|------|
| 조치 후 경과 | 18개월 이상 재고장 없음 |
| 예방 조치 | 연 1회 밸브 스템 점검 스케줄 추가 |
| **핵심 교훈** | **반복 고장은 부품 불량이 아닌 근본 원인(스케일·환경)을 먼저 탐색해야 한다** |

### 케이스 6. 온도조절기 오작동 반복

#### 현장 조건 및 증상

```
┌─────────────────────────────────────────────────────┐
│  현장 정보                                          │
│  • 증상: 특정 방 온도조절기 숫자가 계속 떨림        │
│  • 설정 온도와 상관없이 밸브가 불규칙 개폐          │
│  • 온도조절기 3회 교체해도 동일 증상 반복           │
└─────────────────────────────────────────────────────┘
```

#### 근본 원인 탐색

```
배선 경로 추적

  [제어기] ─── [전선관 A] ─── [온도조절기 Z3]

  전선관 A 내부 구성 점검:
  ├─ 구동기 전원선 2심 (AC 220V)  ← 전력선
  ├─ 온도조절기 신호선 2심         ← 신호선
  └─ 보일러 연동선 2심

  → 전력선과 신호선 동일 전선관 통과 확인!
  → 전자기 유도로 신호선에 노이즈 유입
  → 온도조절기 오작동 원인 특정
```

#### 해결 과정

```
개선 전                      개선 후
┌────────────────┐           ┌────────────────┐
│ 전선관 (혼합)  │           │ 전선관 A (전력)│
│ • 구동기선     │    →      │ • 구동기선     │
│ • 신호선  ← 문제│          │ • 보일러 연동선│
│ • 보일러 연동선│           └────────────────┘
└────────────────┘           ┌────────────────┐
                             │ 전선관 B (신호)│
                             │ • 신호선 (쉴드)│
                             └────────────────┘
```

| 조치 | 내용 |
|------|------|
| 1 | 신호선 전선관 분리 (별도 전선관) |
| 2 | 신호선 → 쉴드 케이블(CVVS)로 교체 |
| 3 | 쉴드 제어기 측 GND 접지 |
| 결과 | 온도조절기 정상, 재발 없음 |
| **핵심 교훈** | **신호선은 반드시 전력선과 분리 배선. 교체 전 노이즈 환경 점검이 먼저** |

## 11.4 실패 사례에서 얻은 설계·시공 교훈

### 케이스 7. 전원 인가 후 제어기 소손

#### 사건 경위

```
┌─────────────────────────────────────────────────────┐
│  실패 경위                                          │
│                                                     │
│  ① 결선 완료 후 도통 점검 생략                      │
│  ② 차단기 ON → 즉시 제어기에서 연기 발생            │
│  ③ 차단기 즉시 OFF                                  │
│  ④ 제어기 내부 회로 소손 확인                       │
│                                                     │
│  원인 추적:                                         │
│  구동기 출력 단자(A1+)에 전원 L선이 연결됨          │
│  → 제어기 내부에서 출력단과 전원단 단락 발생        │
│  → 내부 회로 즉시 소손                              │
└─────────────────────────────────────────────────────┘
```

#### 피해 규모 및 교훈

| 항목 | 내용 |
|------|------|

| 지연 | 2일 (부품 재수급) |
| 원인 | 결선 도통 점검 생략 |
| **핵심 교훈** | **도통 점검은 절약할 수 있는 단계가 아니다. 5분의 점검이 수십만 원의 피해를 막는다** |

## 초보자 Q&A

**Q1. 케이스 스터디의 증상이 우리 집과 비슷한데 똑같이 따라하면 되나요?**
A. 비슷한 증상이라도 현장마다 원인이 다를 수 있습니다. 케이스를 참고해 진단 방향을 잡되, 반드시 Chapter 9~10의 진단 절차대로 직접 확인 후 조치하세요.

**Q2. 스케일(석회질)은 어떻게 예방하나요?**
A. 연수기 설치로 경수를 연수로 전환하거나, 연 1회 밸브 스템 점검 및 세척을 정기 점검에 포함시키면 스케일 축적을 줄일 수 있습니다.

**Q3. 지역난방 아파트에서 작업 전 관리사무소에 어떻게 신고하나요?**
A. 관리사무소 방문 또는 전화로 "세대 내 각방제어 구동기 교체 작업 예정"임을 알리고, 작업 일시와 연락처를 남겨두세요. 대부분의 관리사무소는 세대 내 단순 부품 교체는 별도 승인 없이 허용합니다.

**Q4. 케이스 8처럼 실수로 1차측 밸브를 잠궜을 때 즉시 해야 할 일은?**
A. 즉시 잠근 밸브를 원래대로 열고, 관리사무소에 바로 연락해 단지 계통 압력 정상 여부를 확인 요청하세요. 혼자 판단해 방치하면 단지 전체 피해가 커집니다.

**Q5. 반복 고장이 아닌 처음 발생한 고장도 케이스 스터디처럼 접근해야 하나요?**
A. 네, 동일한 접근법이 유효합니다. "증상 → 원인 계층 분류 → 측정으로 확인 → 단일 조치 → 재시험" 흐름은 처음 발생한 고장에도 그대로 적용됩니다.
