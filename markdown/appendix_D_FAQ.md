## D-1. 기초 개념 FAQ

**Q1. 각방제어 시스템이 없으면 어떤 문제가 생기나요?**

```
A. 각방제어 없이 보일러만 켜면 전체 방이 동일한 온도로 가열됩니다.
   발생하는 문제:
   ┌─────────────────────────────────────────────────────┐
   │ • 빈 방에도 난방비 지출 (에너지 낭비)               │
   │ • 가족별 선호 온도 차이 반영 불가                   │
   │ • 특정 방 과열 또는 저온 발생                       │
   │ • 장기적으로 난방비 과다 지출                       │
   └─────────────────────────────────────────────────────┘
```

---

**Q2. 2선식과 3선식 중 어느 것이 더 좋나요?**

| 상황 | 권장 방식 | 이유 |
|------|-----------|------|
| 신축·새 배선 공사 | 3선식 | 정밀 제어, 장기 유지보수 유리 |
| 기존 2가닥 배선 재활용 | 2선식 | 추가 배선 공사 불필요 |
| 비용 절감이 우선 | 2선식 | 구동기·배선 비용 저렴 |
| 스마트홈 연동 예정 | 3선식 | 위치 피드백 지원 제품 선택 가능 |

> 어느 방식이 절대적으로 우월하지 않습니다. 현장 조건에 맞게 선택하세요.

**Q5. 각방제어를 하면 난방비가 얼마나 절감되나요?**

| 조건 | 예상 절감률 |
|------|------------|
| 낮 시간대 빈 방 차단 | 10~20% |
| 방별 최적 온도 개별 설정 | 5~15% |
| 스케줄 설정 활용 | 10~20% |
| 세 조건 모두 적용 | **최대 25~35%** |

> 실제 절감률은 건물 단열 성능, 외기온도, 거주 패턴에 따라 달라집니다.

---

**Q6. 바닥 난방이 아닌 라디에이터(방열기)에도 적용할 수 있나요?**

A. 네, 가능합니다. 다만 방열기용 구동기(서모스태틱 밸브 헤드)는 바닥 난방용 분배기 구동기와 형태가 다릅니다. 방열기 밸브 규격에 맞는 구동기를 선택해야 합니다.

**Q12. 온도조절기 위치를 선택할 때 가장 중요한 기준이 뭔가요?**

```
온도조절기 위치 선정 3대 원칙
  ① 실제 생활 공간 온도를 대표하는 위치
     → 내벽 중앙부, 바닥에서 1.2~1.5m

  ② 외부 요인 영향이 없는 위치
     → 직사광선·에어컨·가전 발열·외풍 없는 곳

  ③ 배선 경로가 확보되는 위치
     → 벽 뒤 배선 홀·전선관 통과 가능 여부 확인
```

---

**Q13. 분배기 구동기 교체 시 물을 다 빼야 하나요?**

A. 분배기 공급 밸브를 잠그면 해당 루프만 격리할 수 있습니다. 전체 물을 빼지 않아도 구동기 교체 가능합니다. 단, 교체 후 반드시 밸브를 천천히 열어 5분 이상 누수 확인하세요.

---

**Q15. 온도조절기를 벽에 매립 설치하면 뭐가 다른가요?**

| 구분 | 매립형 | 표면 부착형 |
|------|--------|------------|
| 미관 | 깔끔함 | 벽면 돌출 |
| 배선 | 벽 속으로 숨김 | 몰딩 또는 노출 |
| 설치 난이도 | 높음 (타공·배선 작업) | 낮음 |
| 교체 편의성 | 낮음 (배선 홀 재작업) | 높음 |
| 권장 시점 | 신축·리모델링 마감 전 | 기설치 벽면 |

**Q16. 구동기를 직접 구매할 때 어떤 규격을 확인해야 하나요?**

```
구동기 구매 전 반드시 확인할 4가지
  ① 방식: 2선식 or 3선식 (현장 제어기 방식과 일치)
  ② 나사 규격: M28 or M30 (분배기 밸브 규격 실측)
  ③ 전압: AC 220V or AC 24V (제어기 출력 전압 확인)
  ④ 동작 시간: 제품 스펙의 열림·닫힘 시간 확인
```

---

**Q17. 보일러 연동 신호선은 어떤 단자에 연결하나요?**

A. 보일러 제조사마다 다르지만 대부분 "외부 온도조절기 입력", "실내온도조절기", "CH 입력" 등으로 표시됩니다. 무전압 건접점 방식이 가장 일반적이며, 보일러 설명서에서 해당 단자를 확인 후 제어기 BLR 단자와 연결합니다.

## D-3. 사용·조작 FAQ

**Q18. 난방비를 가장 효과적으로 줄이는 방법은?**

```
난방비 절감 실천 순위
  ┌────┬─────────────────────────────────────────────┐
  │ 1위│ 외출 시 모든 존 16°C로 낮추기               │
  │    │ (끄지 말고 낮추는 것이 재가열 에너지 절약)  │
  ├────┼─────────────────────────────────────────────┤
  │ 2위│ 스케줄 설정 활용                            │
  │    │ (기상 1시간 전 자동 가열 → 쾌적하게 절약)  │
  ├────┼─────────────────────────────────────────────┤
  │ 3위│ 사용하지 않는 방 밸브 완전 차단             │
  ├────┼─────────────────────────────────────────────┤
  │ 4위│ 방별 적정 온도 설정 (과도하게 높이지 않기) │
  │    │ (1°C 높일수록 난방비 약 6% 증가)           │
  └────┴─────────────────────────────────────────────┘
```

**Q19. 히스테리시스는 어떻게 설정하는 게 좋나요?**

| 상황 | 권장 히스테리시스 | 이유 |
|------|-------------------|------|
| 일반 주거 (바닥 난방) | 1~2°C | 느린 바닥 온도 응답에 적합 |
| 정밀 온도 유지 필요 | 0.5°C | 온도 변동 최소화 |
| 구동기 소음 심한 경우 | 2~3°C | ON/OFF 횟수 줄이기 |

---

**Q20. 장기간 집을 비울 때 어떻게 설정하나요?**

```
장기 부재 시 권장 설정
  ┌─────────────────────────────────────────────────────┐
  │ 단기 외출 (1일 이내):    16°C 절약 모드 설정        │
  │ 수일 이상 부재:          10~12°C 동파 방지 모드      │
  │ 1개월 이상 장기 공실:    5°C 또는 동파 방지 기능 ON │
  │                                                     │
  │ 주의: 완전히 전원을 끄면 동파 위험!                 │
  │       최소한 동파 방지 온도(5°C)는 유지하세요.     │
  └─────────────────────────────────────────────────────┘
```

---

**Q21. 스케줄 설정은 어떻게 하는 건가요?**

A. 온도조절기 설정 메뉴에서 "스케줄", "프로그램", "예약" 항목을 찾습니다. 요일·시간대별로 목표 온도를 설정합니다.

**예시 설정 (평일 직장인):**

| 시간대 | 설정 온도 | 설명 |
|--------|-----------|------|
| 06:00 | 22°C | 기상 준비 |
| 08:30 | 16°C | 출근 후 절약 |
| 18:00 | 22°C | 귀가 준비 |
| 23:00 | 20°C | 취침 |

---

**Q22. 온도조절기 화면에 "E" 또는 오류 표시가 뜹니다.**

| 표시 | 일반적 의미 | 조치 |
|------|------------|------|
| E1, E2... | 센서 불량 또는 단선 | 제품 매뉴얼 오류 코드 확인 |
| Lo | 배터리 부족 | 배터리 교체 |
| - - - | 센서 연결 불량 | 센서선 접촉 확인 |
| 깜빡임 | 통신 오류 또는 전원 불안정 | 전원 확인, 제어기 재시작 |

---

**Q23. 여름에 각방제어 시스템을 어떻게 관리하나요?**

A. 비난방 기간에는 다음만 지키면 됩니다.
- 온도조절기 전원 유지 (설정온도를 낮게 유지하거나 여름 모드)
- 제어기 전원 유지 (단, 보일러는 온수 전용 모드 전환)
- 월 1회 구동기 LED·분배기 누수 육안 점검

---

**Q24. 온도조절기를 스마트폰 앱으로 제어하려면 어떻게 해야 하나요?**

```
앱 연동 절차
  ① Wi-Fi 또는 Zigbee 지원 온도조절기 선택
  ② 제조사 앱 설치 (App Store / Google Play)
  ③ 앱에서 기기 등록 (QR코드 또는 일련번호 입력)
  ④ 홈 네트워크 Wi-Fi 연결 설정
  ⑤ 원격 제어·스케줄 설정 가능

  주의: 스마트홈 허브(구글홈·애플홈킷·스마트싱스)
        연동 시 온도조절기 통신 프로토콜 사전 확인 필수
```

## D-4. 고장·트러블슈팅 FAQ

**Q25. 전체 방이 갑자기 난방이 안 됩니다. 가장 먼저 뭘 확인해야 하나요?**

```
전체 난방 불가 즉시 확인 순서 (3분 안에 할 수 있는 것)
  1. 분전반 차단기 상태 확인 → 내려가 있으면 올리기
  2. 보일러/HIU 디스플레이 오류 코드 확인
  3. 제어기 전원 LED 확인
  → 위 세 가지 정상이면 Chapter 9.1 진단 흐름도 참조
```

---

**Q26. 특정 방만 계속 뜨겁습니다. 어떻게 하나요?**

A. 해당 방 구동기가 닫히지 않는 것이 원인입니다.

```
즉시 조치:
  1. 온도조절기 설정온도를 현재 온도보다 낮게 설정
  2. 5분 후에도 뜨거우면 해당 구동기 전선 전압 측정
  3. 전압 있는데 닫히지 않으면 구동기 교체
  4. 전압 없는데 제어기 LED 켜져 있으면 결선 확인
```

---

**Q27. 구동기에서 딸깍 소리가 계속 납니다.**

A. 히스테리시스 설정 부족이 가장 흔한 원인입니다.

| 원인 | 확인 방법 | 조치 |
|------|-----------|------|
| 히스테리시스 부족 | 설정 메뉴 확인 | 1~2°C로 조정 |
| 온도조절기 위치 불량 | 외풍·냉기 영향 확인 | 위치 이동 |
| 노이즈 신호 유입 | 배선 분리 여부 확인 | 분리 배선·쉴드 케이블 |

---

**Q28. 보일러는 켜지는데 바닥이 미지근합니다.**

```
바닥 미온 원인 확인 순서
  ① 분배기 해당 루프 유량계 확인 (유량 있는지)
  ② 보일러 공급 온도 확인 (설정값이 너무 낮지 않은지)
  ③ 해당 루프 밸런싱 밸브 확인 (너무 조여져 있지 않은지)
  ④ 루프 배관 길이 확인 (너무 길면 열손실 큼)
  ⑤ 지역난방이면 1차측 공급 온도·압력 관리사무소 확인
```

---

**Q29. 온도조절기 화면이 갑자기 꺼졌습니다.**

| 원인 | 확인 | 조치 |
|------|------|------|
| 배터리 방전 | 배터리 잔량 | 배터리 교체 |
| 전원선 단선 | 전원선 도통 측정 | 전선 재결선 |
| 제어기 전원 공급 불량 | 제어기 출력 전압 측정 | 결선 확인 |
| 온도조절기 고장 | 다른 온도조절기와 교차 테스트 | 교체 |

---

**Q30. 구동기를 교체했는데 작동하는 소리가 없습니다.**

A. 열동형(왁스) 구동기는 코일 가열 시간이 필요해 2~3분 후부터 반응합니다. 교체 직후 30초 안에 동작하지 않는다고 불량이 아닙니다. 5분 이상 기다린 후에도 반응 없으면 전압 측정으로 확인하세요.

---

**Q31. 같은 존인데 어떤 날은 잘 되고 어떤 날은 안 됩니다.**

```
간헐적 오작동 원인

  │ 원인 1: 단자 접촉 불량 (느슨한 나사)               │
  │         → 진동이나 온도 변화 시 접촉 끊어짐         │
  │                                                     │
  │ 원인 2: 노이즈 신호 유입                            │
  │         → 특정 가전 사용 시에만 발생              │
  │                                                     │
  │ 원인 3: 전원 전압 불안정                            │
  │         → 피크 타임에 전압 강하 발생              │
  │                                                     │
  │ 조치: 단자 나사 재조임 → 배선 분리 → 전압 측정 순  │
  └─────────────────────────────────────────────────────┘
```

**Q32. 새로 이사 온 집에 각방제어가 있는데 조작법을 모릅니다.**

A. 온도조절기 본체에 붙은 스티커나 뒷면의 모델명을 확인해 제조사 홈페이지에서 설명서를 다운로드하세요. 모델명을 모르면 사진을 찍어 제조사 A/S 센터에 문의하면 됩니다.

**Q33. 각방제어 시스템 전체를 교체해야 할 시점은 언제인가요?**

## D-5. 비용·효과 FAQ

**Q34. 각방제어 시스템 설치 비용은 얼마나 되나요?**

> 제품 브랜드·시공 업체·지역에 따라 크게 차이납니다. 참고용으로만 활용하세요.

**Q35. 설치 비용 대비 난방비 절감으로 언제 본전이 되나요?**

```
투자 회수 기간 예시 (4존, 설치비 80만원, 월 난방비 15만원 기준)

  난방비 절감률 20% 가정:
  → 월 절감액 = 15만원 × 20% = 3만원
  → 회수 기간 = 80만원 ÷ 3만원 = 약 27개월 (2.2년)

  ※ 단열 성능이 좋고 난방비가 높을수록 회수 기간 단축
```

**Q36. 오래된 아파트에도 설치 효과가 있나요?**

A. 네, 오히려 단열이 부족한 노후 아파트일수록 방별 독립 제어의 효과가 큽니다. 단, 노후 배관 상태와 HIU(지역난방) 성능을 먼저 점검해야 합니다.

## D-6. 지역난방 특화 FAQ

**Q39. 지역난방 아파트에서 각방제어 설치 전 관리사무소에 신고해야 하나요?**

A. 세대 내 전기 장치(구동기·온도조절기·제어기) 교체는 대부분 신고가 불필요합니다. 단, 지역난방 1차측 배관이나 HIU에 영향을 주는 작업은 반드시 관리사무소에 사전 통보하세요.

---

**Q40. 지역난방인데 보일러 연동선이 없어도 되나요?**

A. HIU에 외부 제어 입력 단자가 있으면 연동 가능합니다. 단자가 없으면 연동 없이 구동기·온도조절기만 제어하는 형태로 운용할 수 있습니다. 이 경우 HIU 순환펌프가 항상 가동되므로 에너지 소비가 다소 늘어납니다.

---

**Q41. 지역난방 아파트에서 유독 특정 층만 난방이 안 좋은 이유는?**

```
층별 난방 차이 원인
  ┌─────────────────────────────────────────────────────┐
  │ 원인 1: 단지 공급 계통 압력 배분 불균형             │
  │         (상층부 압력 부족)                          │
  │         → 관리사무소 문의 사항                      │
  │                                                     │
  │ 원인 2: 세대 HIU 스케일 축적                       │
  │         → HIU 세척·점검 의뢰                       │
  │                                                     │
  │ 원인 3: 세대 내 분배기 밸런싱 불량                  │
  │         → 밸런싱 밸브 재조정                       │
  └─────────────────────────────────────────────────────┘
```

---

**Q42. HIU(열교환기) 청소는 직접 해도 되나요?**

A. HIU 외부 필터(스트레이너) 청소는 가이드를 따라 직접 가능합니다. 그러나 HIU 내부 판형 열교환기 스케일 세척, 순환펌프 교체 등은 전문 A/S 기사에게 의뢰하세요.

---

**Q43. 지역난방인데 난방 공급이 시즌제로 운영됩니다. 비수기에 각방제어 장치를 꺼도 되나요?**

A. 제어기·온도조절기는 비수기에도 전원을 유지하는 것이 권장됩니다. 전원을 완전히 끄면 재가동 시 설정이 초기화되거나 에러가 발생할 수 있습니다. 전력 소비가 걱정된다면 대기 모드를 활용하세요.

---

**Q44. 지역난방 공급이 중단될 때(야간 감압 등) 각방제어 장치에 영향이 있나요?**

A. 전기 장치(제어기·온도조절기)에는 영향이 없습니다. 배관 압력이 낮아질 때 2선식 구동기는 스프링으로 닫히고, 3선식은 현재 위치를 유지합니다. 공급 재개 후 정상 동작합니다.

**Q46. 어린 자녀가 있는 집에서 주의할 점은?**

A. 온도조절기 설정 잠금(아동 잠금) 기능을 활성화하세요. 대부분의 현대 온도조절기는 버튼 장기 누름으로 잠금 기능을 제공합니다. 또한 분배기 구동기 전선이 아이의 손에 닿지 않도록 케이블 타이로 정리하세요.

---

**Q47. 분배기에서 물이 샐 때 응급 조치 방법은?**

```
누수 발생 시 응급 조치
  ① 분배기 공급 밸브 즉시 잠금 (우측으로 돌리면 잠김)
  ② 제어기 전원 차단 (물이 전기 장치에 닿지 않도록)
  ③ 바닥의 물 제거 (감전 방지)
  ④ 누수 원인 확인:
     • 구동기 어댑터 풀림 → 재체결
     • 배관 연결부 누수 → 전문가 의뢰
  ⑤ 수리 완료 전 보일러 가동 금지
```
