# Chapter 4. 지역난방 시스템 연동

---

## 한눈에 보기 (핵심 5줄)

1. 지역난방은 외부 열공급소에서 뜨거운 물을 아파트 단지 전체에 공급하는 방식이다.
2. 세대 내 열교환기가 지역난방 열을 받아 세대 자체 난방수로 변환한다.
3. 각방제어 시스템은 열교환기 이후 세대 난방수 배관 단계에서 연동된다.
4. 지역난방 세대는 보일러가 없고 열교환기·유량조절밸브가 핵심 장치다.
5. 압력·유량 조건이 개별 보일러와 달라 각방제어 설치 전 반드시 현장 조건을 확인해야 한다.

---

## 왜 중요한가? — 생활 속 예시

**예시 1 — "난방을 켰는데 바닥이 미지근하다"**

지역난방 아파트에서는 열공급소의 공급 압력·온도가 시간대별로 변합니다.
각방제어 밸브를 모두 열었는데도 유량이 부족하면 바닥이 충분히 따뜻해지지 않습니다.
지역난방 구조를 알아야 "밸브 문제"인지 "공급 측 문제"인지 구분할 수 있습니다.

**예시 2 — "리모델링 후 특정 방만 뜨겁고 나머지는 안 따뜻하다"**

지역난방 연동 각방제어에서 분배기 유량 밸런싱이 안 되면 유량이 많이 흐르는 루프에만 열이 집중됩니다.
개별 보일러 시스템과 달리 지역난방은 유량 밸런싱이 특히 중요합니다.

## 4.1 지역난방 기본 구조와 난방수 흐름

### 지역난방 전체 구조

```
┌─────────────────────────────────────────────────────────────┐
│                      지역난방 공급 계통                       │
│                                                             │
│  [열공급소/열병합발전소]                                       │
│       │  공급수 (고온 고압, 70~120°C)                        │
│       ▼                                                     │
│  [단지 열교환기실 / 기계실]                                    │
│       │  단지 배관                                           │
│       ▼                                                     │
│  [각 동 열공급 배관]                                          │
│       │                                                     │
│       ▼                                                     │
│  [세대 열교환기(HIU: Heat Interface Unit)]  ◀─ 이 지점부터    │
│       │  세대 난방수 (2차측, 40~60°C)          세대 관할      │
│       ▼                                                     │
│  [세대 분배기] → [각방 루프 배관] → [각방 바닥 난방]           │
└─────────────────────────────────────────────────────────────┘
```

### 온도 단계별 비교

| 구간 | 명칭 | 온도 범위 | 관리 주체 |
|------|------|-----------|-----------|
| 열공급소 → 단지 | 1차측 공급수 | 70~120°C | 열공급 회사 |
| 단지 기계실 → 세대 HIU | 1차측 (단지) | 60~90°C | 관리사무소 |
| HIU 이후 세대 배관 | 2차측 (세대) | 40~60°C | 세대 거주자 |
| 각방 루프 바닥 | 세대 난방 루프 | 35~50°C | 세대 거주자 |

> **초보자 TIP**: 각방제어 시스템은 **2차측(세대 난방수)** 구간에만 관여합니다.
> 1차측(열공급소~HIU)은 열공급 회사와 관리사무소 영역이므로 임의로 건드리지 마세요.

## 4.2 세대 단위 각방제어 연동 포인트

### 세대 내 각방제어 연결 구조

```
[HIU(열교환기)]
      │
      │ 세대 난방 공급수 (2차측)
      ▼
[세대 분배기(Manifold)]
      │
      ├─ [구동기 1] ── 거실 루프 ──────────────┐
      ├─ [구동기 2] ── 안방 루프 ──────────────┤
      ├─ [구동기 3] ── 아이방 루프 ────────────┤  (바닥 난방 후)
      └─ [구동기 4] ── 서재 루프 ──────────────┘
                                               │
                                         [분배기 환수]
                                               │
                                          [HIU 환수]
                                               │
                                        [단지 환수 배관]
```

### 연동 포인트별 역할

| 연동 포인트 | 위치 | 역할 | 주의사항 |
|-------------|------|------|----------|
| HIU 공급측 | 세대 현관/보일러실 | 단지 열 → 세대 열 변환 | 임의 조작 금지 |
| HIU 유량조절밸브 | HIU 내부 또는 근처 | 세대 전체 유량 조절 | 난방 용량 설계값 유지 |
| 세대 분배기 | 보일러실 또는 다용도실 | 각방 루프 유량 분배 | 구동기 장착 위치 |
| 각방 구동기 | 분배기 밸브 헤드 | 방별 밸브 개폐 | 2선식/3선식 방식 확인 |
| 제어기 | 분배기 근처 벽면 | 온도조절기 신호 처리 | HIU 연동 신호 여부 확인 |

## 4.3 열교환기 / 분배기 / 밸브 인터페이스

### HIU(열교환기) 구조

```
              [1차측 공급 IN]  (단지 고온수)
                    │
                    ▼
         ┌──────────────────┐
         │  판형 열교환기     │  ← 1차측 ↔ 2차측 열전달
         │  (격리: 1차/2차   │     (물은 섞이지 않음)
         │   물은 분리됨)     │
         └──────────────────┘
                    │
              [1차측 환수 OUT]

         ┌──────────────────┐
         │  2차측 (세대)     │
         │  순환펌프·팽창탱크 │
         │  자동 유량조절밸브 │
         └──────────────────┘
                    │
              [세대 공급 OUT] → 분배기로
              [세대 환수 IN]  ← 분배기에서
```

### 분배기(Manifold) 구조

```
공급 헤더 (상단)
┌──┬──┬──┬──┬──┬──┐
│  │  │  │  │  │  │  ← 구동기 장착 위치
└──┴──┴──┴──┴──┴──┘
  루프1 루프2 루프3 루프4 루프5 루프6
  (거실)(안방)(아이방)(서재)(주방)(욕실)

환수 헤더 (하단)
┌──┬──┬──┬──┬──┬──┐
│  │  │  │  │  │  │  ← 유량계/밸런싱 밸브 위치
└──┴──┴──┴──┴──┴──┘
```

### 분배기 구성 요소

| 부품 | 위치 | 역할 |
|------|------|------|
| 공급 헤더 | 상단 | 세대 공급수를 각 루프로 분배 |
| 환수 헤더 | 하단 | 각 루프 환수를 모아 HIU로 복귀 |
| 구동기 (밸브 헤드) | 공급 헤더 각 루프 | 각방 유량 ON/OFF |
| 유량계 | 환수 헤더 각 루프 | 각 루프 유량 시각 확인 |
| 밸런싱 밸브 | 환수 헤더 | 루프별 유량 균형 조정 |
| 에어벤트 | 헤더 상단 끝 | 배관 내 공기 자동 배출 |
| 드레인 밸브 | 헤더 하단 끝 | 시스템 물 빼기용 |

## 4.4 연동 시 빈번한 문제와 예방 설계

### 문제 유형 매트릭스

```
문제 발생 위치
       │
       ├─ 1차측 (열공급 계통)
       │       ├─ 공급 압력 부족 → 세대 전체 난방 불량
       │       ├─ 공급 온도 저하 → 바닥 미온
       │       └─ 단지 배관 공기 유입 → 소음·유량 불안정
       │
       ├─ HIU (열교환기)
       │       ├─ 스케일(석회질) 축적 → 열교환 효율 저하
       │       ├─ 순환펌프 고장 → 세대 전체 순환 불가
       │       └─ 유량조절밸브 고착 → 유량 과다/부족
       │
       └─ 2차측 (각방제어 구간)
               ├─ 구동기 불량 → 특정 방 개폐 불가
               ├─ 밸런싱 불량 → 특정 루프 과잉 유량
               ├─ 공기 유입 → 배관 소음, 열 전달 불량
               └─ 제어기·온도조절기 불량 → 오제어
```

### 빈번한 문제와 예방책

| # | 문제 | 원인 | 예방 설계 |
|---|------|------|-----------|
| 1 | 특정 방만 뜨거움 | 루프 밸런싱 불량 (짧은 루프에 유량 집중) | 설치 시 루프별 밸런싱 밸브 조정 |
| 2 | 전체 방이 미온 | HIU 열교환 효율 저하 (스케일) | 연 1회 HIU 필터 청소, 스케일 점검 |
| 3 | 배관 소음 (콸콸) | 배관 내 공기 혼입 | 에어벤트 설치, 초기 퍼징(공기 제거) |
| 4 | 난방 응답 느림 | 긴 배관·낮은 유량 | HIU 순환펌프 속도 설정 확인 |
| 5 | 에너지 과다 소비 | 미사용 방 밸브 열림 지속 | 온도조절기 스케줄 설정 활용 |
| 6 | 압력 경보 | 팽창탱크 용량 부족 또는 고장 | 팽창탱크 압력 정기 점검 |

### 지역난방 vs 개별 보일러 각방제어 차이

```
┌─────────────────────────────────────────────────────┐
│           각방제어 설치 전 확인 비교표                  │
├──────────────────┬──────────────┬───────────────────┤
│ 항목             │ 개별 보일러   │ 지역난방           │
├──────────────────┼──────────────┼───────────────────┤
│ 열원 장치        │ 세대 내 보일러│ HIU (열교환기)     │
│ 공급 온도 설정   │ 보일러에서 직접│ 열공급사 + HIU 설정│
│ 순환 펌프        │ 보일러 내장   │ HIU 내 별도 펌프   │
│ 보일러 연동 신호 │ 제어기 → 보일러│ 제어기 → HIU 또는 │
│                  │              │ 유량밸브 직접 제어 │
│ 압력 관리        │ 세대 단독     │ 단지 전체 계통     │
│ 유량 밸런싱 중요도│ 중간          │ 높음               │
│ 전문가 개입 필요 │ 낮음          │ 높음 (초기 설정)   │
└──────────────────┴──────────────┴───────────────────┘
```

### 현장 조건 확인표 (설치 전 필수)

```
✅ 지역난방 각방제어 설치 전 현장 조건 체크리스트

[ ] 1. 세대 HIU 모델명 및 제조사 확인
[ ] 2. HIU 2차측 공급 온도 확인 (설계값 vs 실측값)
[ ] 3. 세대 2차측 공급 압력 확인 (정상 범위: 1.0~2.0 bar)
[ ] 4. 분배기 루프 수 및 각 루프 길이 확인
[ ] 5. 기존 구동기 방식 확인 (2선식 / 3선식)
[ ] 6. HIU → 제어기 연동 방식 확인 (가능 여부)
[ ] 7. 분배기 위치 및 제어기 설치 공간 확인
[ ] 8. 분전반에서 제어기까지 전원 공급 경로 확인
```

## 문제 발생 시 즉시 대응

| 증상 | 확인 순서 |
|------|-----------|
| 세대 전체 난방 불량 | ① 관리사무소에 1차측 공급 정상 여부 확인 → ② HIU 순환펌프 동작 확인 → ③ 제어기 전원 확인 |
| 특정 방만 난방 불량 | ① 해당 구동기 동작 확인 → ② 분배기 유량계로 해당 루프 유량 확인 → ③ 밸런싱 밸브 조정 |
| 배관 소음 | ① 에어벤트 열어 공기 배출 → ② 분배기 드레인으로 공기 제거 → ③ 순환펌프 속도 조정 |
| 압력 경보 | ① 팽창탱크 압력 측정 → ② 물 보충 또는 전문가 점검 의뢰 |

> **전문가 의뢰 기준**: HIU 내부 부품 교체, 1차측 배관 작업, 단지 계통 압력 조정은 반드시 전문가(열공급 회사 또는 공인 설비 업체)에게 의뢰하세요.

## 초보자 Q&A

**Q1. 지역난방 아파트에서 각방제어 리모델링이 가능한가요?**
A. 네, 가능합니다. 다만 관리사무소에 사전 신고·승인이 필요한 경우가 많습니다. 또한 HIU 연동 방식이 아파트 단지마다 달라 사전에 관리사무소 및 HIU 제조사에 확인하세요.

**Q2. 지역난방인데 세대 내 보일러를 추가 설치할 수 있나요?**
A. 지역난방 공급 약관상 세대 내 별도 보일러 설치가 금지된 경우가 많습니다. 반드시 열공급 회사와 관리사무소에 문의 후 진행하세요.

**Q3. HIU와 각방제어 제어기 연동이 꼭 필요한가요?**
A. 연동하지 않아도 각방제어는 동작하지만, 모든 존이 난방 불필요 상태일 때 HIU 순환펌프가 계속 가동돼 에너지 낭비가 생깁니다. 제어기 → HIU 절약모드 신호 연동을 권장합니다.

**Q4. 겨울에 지역난방 공급이 중단되면 각방제어 장치가 고장나나요?**
A. 각방제어 전기 장치(제어기·온도조절기)는 영향 없습니다. 단, 장기간 시스템을 방치하면 배관 내 공기 유입이 생길 수 있으므로 재가동 시 퍼징을 권장합니다.

**Q5. 루프 길이가 다른데 밸런싱은 어떻게 하나요?**
A. 환수 헤더의 유량계를 보면서 밸런싱 밸브를 조정합니다. 짧은 루프(유량 많음)의 밸런싱 밸브를 조여 유량을 줄이고, 긴 루프는 열어 균등하게 맞춥니다. 정밀 밸런싱은 설비 전문가에게 의뢰를 권장합니다.
