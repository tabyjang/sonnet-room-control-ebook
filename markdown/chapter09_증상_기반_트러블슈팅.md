# Chapter 9. 증상 기반 트러블슈팅

---

## 한눈에 보기 (핵심 5줄)

1. 고장 진단은 "전체 → 부분" 순서로 범위를 좁혀가는 것이 가장 빠르다.
2. 증상만 보면 원인이 여러 개일 수 있으므로 반드시 **순서대로** 점검한다.
3. 전기 측정(멀티미터)을 두려워하지 말자. 전압·저항 두 가지만 알면 대부분 진단 가능하다.
4. 1차 진단으로 해결 안 되면 무리하게 분해하지 말고 전문가에게 의뢰한다.
5. 같은 증상도 원인이 다를 수 있으므로 추측이 아닌 측정으로 확인한다.

---

## 왜 중요한가? — 생활 속 예시

**예시 1 — "구동기를 교체했는데도 여전히 그 방이 안 따뜻하다"**

증상만 보고 구동기 불량이라 판단해 교체했지만, 실제 원인은 온도조절기 신호선 단선이었습니다.
순서대로 진단했다면 비용과 시간을 절약할 수 있었습니다.

**예시 2 — "전체 방이 갑자기 난방이 안 된다"**

제어기 불량이라 생각했지만, 실제로는 분전반 차단기가 내려가 있었습니다.
가장 단순한 원인부터 확인하는 습관이 중요합니다.


## 9.1 난방이 전혀 되지 않음

### 증상 정의

- 모든 방(전 존)의 바닥이 차갑다
- 온도조절기 설정온도를 올려도 변화 없다
- 보일러 버너가 점화되지 않는다

### 진단 흐름도

```
[증상] 전체 난방 불가
          │
          ▼
  ① 보일러/HIU 전원·오류 확인
     └─ 보일러 오류 코드 있음?
          ├─ YES → 보일러 매뉴얼 오류코드 처리
          └─ NO  → ②로
          │
          ▼
  ② 분전반 제어기 차단기 확인
     └─ 차단기 내려가 있음?
          ├─ YES → 차단기 올림 → 재기동 확인
          └─ NO  → ③으로
          │
          ▼
  ③ 제어기 전원 LED 확인
     └─ LED 꺼져 있음?
          ├─ YES → 제어기 전원 결선 확인
          │         L/N 단자 전압 측정 (정상: AC 220V)
          └─ NO  → ④로
          │
          ▼
  ④ 온도조절기 전체 확인
     └─ 모든 온도조절기 화면 꺼짐?
          ├─ YES → 제어기→온도조절기 전원선 확인
          └─ NO  → ⑤로
          │
          ▼
  ⑤ 보일러 연동선 확인
     └─ BLR 단자 신호 출력 여부
          ├─ 신호 없음 → 온도조절기 신호선 도통 확인
          └─ 신호 있음 → 보일러 측 입력 단자 확인
          │
          ▼
  ⑥ 원인 미파악 → 전문가 의뢰
```

### 전체 난방 불가 원인·조치 매트릭스

| 원인 | 확인 방법 | 조치 |
|------|-----------|------|
| 차단기 트립 | 분전반 시각 확인 | 차단기 ON, 과부하 원인 제거 |
| 보일러 오류 | 보일러 디스플레이 오류 코드 | 보일러 매뉴얼 따라 리셋·수리 |
| 제어기 전원 불량 | L/N 단자 전압 측정 | 결선 재확인, 제어기 교체 |
| BLR 연동선 단선 | BLR 단자 도통 측정 | 연동선 재결선 |
| 보일러 입력 단자 불량 | 보일러 단자 전압 측정 | 보일러 수리 의뢰 |
| 정전 | 세대 전체 전원 확인 | 전기 복구 대기 |

## 9.2 특정 방만 과열 또는 저온

### 증상 정의

- 한 방만 설정온도 이상으로 과열되거나
- 한 방만 난방이 전혀 되지 않음
- 다른 방은 정상

### 진단 흐름도

```
[증상] 특정 방(Zone N) 이상
          │
     ┌────┴────┐
     │         │
  [과열]     [저온/불가]
     │         │
     ▼         ▼
  밸브가      밸브가
  안 닫힘     안 열림
     │         │
     ▼         ▼
① 온도조절기  ① 온도조절기
  신호 확인     신호 확인
  (설정온도     (설정온도
   이상인데      미만인데
   계속 열림)    신호 없음)
     │         │
     ▼         ▼
② 제어기     ② 제어기
  해당 존      해당 존
  LED 확인     LED 확인
     │         │
     ▼         ▼
③ 구동기     ③ 구동기
  전선 전압    전선 전압
  측정         측정
  (전압 있는데  (전압 없으면
   닫히지 않음) → 제어기 불량)
     │         │
     ▼         ▼
④ 구동기     ④ 구동기
  수동 테스트   수동 테스트
  (수동으로     (수동으로
   닫히면       열리면
   구동기 불량) 구동기 정상
               → 결선 재확인)
```

### 특정 방 과열 원인·조치표

| 원인 | 확인 방법 | 조치 |
|------|-----------|------|
| 온도조절기 접점 고착 (항상 ON) | 온도조절기 분리 후 단자 도통 측정 | 온도조절기 교체 |
| 구동기 코일 고착 (밸브 닫힘 불가) | 수동으로 밸브 닫기 시도 | 구동기 교체 |
| 제어기 해당 존 출력 고착 | 다른 온도조절기 단자에 연결 테스트 | 제어기 교체 |
| 2선식 L선 상시 통전 | 해당 구동기 단자 전압 측정 (항상 220V) | 결선 재확인 |
| 밸브 스템 이물질 낀 경우 | 구동기 제거 후 밸브 스템 확인 | 밸브 청소 또는 교체 |

### 특정 방 저온 원인·조치표

| 원인 | 확인 방법 | 조치 |
|------|-----------|------|
| 온도조절기 신호선 단선 | 신호선 양단 도통 측정 | 신호선 재결선 또는 교체 |
| 구동기 코일 단선 | 구동기 코일 저항 측정 (∞ → 단선) | 구동기 교체 |
| 제어기 해당 존 출력 불량 | 출력 단자 전압 측정 (0V → 불량) | 제어기 교체 |
| 밸브 스프링 파손 (2선식) | 구동기 제거 후 스프링 상태 확인 | 구동기 교체 |
| 루프 배관 막힘 | 분배기 유량계로 해당 루프 유량 확인 | 배관 세척 의뢰 |
| 온도조절기 센서 오차 | 기준 온도계와 비교 측정 | 보정값(오프셋) 조정 |

## 9.3 구동기 소음·오동작·반복 개폐

### 증상 정의

- 분배기 근처에서 딸깍·윙윙·진동 소리가 반복된다
- 구동기가 짧은 시간 간격으로 열렸다 닫혔다를 반복한다
- 특정 구동기가 작동 중 멈추거나 비정상 위치에 걸린다

### 소음·오동작 원인 진단표

```
소음 유형에 따른 원인 분류
┌──────────────────┬──────────────────────────────────────┐
│ 소음 유형        │ 추정 원인                            │
├──────────────────┼──────────────────────────────────────┤
│ 딸깍 반복        │ 히스테리시스 미설정 → 잦은 ON/OFF    │
│ (2~5분 간격)     │ 온도조절기 설치 위치 불량 (냉기 유입) │
├──────────────────┼──────────────────────────────────────┤
│ 윙윙·진동음      │ 구동기 모터 베어링 마모              │
│ (3선식 모터형)   │ 밸브 스템 이물질                     │
├──────────────────┼──────────────────────────────────────┤
│ 물 흐르는 소리   │ 밸브 완전 닫힘 불량 (미세 개방)      │
│ (콸콸/쉬이)      │ 배관 내 공기 혼입                    │
├──────────────────┼──────────────────────────────────────┤
│ 소리 없이        │ 제어기 신호 불안정                   │
│ 반복 동작        │ 온도조절기 신호선 노이즈             │
└──────────────────┴──────────────────────────────────────┘
```

### 반복 개폐 진단 흐름도

```
[증상] 구동기 반복 개폐
          │
          ▼
  ① 히스테리시스 설정 확인
     └─ 설정값이 0.5°C 미만?
          ├─ YES → 히스테리시스 1°C 이상으로 조정
          └─ NO  → ②로
          │
          ▼
  ② 온도조절기 위치 확인
     └─ 에어컨·창문·냉기 영향 있음?
          ├─ YES → 온도조절기 이동 설치
          └─ NO  → ③으로
          │
          ▼
  ③ 신호선 노이즈 확인
     └─ 신호선이 전원선과 같은 관 통과?
          ├─ YES → 분리 배선 또는 쉴드 케이블 교체
          └─ NO  → ④로
          │
          ▼
  ④ 구동기 기계적 상태 확인
     └─ 밸브 스템 이물질 또는 마모
          ├─ YES → 구동기 교체
          └─ NO  → 제조사 A/S 의뢰
```

### 조치 방법 요약표

| 증상 | 원인 | 조치 |
|------|------|------|
| 딸깍 반복 | 히스테리시스 부족 | 온도조절기 히스테리시스 1~2°C로 설정 |
| 딸깍 반복 | 센서 위치 불량 | 냉기·열원 없는 위치로 이동 |
| 윙윙음 | 모터 마모 | 구동기 교체 |
| 물 소리 | 밸브 완전 닫힘 불가 | 구동기 교체 (스프링 약화) |
| 신호 없이 동작 | 노이즈 유입 | 분리 배선, 쉴드 케이블 적용 |

## 9.4 온도 표시 불안정·제어 불량

### 증상 정의

- 온도조절기 숫자가 계속 변하거나 깜빡인다
- 설정온도 도달 후에도 난방이 계속된다
- 실제 방 온도와 표시 온도가 3°C 이상 차이난다
- 온도조절기가 갑자기 재부팅된다

### 온도 표시 불안정 진단표

```
┌──────────────────────────────────────────────────────┐
│  온도 표시 이상 증상별 원인 매트릭스                  │
├────────────────┬─────────────────────────────────────┤
│ 증상           │ 원인 및 확인 방법                   │
├────────────────┼─────────────────────────────────────┤
│ 숫자 계속 변동 │ ① 신호선 노이즈 (전원선 근접)       │
│                │ ② 온도조절기 설치 위치 불량         │
│                │    (에어컨·가전 영향)               │
│                │ ③ 온도조절기 내부 센서 불량         │
├────────────────┼─────────────────────────────────────┤
│ 실제 온도와    │ ① 온도조절기 위치 불량              │
│ 큰 차이        │    (외벽·직사광선·발열체 근처)      │
│                │ ② 센서 오차 (보정 필요)             │
│                │ ③ 배선 홀 밀봉 미처리 (외풍 유입)  │
├────────────────┼─────────────────────────────────────┤
│ 설정온도 도달  │ ① 히스테리시스 설정 오류            │
│ 후에도 난방    │ ② 온도조절기 접점 고착              │
│ 계속           │ ③ 제어기 해당 존 출력 고착          │
├────────────────┼─────────────────────────────────────┤
│ 갑자기 재부팅  │ ① 전원 전압 불안정                  │
│                │ ② 온도조절기 내부 회로 불량         │
│                │ ③ 전원선 접촉 불량 (느슨한 단자)   │
└────────────────┴─────────────────────────────────────┘
```

### 제어 불량 단계별 점검 절차

```
[증상] 온도 제어 불량
          │
          ▼
  ① 기준 온도계로 실제 온도 측정
     └─ 기준 온도계와 온도조절기 표시 차이?
          ├─ 차이 ≤ 3°C → 보정값 조정
          └─ 차이 > 3°C → 위치 변경 검토 → ②로
          │
          ▼
  ② 온도조절기 설치 위치 재점검
     └─ 냉기·열원·직사광선 영향?
          ├─ YES → 위치 변경 후 재측정
          └─ NO  → ③으로
          │
          ▼
  ③ 배선 홀 밀봉 상태 확인
     └─ 홀을 통해 외풍 유입?
          ├─ YES → 실리콘으로 밀봉
          └─ NO  → ④로
          │
          ▼
  ④ 온도조절기 센서 불량 의심
     └─ 동일 모델로 교체 후 재측정
          ├─ 정상 → 기존 온도조절기 교체
          └─ 동일 → 제어기 채널 불량 확인
```

| 2 | 차단기·전원 확인 생략 | 가장 먼저 차단기·전원 상태부터 확인 |
| 3 | 멀티미터 없이 눈으로만 판단 | 전압·저항은 반드시 멀티미터로 측정 |
| 4 | 온도 표시 이상 시 제어기 교체 | 온도조절기 위치·보정부터 확인 후 교체 |
| 5 | 여러 원인 동시에 수정 | 한 번에 하나씩 수정 후 재시험 (원인 특정) |

## 초보자 Q&A

**Q1. 멀티미터가 없어도 진단할 수 있나요?**
A. 일부 증상(차단기·누수·LED 확인)은 육안으로 가능하지만, 단선·단락·전압 문제는 멀티미터 없이 정확히 진단하기 어렵습니다. 5~10만원대 기본 멀티미터 한 개는 꼭 갖춰두세요.

**Q2. 히스테리시스는 어디서 설정하나요?**
A. 온도조절기 설정 메뉴에서 "차동폭", "히스테리시스", "편차" 등의 항목으로 제공됩니다. 제조사별로 메뉴 이름이 다르니 제품 설명서를 확인하세요. 일반적으로 0.5~2°C 범위로 설정합니다.

**Q3. 구동기를 수동으로 열고 닫는 방법이 있나요?**
A. 열동형 구동기는 대부분 수동 잠금/개방 버튼(수동 핸들)이 없습니다. 단, 일부 제품은 어댑터에 수동 개폐 기능이 있습니다. 모터형 구동기는 수동 조작 레버가 있는 제품도 있습니다. 제품 매뉴얼을 확인하세요.

**Q4. 겨울에 장기 공실 후 돌아왔을 때 전체 난방이 안 됩니다. 왜일까요?**
A. 장기 공실 중 정전·동파·배관 공기 유입이 생겼을 가능성이 높습니다. 9.1 진단 흐름도 순서대로 점검하되, 배관 에어 제거와 분배기 에어벤트 확인을 먼저 진행하세요.

**Q5. 진단해도 원인을 모르겠을 때는 어떻게 하나요?**
A. 이 전자책의 진단 흐름도를 끝까지 따라갔는데도 원인을 찾지 못했다면 전문가에게 의뢰하세요. 시스템 모델명·증상·이미 점검한 항목을 메모해두면 A/S 처리 시간을 단축할 수 있습니다.
