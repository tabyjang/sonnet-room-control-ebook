## A-3. 설치 단계별 체크리스트

### STEP 0. 안전 확인

```
[ ] 분전반 차단기 OFF 확인
[ ] 검전기로 제어기 설치 위치 무전압 확인
[ ] 분배기 공급 밸브 잠금 (구동기 교체 시)
[ ] 작업 구역 정리 완료
[ ] 절연 장갑·절연 드라이버 착용
```

### STEP 1. 제어기 설치

```
[ ] 분배기에서 1m 이내 위치 선정
[ ] 바닥에서 1.2~1.5m 높이 확인
[ ] 브래킷 또는 DIN 레일 수평 고정
[ ] 상하 5cm 환기 공간 확보
[ ] 전원선 경로 확인 (미결선 상태)
```

### STEP 2. 구동기 장착

```
[ ] 분배기 밸브 나사 규격 재확인
[ ] 어댑터 체결 완료 (토크 적정)
[ ] 구동기 수직 상향 또는 허용 방향 장착
[ ] 손으로 흔들어도 흔들리지 않음 확인
[ ] 장착 후 물 채우기 → 5분 이상 누수 관찰
```

### STEP 3. 온도조절기 베이스 설치

```
[ ] 각 방 내벽, 바닥에서 1.2~1.5m 높이 위치 확정
[ ] 직사광선·에어컨·가전 발열 영향 없는 위치 확인
[ ] 수평계로 수평 맞추고 고정
[ ] 배선 통과 홀 위치 확인
```

### STEP 4. 배선 작업

```
[ ] 전선관·몰딩 경로 계획 및 고정
[ ] 구동기 전선 포설 완료
[ ] 온도조절기 신호선 포설 완료
[ ] 보일러/HIU 연동선 포설 완료
[ ] 전력선과 신호선 분리 배선 확인 (10cm 이상 이격)
[ ] 전선 마킹 태그 부착 완료
[ ] 케이블 타이로 배선 정리 완료
```

### STEP 5. 결선 작업

```
[ ] 제어기 L/N/GND 전원선 결선
[ ] 제어기 BLR 보일러 연동선 결선
[ ] 제어기 Z1~ZN 온도조절기 신호선 결선
[ ] 제어기 A1~AN 구동기 출력선 결선
[ ] 결선도와 실제 배선 대조 완료
[ ] 남은 전선 끝 절연 처리
[ ] 커버 및 케이스 닫기 전 내부 점검
```

### STEP 6. 결선 점검

```
[ ] 차단기 OFF 상태 재확인
[ ] L-N 간 저항 측정 → 무한대(∞) 확인
[ ] 구동기 전선 도통 측정 → 정상 코일 저항 확인
[ ] 신호선 도통 측정 → 수 Ω 이하 확인
[ ] 접지선 연결 확인
[ ] 모든 단자 나사 조임 재확인
```

## A-4. 시운전 체크리스트

### 전원 인가 전 최종 확인

```
[ ] 배관 물 채우기 완료 (누수 없음)

[ ] 보일러/HIU 정상 가동 확인
```
