# 각방제어 온도조절 시스템 통합 전자책 목차

## 1. 문서 개요
- 문서명: 각방제어 온도조절 시스템 통합 매뉴얼
- 목적: 입문 학습, 현장 설치/시운전, 유지보수 트러블슈팅을 한 권으로 제공
- 대상 독자: 입문 기술자, 시공 기사, A/S 기사, 설비 관리자, 기술 영업
- 적용 범위: 2선식, 3선식, 제어기, 구동기, 온도조절기, 지역난방 연동 시스템

## 2. 전자책 구성 원칙
- 학습 흐름: 기초 개념 -> 설계/배선 -> 설치/시운전 -> 고장진단/유지관리
- 실무 흐름: 준비 -> 시공 -> 검증 -> 인수인계 -> 사후관리
- 공통 템플릿: 개념 요약, 현장 절차, 체크리스트, 주의사항, Q&A

## 3. 파트 구성
- Part A. 입문-전문가 단계형 학습
- Part B. 실무자용 설치-시운전 매뉴얼
- Part C. 문제해결 중심 트러블슈팅/케이스북
- 부록. 양식, 점검표, 용어사전

## 4. 상세 목차

### Part A. 입문-전문가 단계형 학습

#### Chapter 1. 각방제어 시스템 개요
- 1.1 각방제어의 필요성과 에너지 절감 효과
- 1.2 시스템 기본 구성(열원, 배관, 말단 제어)
- 1.3 존(Zone) 제어 개념과 운전 시나리오
- 1.4 중앙제어와 개별제어의 역할 분담

#### Chapter 2. 2선식과 3선식 비교 이해
- 2.1 2선식 구조와 동작 원리
- 2.2 3선식 구조와 동작 원리
- 2.3 배선 복잡도, 안정성, 확장성 비교
- 2.4 현장 적용 기준(신축/리모델링/교체)

#### Chapter 3. 핵심 장치 동작 원리
- 3.1 제어기(Controller) 기능과 제어 로직
- 3.2 구동기(Actuator) 종류와 밸브 연동 방식
- 3.3 온도조절기(Thermostat) 센싱 및 제어 특성
- 3.4 센서 오차, 위치 선정, 보정 기준

#### Chapter 4. 지역난방 시스템 연동
- 4.1 지역난방 기본 구조와 난방수 흐름
- 4.2 세대 단위 각방제어 연동 포인트
- 4.3 열교환기/분배기/밸브 인터페이스
- 4.4 연동 시 빈번한 문제와 예방 설계

### Part B. 실무자용 설치-시운전 매뉴얼

#### Chapter 5. 설계 및 시공 전 준비

#### Chapter 6. 배선/결선 표준

- 6.2 3선식 표준 결선도

#### Chapter 7. 설치 절차 SOP
- 7.1 공정 순서(기기 설치 -> 배선 -> 결선 점검)
