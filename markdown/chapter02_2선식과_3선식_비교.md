# Chapter 2. 2선식과 3선식 비교 이해

---

## 한눈에 보기 (핵심 5줄)

1. 각방제어 구동기(전동밸브)는 **전선 2가닥(2선식)**과 **전선 3가닥(3선식)** 두 방식으로 나뉜다.
2. 2선식은 전선이 적어 시공이 단순하지만, 밸브가 완전히 열렸는지 닫혔는지 피드백이 없다.
3. 3선식은 전선이 한 가닥 더 있어 밸브 동작을 더 정확히 제어하고 상태 확인이 가능하다.
4. 신축 공사는 3선식, 기존 배선 재활용 리모델링은 2선식이 유리한 경우가 많다.
5. 두 방식을 혼용하면 오작동이 생기므로, 현장 조건에 맞게 한 가지로 통일한다.

---

## 왜 중요한가? — 생활 속 예시

**예시 1 — "전선을 잘못 연결해서 방이 항상 뜨겁다"**

2선식 구동기를 3선식 제어기에 연결하거나, 반대의 경우 밸브가 제대로 개폐되지 않습니다.
한 방이 계속 가열되거나 전혀 난방이 안 되는 증상이 생깁니다.
방식을 정확히 구분해야 이런 오결선을 막을 수 있습니다.

**예시 2 — "리모델링 중 기존 전선을 재활용하고 싶다"**

기존 건물에 이미 깔린 배선이 2가닥이라면 3선식 구동기를 쓸 수 없습니다.
배선 방식을 먼저 파악해야 구동기·제어기 선택이 가능합니다.

---

## 2.1 2선식 구조와 동작 원리

### 구조

2선식은 구동기와 제어기 사이를 **전선 2가닥**으로 연결합니다.

```
제어기
 ├─ L선(전원) ──────────────┐
 │                          ▼
 │                    [구동기 코일]
 │                          │
 └─ N선(중성) ──────────────┘
```

| 선 | 역할 |
|----|------|
| L선 (Live) | 전원 공급 |
| N선 (Neutral) | 전원 복귀(중성선) |

### 동작 원리

- 제어기가 L선에 전압을 인가하면 구동기 코일이 통전 → 밸브 **열림**
- 제어기가 전압을 차단하면 스프링 복귀 → 밸브 **닫힘**
- 즉, **전압 인가 = 열림 / 전압 차단 = 닫힘** 구조 (상시 닫힘, N.C. 방식이 일반적)

### 특징 요약

| 항목 | 내용 |
|------|------|
| 전선 수 | 2가닥 |
| 피드백 | 없음 (개폐 상태 확인 불가) |
| 제어 방식 | ON/OFF |
| 복귀 방식 | 스프링 자동 복귀 |
| 동작 시간 | 약 2~3분 (열림/닫힘) |

> **초보자 TIP**: 2선식 구동기는 전원을 끊으면 스프링 힘으로 자동으로 닫힙니다.
> 정전 시 모든 밸브가 닫히므로 난방이 중단됩니다. 이는 안전 설계입니다.

---

## 2.2 3선식 구조와 동작 원리

### 구조

3선식은 구동기와 제어기 사이를 **전선 3가닥**으로 연결합니다.

```
제어기
 ├─ 공통선(COM) ─────────────┐
 ├─ 열림선(OPEN) ─────────── [모터 권선 A]
 └─ 닫힘선(CLOSE) ────────── [모터 권선 B]
                              ↕
                         [밸브 모터]
```

| 선 | 색상(일반) | 역할 |
|----|-----------|------|
| COM (공통) | 흑색 또는 갈색 | 공통 기준선 |
| OPEN (열림) | 백색 또는 주황 | 밸브 열기 신호 |
| CLOSE (닫힘) | 적색 또는 청색 | 밸브 닫기 신호 |

> **주의**: 색상은 제조사마다 다릅니다. 반드시 제품 매뉴얼의 단자 표기를 기준으로 결선하세요.

### 동작 원리

- COM + OPEN에 전압 → 모터가 열림 방향 회전 → 밸브 **열림**
- COM + CLOSE에 전압 → 모터가 닫힘 방향 회전 → 밸브 **닫힘**
- 모터 구동이므로 스프링 불필요 → **정확한 위치 제어 가능**

### 특징 요약

| 항목 | 내용 |
|------|------|
| 전선 수 | 3가닥 |
| 피드백 | 일부 제품은 위치 신호 제공 |
| 제어 방식 | 모터 양방향 구동 |
| 복귀 방식 | 모터 역회전 (스프링 없음) |
| 동작 시간 | 약 2~5분 (제품별 상이) |

> **초보자 TIP**: 3선식은 정전 시 밸브가 현재 위치에서 멈춥니다 (스프링 복귀 없음).
> 정전 전 밸브가 열려 있었다면 정전 후에도 열린 상태를 유지합니다.

## 2.3 배선 복잡도, 안정성, 확장성 비교

### 종합 비교표

| 비교 항목 | 2선식 | 3선식 |
|-----------|-------|-------|
| 배선 수 | 2가닥 | 3가닥 |
| 시공 난이도 | 낮음 (단순) | 중간 |
| 배선 비용 | 낮음 | 중간 |
| 밸브 동작 정확성 | 보통 | 높음 |
| 상태 피드백 | 없음 | 일부 가능 |
| 정전 시 밸브 상태 | 닫힘 (안전) | 현 위치 유지 |
| 고장 진단 용이성 | 낮음 | 높음 |
| 비용 (구동기) | 저렴 | 다소 높음 |
| 국내 신축 적용 | 점차 감소 | 증가 추세 |

### 안정성 관점

- **2선식 안전 포인트**: 정전 시 자동 닫힘 → 과열 방지에 유리
- **3선식 안전 포인트**: 모터 토크가 크고 동작이 확실 → 결빙·오염된 밸브에서도 동작 신뢰성 높음

### 확장성 관점

- 2선식: 배선 수가 적어 존 추가 시 전선관 여유 공간 필요
- 3선식: 존 추가 시 가닥 수가 많아 전선관 용량 계획이 중요

---

## 2.4 현장 적용 기준 (신축 / 리모델링 / 교체)

### 신축 현장

- **권장: 3선식**
- 배선을 새로 설계하므로 가닥 수 제약 없음
- 고급 아파트·스마트홈 연동 요구가 증가하는 추세
- 장기 유지보수·진단 편의성 우수

### 리모델링 현장 (기존 배선 재활용)

- **기존 2가닥 배선 재활용 시: 2선식 유리**
- 배선 공사 없이 구동기만 교체 가능
- 단, 배선 상태(단선·피복 손상) 반드시 점검 후 재활용

### 구동기 단순 교체

- **기존 방식 그대로 유지가 원칙**
- 2선식 구동기를 3선식으로 교체 시 배선 추가 공사 필요
- 교체 전 현장 배선 방식 확인 필수

### 판단 흐름도

```
현장 확인
    │
    ├─ 기존 배선 있음?
    │       ├─ 2가닥 → 2선식 구동기 선택
    │       └─ 3가닥 → 3선식 구동기 선택
    │
    └─ 신규 배선?
            └─ 3선식 권장 (특수 요건 없으면)
```

## 문제 발생 시 즉시 대응

| 증상 | 원인 추정 | 1차 조치 |
|------|-----------|----------|
| 밸브가 열리지 않음 | 결선 오류 또는 구동기 불량 | 전원 OFF → 단자 결선 확인 |
| 밸브가 닫히지 않음 (과열) | 2선식 L선 상시 통전 | 해당 존 제어기 출력 점검 |
| 밸브 동작 소리 없음 | 전원 미인가 또는 구동기 불량 | 제어기 출력 전압 측정 (멀티미터) |
| 특정 방만 항상 열림 | OPEN/CLOSE 선 반대 결선 | 단자 재결선 |

> 전기 작업 전 반드시 **분전반 차단기 OFF** 후 **검전기로 무전압 확인** 후 작업하세요.

## 초보자 Q&A

**Q1. 집에 있는 구동기가 2선식인지 3선식인지 어떻게 알 수 있나요?**
A. 구동기에 연결된 전선 가닥 수를 세어보세요. 2가닥이면 2선식, 3가닥이면 3선식입니다. 또는 구동기 바디에 적힌 모델명을 검색하거나 제조사 스티커를 확인하세요.

**Q2. 2선식 구동기가 더 저렴한가요?**
A. 일반적으로 2선식이 더 저렴합니다. 단, 장기적으로 고장 진단·교체 비용을 고려하면 3선식이 유리한 경우도 있습니다.

**Q3. 3선식에서 OPEN선과 CLOSE선을 동시에 연결하면 어떻게 되나요?**
A. 모터에 상반된 명령이 동시에 인가돼 과전류가 흐를 수 있습니다. 구동기 내부 보호 회로가 동작하거나, 모터가 손상될 수 있습니다. 절대 동시 인가하지 마세요.

**Q4. 2선식과 3선식을 같은 시스템에 섞어 써도 되나요?**
A. 원칙적으로 혼용하지 않는 것이 좋습니다. 제어기에 따라 혼용을 지원하는 제품도 있지만, 이 경우 제조사 매뉴얼을 반드시 확인하고 단자 설정을 맞춰야 합니다.

**Q5. 스프링 복귀 2선식 구동기가 오래되면 스프링이 망가지나요?**
A. 네, 반복 동작으로 스프링 탄성이 저하될 수 있습니다. 밸브가 완전히 닫히지 않아 미세 누수나 난방 불균형이 생기면 구동기 교체를 검토하세요.
