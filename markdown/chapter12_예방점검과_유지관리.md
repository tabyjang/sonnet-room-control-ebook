# Chapter 12. 예방점검과 유지관리

---

## 한눈에 보기 (핵심 5줄)

1. 예방점검은 고장이 나기 전에 문제를 발견하는 가장 경제적인 유지관리 방법이다.
2. 월간·분기·연간 주기로 점검 항목을 나누면 빠짐없이 관리할 수 있다.
3. 구동기·온도조절기·제어기는 부품 수명이 있으므로 교체 주기를 미리 파악해두어야 한다.
4. 난방 시즌 시작(가을)과 종료(봄) 시점의 계절 전환 점검이 특히 중요하다.
5. 점검 기록을 남기면 반복 고장 원인 추적과 A/S 요청 시 빠른 처리가 가능하다.

---

## 왜 중요한가? — 생활 속 예시

**예시 1 — "겨울 첫날 보일러를 켰는데 갑자기 난방이 안 된다"**

여름 동안 방치했던 구동기 코일이 굳어버리거나 배관에 공기가 유입된 경우입니다.
난방 시즌 전 계절 점검을 해두면 추운 날 갑자기 난방이 멈추는 상황을 막을 수 있습니다.

**예시 2 — "10년 동안 한 번도 점검하지 않았는데 괜찮겠지"**

구동기 스프링 탄성 저하, 온도조절기 센서 오차 누적, 단자 접촉 저항 증가는 서서히 진행됩니다.
눈에 보이지 않는 노화는 어느 날 갑자기 큰 고장으로 나타납니다.

## 12.1 월간·분기·연간 점검 항목

### 점검 주기 전체 구조

```
┌─────────────────────────────────────────────────────────────┐
│  각방제어 시스템 정기 점검 캘린더                            │
├─────────┬───────────────────────────────────────────────────┤
│  주기   │  주요 점검 항목                                   │
├─────────┼───────────────────────────────────────────────────┤
│  월간   │  온도조절기 표시 확인 / 이상 소음 확인            │
│  (매월) │  구동기 동작 이상 육안 확인                       │
├─────────┼───────────────────────────────────────────────────┤
│  분기   │  단자 나사 조임 확인 / 전선 피복 상태             │
│  (3개월)│  구동기 열림·닫힘 동작 시험 / 누수 점검           │
├─────────┼───────────────────────────────────────────────────┤
│  연간   │  에어 제거 (퍼징) / HIU 필터 청소                 │
│  (12개월│  온도 보정 재확인 / 전체 존 동작 시험             │
│   →     │  밸런싱 밸브 재조정 / 배선 절연 저항 측정        │
│  계절별)│                                                   │
└─────────┴───────────────────────────────────────────────────┘
```

### 월간 점검표

| 점검 항목 | 방법 | 정상 기준 | 이상 시 조치 |
|-----------|------|-----------|-------------|
| 온도조절기 화면 표시 | 육안 확인 | 정상 숫자 표시, 깜빡임 없음 | Chapter 9.4 참조 |
| 비정상 소음 | 청각 확인 | 딸깍·윙윙·물 소리 없음 | Chapter 9.3 참조 |
| 구동기 외관 | 육안 확인 | 변형·이탈·누수 없음 | 이탈 시 재장착 |
| 분배기 누수 | 육안 확인 | 물기·결로 없음 | 밸브 잠금 후 조치 |
| 제어기 LED | 육안 확인 | 전원 LED 정상 점등 | 전원 결선 확인 |

### 분기 점검표

| 점검 항목 | 방법 | 정상 기준 | 이상 시 조치 |
|-----------|------|-----------|-------------|
| 단자 나사 조임 | 드라이버로 재조임 확인 | 헛돌지 않음 | 재조임 또는 단자 교체 |
| 전선 피복 상태 | 육안 확인 | 균열·경화·벗겨짐 없음 | 절연 테이프 또는 전선 교체 |
| 구동기 동작 시험 | 온도조절기로 ON/OFF 시험 | 3~5분 내 열림·닫힘 | Chapter 9.2 참조 |
| 온도조절기 온도 비교 | 기준 온도계와 비교 | 차이 ±2°C 이내 | 보정값 재조정 |
| 보일러 연동 시험 | 1존 요청 → 보일러 기동 | 정상 기동 | BLR 단자 확인 |

### 연간 점검표

| 점검 항목 | 방법 | 정상 기준 | 이상 시 조치 |
|-----------|------|-----------|-------------|
| 배관 에어 제거 (퍼징) | 에어벤트 개방, 물 보충 | 에어벤트에서 물만 나옴 | 반복 퍼징 |
| HIU 필터 청소 | 필터 탈거 후 세척 | 이물질 없는 상태 | 필터 세척 또는 교체 |
| 분배기 밸런싱 재확인 | 유량계 눈금 비교 | 루프 간 편차 20% 이내 | 밸런싱 밸브 재조정 |
| 전체 온도 보정 재확인 | 기준 온도계 vs 조절기 비교 | 차이 ±2°C 이내 | 오프셋 재설정 |
| 배선 절연 저항 | 멀티미터 저항 측정 | 수 MΩ 이상 (심선~외피) | 전선 교체 |
| 구동기 수명 확인 | 동작 횟수·연식 확인 | 설계 수명 이내 | 예방적 교체 검토 |

## 12.2 부품 수명과 교체 주기 가이드

### 주요 부품별 수명 기준

```
┌─────────────────────────────────────────────────────────────┐
│  부품 수명 및 교체 주기 가이드                               │
├──────────────────┬────────────┬──────────────┬──────────────┤
│  부품             │  설계 수명  │  실사용 수명  │  교체 권장   │
│                  │  (제조사)  │  (현장 기준)  │  시점        │
├──────────────────┼────────────┼──────────────┼──────────────┤
│ 열동형 구동기    │ 10만 회    │ 5~10년       │ 7년 이상 또는│
│ (왁스 타입)      │ 동작 기준  │              │ 동작 불량 시 │
├──────────────────┼────────────┼──────────────┼──────────────┤
│ 모터형 구동기    │ 10~15년    │ 8~12년       │ 10년 이상 또는│
│ (3선식)          │            │              │ 소음 발생 시 │
├──────────────────┼────────────┼──────────────┼──────────────┤
│ 온도조절기       │ 10~15년    │ 7~12년       │ 센서 오차    │
│ (유선형)         │            │              │ ±3°C 초과 시 │
├──────────────────┼────────────┼──────────────┼──────────────┤
│ 온도조절기       │ 배터리     │ 1~2년        │ 화면 어두워짐│
│ (배터리 무선형)  │ 1~2년      │              │ 또는 반응 느림│
├──────────────────┼────────────┼──────────────┼──────────────┤
│ 제어기           │ 10~15년    │ 10~15년      │ 채널 불량    │
│                  │            │              │ 또는 과열 시 │
├──────────────────┼────────────┼──────────────┼──────────────┤
│ 분배기 밸브      │ 20~30년    │ 15~25년      │ 누수·개폐    │
│                  │            │              │ 불량 시      │
├──────────────────┼────────────┼──────────────┼──────────────┤
│ HIU 순환펌프     │ 10~15년    │ 8~12년       │ 소음·진동    │
│ (지역난방)       │            │              │ 또는 유량 저하│
└──────────────────┴────────────┴──────────────┴──────────────┘
```

> **주의**: 수명은 사용 환경·물 수질·동작 빈도에 따라 크게 달라집니다.
> 위 기준은 참고값이며, 이상 증상 발생 시 수명과 관계없이 교체를 검토하세요.

### 교체 판단 흐름도

```
부품 교체 판단
          │
          ▼
  ① 이상 증상 발생?
     └─ YES → 즉시 교체 검토
     └─ NO  → ②로
          │
          ▼
  ② 설치 후 경과 연수 확인
     └─ 권장 교체 시점 초과?
          ├─ YES → 예방적 교체 권장
          └─ NO  → 분기 점검 유지
          │
          ▼
  ③ 연간 점검에서 성능 저하 확인?
     └─ 온도 오차 증가 / 동작 시간 지연
          ├─ YES → 교체 준비
          └─ NO  → 다음 점검까지 유지
```

## 12.3 계절 전환기 점검 (난방 시작·종료)

### 난방 시작 전 점검 (가을, 10월 전후)

```
┌─────────────────────────────────────────────────────────────┐
│  난방 시즌 시작 전 점검 체크리스트                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  전기·제어 계통                                             │
│  [ ] 제어기 전원 ON → LED 정상 확인                        │
│  [ ] 모든 온도조절기 전원 및 화면 정상 확인                 │
│  [ ] 온도 표시값 기준 온도계와 비교 (보정 필요 시 조정)     │
│  [ ] 배터리형 온도조절기 배터리 교체                        │
│                                                             │
│  배관·분배기 계통                                           │
│  [ ] 분배기 공급 밸브 열기 → 누수 확인                     │
│  [ ] 에어벤트로 배관 에어 제거 (퍼징)                      │
│  [ ] 분배기 유량계 눈금 확인 (유량 정상 여부)              │
│  [ ] 각 루프 구동기 동작 시험 (전 존 열림·닫힘)            │
│                                                             │
│  보일러·HIU 계통                                           │
│  [ ] 보일러/HIU 전원 ON → 오류 코드 없음 확인              │
│  [ ] HIU 필터 청소 (지역난방)                              │
│  [ ] 보일러 공급 온도 설정 확인                            │
│  [ ] 보일러 연동 시험 (존 요청 → 보일러 기동)              │
│                                                             │
│  최종 확인                                                  │
│  [ ] 전 존 난방 동작 및 온도 응답 확인                     │
│  [ ] 비정상 소음·누수 없음 확인                            │
└─────────────────────────────────────────────────────────────┘
```

## 12.4 유지관리 기록 표준화

### 점검 기록의 중요성

```
점검 기록 보유 시 장점
┌──────────────────────────────────────────────────────────┐
│  ① 반복 고장 원인 추적                                   │
│     "3개월 전에도 같은 존에서 같은 증상 있었음"          │
│     → 패턴 발견 → 근본 원인 탐색 가능                   │
│                                                          │
│  ② A/S 요청 시 빠른 처리                                 │
│     설치 일자·부품 모델·과거 증상 제공                   │
│     → 기사 방문 전 사전 준비 가능                        │
│                                                          │
│  ③ 보증 기간 확인                                        │
│     교체 일자 기록 → 보증 기간 내 무상 수리 여부 확인   │
│                                                          │
│  ④ 인수인계 자료                                         │
│     집 매도·임대 시 유지관리 이력 제공                   │
└──────────────────────────────────────────────────────────┘
```

## 초보자 Q&A

**Q1. 정기 점검을 직접 해야 하나요, 업체에 맡겨야 하나요?**
A. 월간·분기 점검은 이 전자책을 참고해 직접 가능합니다. 연간 점검은 배선 절연 저항 측정이나 HIU 내부 점검 등 전문 장비가 필요한 항목이 있으므로, 설치 업체 또는 A/S 기사에게 의뢰를 권장합니다.

**Q2. 에어 제거(퍼징)는 어떻게 하나요?**
A. 분배기 상단 에어벤트를 조금 열어두고, 분배기 드레인 밸브를 이용해 물을 보충하면서 에어벤트에서 공기가 나오면 열어두고 물만 나오면 잠급니다. 처음 해보는 경우 설치 업체에 시연을 요청하세요.

**Q3. 배터리형 온도조절기는 배터리가 얼마나 자주 교체되나요?**
A. 사용 빈도와 배터리 용량에 따라 다르지만 일반적으로 1~2년마다 교체합니다. 화면이 어두워지거나 반응이 늦어지면 교체 시점입니다. 충전식 배터리 사용은 전압 불안정으로 오작동을 유발할 수 있어 권장하지 않습니다.

**Q4. 유량계가 없는 분배기는 밸런싱을 어떻게 확인하나요?**
A. 각 루프 배관의 바닥 온도를 손으로 느끼거나 접촉식 온도계로 비교합니다. 온도 차이가 크면 유량 불균형을 의심하고 밸런싱 밸브를 조정하세요. 정밀 밸런싱은 전문가 의뢰를 권장합니다.

**Q5. 점검 기록은 어떤 형식으로 보관하는 게 좋나요?**
A. 이 장에 수록된 양식을 출력해 바인더에 보관하거나, 스마트폰 사진과 메모 앱을 활용해도 됩니다. 중요한 것은 날짜·항목·이상 내용·조치 내용을 빠짐없이 남기는 것입니다.
