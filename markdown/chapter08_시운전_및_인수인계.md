# Chapter 8. 시운전 및 인수인계

---

## 한눈에 보기 (핵심 5줄)

1. 시운전은 설치 완료 후 시스템이 설계대로 동작하는지 확인하는 최종 검증 단계다.
2. 전원 인가 전 사전 점검 → 존별 동작 시험 → 보일러 연동 시험 → 전체 통합 시험 순서로 진행한다.
3. 각 존의 온도 요청 → 밸브 열림 → 보일러 기동 → 온도 도달 → 밸브 닫힘 흐름이 정상 동작하는지 확인한다.
4. 시운전 완료 후 사용자에게 조작 방법·주의사항·비상 대처법을 반드시 인수인계한다.
5. 인수인계 체크리스트에 서명을 받고 보관하면 향후 분쟁·오작동 책임 소재를 명확히 할 수 있다.

---

## 왜 중요한가? — 생활 속 예시

**예시 1 — "설치는 됐는데 보일러가 안 켜진다"**

존별 밸브는 정상 동작하는데 보일러가 켜지지 않으면
제어기 → 보일러 연동 신호선 문제입니다.
시운전 단계에서 연동 시험을 순서대로 하면 이 문제를 즉시 발견할 수 있습니다.

**예시 2 — "사용자가 온도조절기 조작법을 몰라서 계속 전화가 온다"**

설치만 하고 사용법을 알려주지 않으면 사용자가 불편을 겪습니다.
인수인계 때 조작법·스케줄 설정·비상 대처를 직접 시연하면 사후 민원을 줄일 수 있습니다.

## 8.1 전원 인가 전 사전 점검

### 전원 인가 전 최종 점검 흐름도

```
전원 인가 전 사전 점검 순서
          │
          ▼
  ① 결선 최종 육안 확인
     • 단자 기호와 전선 색상 일치 여부
     • 느슨한 단자 나사 재조임
     • 피복 손상 전선 교체
          │
          ▼
  ② 멀티미터 저항 점검 (차단기 OFF 상태)
     • L-N 간: 무한대 (∞) 확인
     • 구동기 전선 양단: 코일 저항값 (수십~수백 Ω)
     • 신호선 양단: 수 Ω 이하
          │
          ▼
  ③ 분배기 물 채우기 및 누수 확인
     • 공급 밸브 서서히 개방
     • 에어벤트로 공기 제거
     • 5분 이상 누수 관찰
          │
          ▼
  ④ 보일러/HIU 정상 가동 확인
     • 보일러 전원 ON
     • 오류 코드 없음 확인
     • 공급 온도 정상 범위 확인
          │
          ▼
  ⑤ 이상 없으면 제어기 차단기 ON
```

### 전원 인가 전 점검표

| 점검 항목 | 정상 기준 | 이상 시 조치 |
|-----------|-----------|-------------|
| L-N 간 저항 | 무한대 (∞) | 단락 지점 탐색 후 재결선 |
| 구동기 코일 저항 | 50~500 Ω (제품별) | 0Ω → 단락 / ∞ → 단선, 구동기 교체 |
| 신호선 도통 | 수 Ω 이하 | ∞ → 단선, 경로 재확인 |
| 배관 누수 | 누수 없음 | 밸브 잠금 후 어댑터 재체결 |
| 보일러 오류 코드 | 없음 | 보일러 매뉴얼 오류 코드 조치 |

## 8.2 존별 동작 시험 (난방 요구·밸브 동작·온도 응답)

### 존별 동작 시험 흐름도

```
존별 동작 시험 (Zone 1부터 순서대로 반복)
          │
          ▼
  ① 해당 존 온도조절기 설정온도를
     현재 실내온도보다 3~5°C 높게 설정
          │
          ▼
  ② 제어기 해당 존 LED 점등 확인
     (난방 요청 수신 표시)
          │
          ▼
  ③ 해당 구동기 동작 확인
     열동형: 2~3분 후 밸브 열림 (구동기 몸체 미열)
     모터형: 동작음 후 밸브 열림
          │
          ▼
  ④ 분배기 유량계(있는 경우) 유량 증가 확인
          │
          ▼
  ⑤ 보일러/HIU 기동 확인
     (제어기 BLR 신호 출력 → 보일러 버너 점화)
          │
          ▼
  ⑥ 해당 존 온도조절기 설정온도를
     현재 온도보다 낮게 재설정
          │
          ▼
  ⑦ 밸브 닫힘 확인
     열동형: 3~5분 후 냉각 복귀
     모터형: 동작음 후 닫힘
          │
          ▼
  ⑧ 다음 존으로 이동 → ①부터 반복
```

### 존별 동작 시험 기록표

| 존 | 위치 | 온도조절기 반응 | 구동기 열림 | 유량 확인 | 구동기 닫힘 | 판정 |
|----|------|----------------|-------------|-----------|-------------|------|
| Z1 | 거실 | ○ / ✕ | ○ / ✕ | ○ / ✕ | ○ / ✕ | 정상 / 불량 |
| Z2 | 안방 | ○ / ✕ | ○ / ✕ | ○ / ✕ | ○ / ✕ | 정상 / 불량 |
| Z3 | 아이방 | ○ / ✕ | ○ / ✕ | ○ / ✕ | ○ / ✕ | 정상 / 불량 |
| Z4 | 서재 | ○ / ✕ | ○ / ✕ | ○ / ✕ | ○ / ✕ | 정상 / 불량 |

### 온도 응답 확인 기준

```
온도 응답 정상 범위
┌─────────────────────────────────────────────────────┐
│                                                     │
│  설정온도: 22°C / 현재 실내온도: 18°C               │
│                                                     │
│  ┌──── 밸브 열림 (2~3분)                           │
│  │                                                  │
│  │ 난방수 순환 시작                                 │
│  │   ↓                                              │
│  │ 바닥 온도 상승 (15~30분 후 체감)                 │
│  │   ↓                                              │
│  │ 실내 온도 22°C 도달 (30~60분, 건물 단열 따라 상이)│
│  │   ↓                                              │
│  └──── 밸브 닫힘 신호 출력                          │
│                                                     │
│  ※ 응답 시간은 건물 단열·외기온도·바닥 면적에 따라  │
│     달라지므로 수치보다 흐름(열림→상승→닫힘)을 확인 │
└─────────────────────────────────────────────────────┘
```

## 8.3 통신·연동 시험 및 로그 확인

### 보일러/HIU 연동 시험

```
연동 시험 순서
          │
          ▼
  ① 모든 존 온도조절기를 현재 온도보다 낮게 설정
     → 모든 존 난방 요청 없음 상태
          │
          ▼
  ② 보일러/HIU 정지 상태 확인
     (제어기 BLR 신호 없음 → 보일러 대기)
          │
          ▼
  ③ Zone 1 온도조절기 설정온도 높임
     → 제어기 BLR 신호 출력
     → 보일러 기동 확인
          │
          ▼
  ④ Zone 1 온도조절기 설정온도 낮춤
     → 모든 존 요청 없음
     → 보일러 정지 확인
          │
          ▼
  ⑤ 연동 정상 판정
```

### 연동 시험 결과 기록표

| 시험 항목 | 기대 동작 | 실제 동작 | 판정 |
|-----------|-----------|-----------|------|
| 1존 요청 → 보일러 기동 | 보일러 버너 ON | | ○ / ✕ |
| 모든 존 해제 → 보일러 정지 | 보일러 버너 OFF | | ○ / ✕ |
| 2존 동시 요청 | 2개 밸브 열림 + 보일러 ON | | ○ / ✕ |
| 정전 복귀 후 재기동 | 이전 설정값 유지 | | ○ / ✕ |

### 스마트홈/앱 연동 시험 (해당 제품)

| 시험 항목 | 확인 내용 |
|-----------|-----------|
| 앱 설치 및 기기 등록 | 온도조절기 앱 페어링 성공 여부 |
| 원격 온도 확인 | 앱 표시 온도 = 현장 온도조절기 표시 일치 |
| 원격 설정 변경 | 앱에서 설정온도 변경 → 밸브 동작 확인 |
| 스케줄 설정 | 요일·시간대별 온도 스케줄 동작 확인 |
| 알림 기능 | 오류·온도 이탈 알림 수신 확인 |

## 8.4 사용자 인수인계 체크리스트

### 인수인계 전달 내용 목록

```
┌─────────────────────────────────────────────────────┐
│  사용자 인수인계 필수 전달 사항                      │
├─────────────────────────────────────────────────────┤
│                                                     │
│  1. 시스템 구성 설명                                │
│     • 제어기·구동기·온도조절기 위치 안내            │
│     • 각 존(방)과 온도조절기 대응 관계 설명         │
│                                                     │
│  2. 온도조절기 기본 조작법                          │

│                                                     │

│     • 외출 시 설정온도 낮추기 (16°C 권장)           │
│     • 장기 부재 시 동파 방지 모드 설정              │

│                                                     │
│  4. 비상 대처 방법                                  │
│     • 난방 이상 시 1차 확인 순서                    │
│     • 분전반 차단기 위치 및 리셋 방법               │
│     • 누수 발생 시 분배기 밸브 잠금 위치            │
│     • A/S 연락처 (제조사·설치업체)                  │
│                                                     │
│  5. 정기 점검 안내                                  │
│     • 월 1회: 구동기 동작 이상 여부 육안 확인       │
│     • 연 1회: HIU 필터 청소, 분배기 에어 제거       │
└─────────────────────────────────────────────────────┘
```

| 3 | 밸브 열림·닫힘 동작 직접 확인 | ○ / ✕ |
| 4 | 스케줄 설정 방법 시연 | ○ / ✕ |

| 6 | 분배기 긴급 차단 밸브 위치 안내 | ○ / ✕ |

| # | 실수 | 올바른 방법 |

## 문제 발생 시 즉시 대응

| 증상 | 원인 추정 | 조치 |
|------|-----------|------|
| 전원 인가 후 제어기 LED 없음 | 전원 결선 불량 또는 차단기 용량 부족 | 결선 재확인, 차단기 용량 확인 |
| 특정 존 구동기 무반응 | 결선 오류 또는 구동기 불량 | 해당 존 단자 전압 측정 |
| 보일러 미기동 | BLR 연동선 불량 | BLR 단자 전압·접촉 재확인 |
| 온도가 설정값까지 오르지 않음 | 유량 부족 또는 단열 불량 | 분배기 밸런싱 재조정, 창문·문 확인 |
| 스케줄 설정 후 동작 안 함 | 시각 설정 오류 | 온도조절기 현재 시각 재설정 |

## 초보자 Q&A

**Q1. 시운전 시 구동기가 열리는 데 3분 넘게 걸립니다. 고장인가요?**
A. 열동형(왁스) 구동기는 코일 가열 후 왁스가 팽창하는 시간이 필요해 2~5분이 소요됩니다. 5분 이상 지나도 열리지 않으면 구동기 불량을 의심하세요.

**Q2. 온도조절기 설정온도를 현재보다 높게 설정했는데 밸브가 안 열립니다.**
A. ① 제어기 해당 존 LED 확인 → ② 구동기 전선 전압 측정 순서로 점검하세요. 제어기 LED가 켜졌는데 구동기가 반응 없으면 구동기 불량, LED도 안 켜지면 온도조절기 신호선 문제입니다.

**Q3. 사용자가 스케줄 설정을 어렵다고 합니다. 어떻게 설명하면 좋나요?**
A. "평일 오전 7시에 22도, 낮 10시에 18도로 자동으로 바뀌게 하는 기능"처럼 생활 언어로 설명합니다. 제조사 앱이 있으면 앱 화면에서 직접 같이 설정해보는 것이 가장 효과적입니다.

**Q4. 인수인계 서류를 꼭 종이로 받아야 하나요?**
A. 사진·이메일·문자로 대체할 수 있지만, 분쟁 발생 시 종이 서명이 가장 명확한 증거가 됩니다. 최소한 설치 완료 사진과 연락처 교환 기록을 남겨두세요.

**Q5. 시운전 완료 후 얼마나 지나야 실제 난방이 안정되나요?**
A. 시운전 직후에는 배관 내 공기 잔류·바닥 열용량 때문에 온도 응답이 다소 느립니다. 첫 2~3일간 정상 운전 후 안정화됩니다. 이 기간 중 이상 증상은 사용자에게 미리 안내해두세요.
