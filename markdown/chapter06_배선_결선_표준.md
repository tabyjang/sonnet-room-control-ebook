# Chapter 6. 배선/결선 표준

---

## 한눈에 보기 (핵심 5줄)

1. 결선은 전자책에서 가장 오류가 잦은 단계이므로 반드시 순서를 지켜 진행한다.
2. 2선식과 3선식 결선도는 완전히 다르므로 현장 방식을 먼저 확인 후 결선한다.
3. 전원선(L/N)·신호선·접지선을 절대 혼용하지 않는다.
4. 노이즈 대응(쉴드선 사용·접지 처리)을 하지 않으면 온도조절기 오작동이 발생한다.
5. 결선 완료 후 멀티미터로 도통 점검을 반드시 수행하고 전원을 인가한다.

---

## 왜 중요한가? — 생활 속 예시

**예시 1 — "전원을 넣었더니 제어기 퓨즈가 끊어졌다"**

L선(전원)과 N선(중성)을 구동기 단자에 반대로 연결하거나,
신호선 단자에 전원선을 연결하면 과전류로 퓨즈가 끊어집니다.
결선 전 도면을 한 번 더 확인하는 습관이 이 사고를 막습니다.

**예시 2 — "온도조절기 숫자가 계속 깜빡이거나 오작동한다"**

신호선이 전원선과 같은 전선관에 나란히 지나가면
전자기 노이즈가 신호선에 유입돼 오작동이 생깁니다.
노이즈 대응 배선 규칙을 지키면 이런 문제를 예방할 수 있습니다.

## 6.1 2선식 표준 결선도

### 2선식 시스템 전체 결선도

```
분전반
  │
  │ AC 220V
  ▼
┌────────────────────────────────────────────────┐
│                   제어기                        │
│                                                │
│  [L] ──── 전원 입력                            │
│  [N] ──── 전원 입력                            │
│  [GND] ── 접지                                 │
│                                                │
│  [BLR+] ─────────────────────────────────────┐│
│  [BLR-] ─────────────────────────────────────┼┤  → 보일러
│                                              └┤    콜-포-히트
│  [Z1] ──── Zone1 온도조절기 신호 입력           │
│  [Z2] ──── Zone2 온도조절기 신호 입력           │
│  [Z3] ──── Zone3 온도조절기 신호 입력           │
│  [Z4] ──── Zone4 온도조절기 신호 입력           │
│                                                │
│  [A1+] ─────────────────────────────────────── → 구동기 1 (L)
│  [A1-] ─────────────────────────────────────── → 구동기 1 (N)
│  [A2+] ─────────────────────────────────────── → 구동기 2 (L)
│  [A2-] ─────────────────────────────────────── → 구동기 2 (N)
│  [A3+] ─────────────────────────────────────── → 구동기 3 (L)
│  [A3-] ─────────────────────────────────────── → 구동기 3 (N)
│  [A4+] ─────────────────────────────────────── → 구동기 4 (L)
│  [A4-] ─────────────────────────────────────── → 구동기 4 (N)
└────────────────────────────────────────────────┘

온도조절기 연결:
  온도조절기 1 ── 신호선 (2심) ── [Z1, COM]
  온도조절기 2 ── 신호선 (2심) ── [Z2, COM]
  온도조절기 3 ── 신호선 (2심) ── [Z3, COM]
  온도조절기 4 ── 신호선 (2심) ── [Z4, COM]

구동기 연결:
  구동기 1 ── 전선 (2심) ── [A1+, A1-]
  구동기 2 ── 전선 (2심) ── [A2+, A2-]
  구동기 3 ── 전선 (2심) ── [A3+, A3-]
  구동기 4 ── 전선 (2심) ── [A4+, A4-]
```

### 2선식 단자별 결선 요약표

| 단자 기호 | 연결 대상 | 전선 규격 | 비고 |
|-----------|-----------|-----------|------|
| L | 분전반 차단기 L | 2.5㎟ 이상 | 전원선 |
| N | 분전반 차단기 N | 2.5㎟ 이상 | 중성선 |
| GND | 분전반 접지 | 2.5㎟ 이상 | 접지선 (녹색/황녹) |
| BLR+/- | 보일러 외부 입력 | 0.75㎟ 이상 | 무전압 접점 출력 |
| Z1~ZN | 각 온도조절기 신호 | 0.5~0.75㎟ | 신호선 (쉴드 권장) |
| COM | 온도조절기 공통 | 0.5~0.75㎟ | 신호선 공통 기준 |
| A1~AN +/- | 각 구동기 전원 출력 | 0.75~1.5㎟ | AC 220V 출력 |

## 6.2 3선식 표준 결선도

### 3선식 시스템 전체 결선도

```
분전반
  │
  │ AC 220V
  ▼
┌────────────────────────────────────────────────┐
│                   제어기                        │
│                                                │
│  [L] ──── 전원 입력                            │
│  [N] ──── 전원 입력                            │
│  [GND] ── 접지                                 │
│                                                │
│  [BLR+/BLR-] ──────────────────────────────── → 보일러/HIU
│                                                │
│  [Z1~ZN] ── 온도조절기 신호 입력               │
│  [COM]   ── 온도조절기 공통                    │
│                                                │
│  [A1-COM] ──────────────────────────────────── → 구동기1 COM
│  [A1-OPN] ──────────────────────────────────── → 구동기1 OPEN
│  [A1-CLS] ──────────────────────────────────── → 구동기1 CLOSE
│                                                │
│  [A2-COM] ──────────────────────────────────── → 구동기2 COM
│  [A2-OPN] ──────────────────────────────────── → 구동기2 OPEN
│  [A2-CLS] ──────────────────────────────────── → 구동기2 CLOSE
│                                                │
│  [A3-COM] ──────────────────────────────────── → 구동기3 COM
│  [A3-OPN] ──────────────────────────────────── → 구동기3 OPEN
│  [A3-CLS] ──────────────────────────────────── → 구동기3 CLOSE
└────────────────────────────────────────────────┘
```

### 3선식 구동기 단자 결선 상세

```
제어기 출력 단자          구동기 단자
                          (제품마다 색상 다름 — 반드시 매뉴얼 확인)
  [A1-COM] ─── 흑(공통) ─── [COM / C]
  [A1-OPN] ─── 백(열림) ─── [OPEN / O / NO]
  [A1-CLS] ─── 적(닫힘) ─── [CLOSE / C / NC]
```

### 3선식 단자별 결선 요약표

| 단자 기호 | 연결 대상 | 전선 규격 | 비고 |
|-----------|-----------|-----------|------|
| L, N | 분전반 차단기 | 2.5㎟ 이상 | 전원 |
| GND | 접지 | 2.5㎟ 이상 | 녹색/황녹 |
| BLR | 보일러/HIU | 0.75㎟ | 무전압 접점 |
| Z1~ZN, COM | 온도조절기 | 0.5~0.75㎟ | 신호선 |
| AN-COM | 구동기 공통(COM) | 0.75~1.5㎟ | 3선 중 공통 |
| AN-OPN | 구동기 열림(OPEN) | 0.75~1.5㎟ | 3선 중 열림 |
| AN-CLS | 구동기 닫힘(CLOSE) | 0.75~1.5㎟ | 3선 중 닫힘 |

## 6.3 전원·신호선·접지·노이즈 대응

### 전선 종류별 역할 구분

```
┌──────────────────────────────────────────────────────┐
│               전선 종류 구분                          │
├─────────────┬────────────────┬───────────────────────┤
│ 구분        │ 해당 전선       │ 특징                  │
├─────────────┼────────────────┼───────────────────────┤
│ 전원선      │ L, N, GND      │ 굵은 단면적, 절연 피복 │
│             │ 구동기 출력선  │ (2.5㎟ 이상)          │
├─────────────┼────────────────┼───────────────────────┤
│ 신호선      │ 온도조절기     │ 가는 단면적            │
│             │ Z1~ZN, COM     │ (0.5~0.75㎟)          │
│             │ 보일러 연동    │ 쉴드 처리 권장         │
├─────────────┼────────────────┼───────────────────────┤
│ 접지선      │ GND            │ 녹색 또는 황녹색 피복  │
│             │ 쉴드 접지      │ (2.5㎟ 이상)          │
└─────────────┴────────────────┴───────────────────────┘
```

### 전선 색상 표준 (국내 일반)

| 전선 용도 | 권장 색상 | 비고 |
|-----------|-----------|------|
| L선 (전원 Hot) | 적색 또는 갈색 | |
| N선 (중성) | 청색 또는 흑색 | |
| 접지 (GND) | 녹색 또는 황녹색 | 절대 다른 용도 사용 금지 |
| 신호선 (+) | 백색 또는 적색 | 제조사 지정 색 우선 |
| 신호선 (-/COM) | 흑색 또는 청색 | 제조사 지정 색 우선 |

> **주의**: 색상은 관례이며 강제 규격이 아닙니다.
> **반드시 단자 기호(L·N·GND·Z·COM 등)를 기준으로 결선**하세요.

### 노이즈 대응 원칙

```
노이즈 유입 경로
┌──────────────────────────────────────────────────┐
│  전원선 (L/N)                                    │
│      │                                           │
│      │  전자기 유도                               │
│      ▼                                           │
│  신호선 (Z/COM)  ← 노이즈 유입 → 오작동 발생     │
└──────────────────────────────────────────────────┘

노이즈 차단 방법
┌──────────────────────────────────────────────────┐
│ ① 전원선과 신호선 분리 배선                       │
│    (같은 전선관 금지, 최소 10cm 이격)             │
│                                                  │
│ ② 신호선에 쉴드(Shield) 케이블 사용              │
│    쉴드층 → 제어기 GND에 단자 측만 접지           │
│    (양단 접지 금지 → 루프 전류 유발)             │
│                                                  │
│ ③ 긴 신호선 구간 (10m 이상)                      │
│    → 트위스트 페어(꼬임 쌍) 케이블 사용          │
│                                                  │
│ ④ 신호선과 동력선 교차 시                        │
│    → 직각으로 교차 (평행 배선 금지)              │
└──────────────────────────────────────────────────┘
```

### 쉴드 케이블 접지 방법

```
온도조절기 측          제어기 측
(이 쪽 쉴드는          (이 쪽 쉴드를
 절연 처리)             GND에 접지)

   [온도조절기]                [제어기 GND]
        │                           │
   ─────┼──── 신호선(심선) ──────────┼─────
        │                           │
   ─────┼──── 쉴드(차폐층) ──────────┼──── GND 접지
        │                           │
        ×  (이 쪽은 접지 안 함)
```

## 6.4 배선 실수 사례와 교정 방법

### 오결선 유형 및 교정

```
┌────────────────────────────────────────────────────────────┐
│  오결선 유형 1: 전원선과 신호선 단자 혼용                   │
│                                                            │
│  [잘못된 결선]                     [올바른 결선]           │
│  제어기 Z1 단자 ─── L선 (AC220V)   제어기 Z1 ─── 신호선   │
│                                                            │
│  증상: 제어기 내부 손상, 퓨즈 단락                         │
│  교정: 전원 OFF → 단자 재확인 → 올바른 선으로 재결선       │
└────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────┐
│  오결선 유형 2: 3선식 OPEN/CLOSE 반전                      │
│                                                            │
│  [잘못된 결선]                     [올바른 결선]           │
│  A1-OPN ─── 구동기 CLOSE           A1-OPN ─── 구동기 OPEN │
│  A1-CLS ─── 구동기 OPEN            A1-CLS ─── 구동기 CLOSE│
│                                                            │
│  증상: 밸브가 반대로 동작 (열어야 닫힘, 닫아야 열림)       │
│  교정: 전원 OFF → OPN/CLS 선 교환 재결선                   │
└────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────┐
│  오결선 유형 3: 접지선 미연결                              │
│                                                            │
│  [잘못된 결선]         [올바른 결선]                       │
│  GND 단자 ─── 미연결   GND 단자 ─── 분전반 접지            │
│                                                            │
│  증상: 누전 시 감전 위험, 노이즈 증가, 오작동              │
│  교정: 전원 OFF → GND 선 추가 결선 → 접지 저항 측정        │
└────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────┐
│  오결선 유형 4: 존 번호 혼용 (Z1과 Z2 선 교차)            │
│                                                            │
│  [잘못된 결선]                     [올바른 결선]           │
│  거실 온도조절기 ─── Z2 단자        거실 온도조절기 ─── Z1 │
│  안방 온도조절기 ─── Z1 단자        안방 온도조절기 ─── Z2 │
│                                                            │
│  증상: 거실 온도 올리면 안방 밸브가 열림                   │
│  교정: 존 번호 매핑표 작성 후 선 순서 재확인               │
└────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────┐
│  오결선 유형 5: 보일러 연동선 극성 반전                    │
│                                                            │
│  [잘못된 결선]                     [올바른 결선]           │
│  BLR+ ─── 보일러 (-) 단자          BLR+ ─── 보일러 (+)    │
│  BLR- ─── 보일러 (+) 단자          BLR- ─── 보일러 (-)    │
│                                                            │
│  증상: 보일러가 항상 켜지거나 전혀 안 켜짐                 │
│  교정: 보일러 매뉴얼 단자 기호 확인 후 재결선              │
└────────────────────────────────────────────────────────────┘
```

### 결선 완료 후 도통 점검 절차

```
결선 완료
    │
    ▼
① 전원 OFF (차단기 내림) 확인
    │
    ▼
② 멀티미터 → 저항(Ω) 모드 설정
    │
    ▼
③ 각 구동기 전선 양단 도통 측정
   (정상: 수십~수백 Ω / 단선: 무한대 / 단락: 0Ω)
    │
    ▼
④ 각 신호선 양단 도통 측정
   (정상: 수 Ω 이하 / 단선: 무한대)
    │
    ▼
⑤ 전원선(L)과 중성선(N) 간 저항 측정
   (정상: 무한대 / 단락: 0Ω → 결선 오류)
    │
    ▼
⑥ 이상 없으면 차단기 올리고 전원 인가
    │
    ▼
⑦ 각 존 동작 확인 (Chapter 8 시운전 절차로 이동)
```

### 오결선 빠른 진단표

| 증상 | 추정 오결선 유형 | 확인 방법 |
|------|-----------------|-----------|
| 전원 인가 즉시 차단기 트립 | 전원-신호선 혼용 또는 단락 | 전원 OFF 후 멀티미터 저항 측정 |
| 특정 방 밸브 동작 반대 | 3선식 OPEN/CLOSE 반전 | 해당 구동기 선 교환 |
| 모든 방 밸브 무반응 | 전원 미인가 또는 GND 미연결 | 전원 전압 측정 |
| 보일러 미기동 | 보일러 연동선 오결선 | BLR 단자 전압·극성 확인 |
| 온도조절기 표시 오작동 | 신호선 노이즈 또는 존 혼용 | 존 번호 매핑 재확인 |
| 특정 방 다른 방 제어 | 존 번호 혼용 | 신호선 추적 후 재결선 |

- [ ] 쉴드 케이블을 한쪽(제어기 측)만 접지하는 이유를 이해했다
- [ ] 오결선 5가지 유형과 교정 방법을 알고 있다
- [ ] 결선 완료 후 도통 점검 절차를 순서대로 말할 수 있다

## 자주 하는 실수 TOP 5

| # | 실수 | 올바른 방법 |
|---|------|------------|
| 1 | 전원 인가 상태에서 결선 작업 | 반드시 차단기 OFF + 검전기 확인 후 작업 |
| 2 | 단자 기호 확인 없이 색상만 보고 결선 | 단자 기호(L·N·Z·COM 등) 기준으로 결선 |
| 3 | 전원선과 신호선 같은 전선관 배선 | 분리 배선, 최소 10cm 이격 |
| 4 | 쉴드 케이블 양단 모두 접지 | 제어기 측(단자 측) 한쪽만 GND 접지 |
| 5 | 도통 점검 없이 전원 인가 | 결선 후 멀티미터 도통 점검 필수 |

## 문제 발생 시 즉시 대응

| 증상 | 즉시 조치 |
|------|-----------|
| 차단기 트립 | 즉시 전원 OFF → 멀티미터로 단락 지점 탐색 |
| 제어기 전원 LED 안 켜짐 | 전원선(L/N) 결선 확인 → 차단기 용량 확인 |
| 구동기 전체 무반응 | 제어기 출력 전압 측정 → 구동기 전선 도통 확인 |
| 연기·냄새 발생 | 즉시 차단기 OFF → 전문가 점검 의뢰 (재가동 금지) |

> **안전 최우선 원칙**
> 연기·타는 냄새·이상한 소리가 나면 즉시 차단기를 내리고
> 원인이 파악될 때까지 절대 재가동하지 마세요.

## 초보자 Q&A

**Q1. 전선 굵기(단면적)는 어떻게 고르나요?**
A. 전원선·구동기 출력선은 2.5㎟ 이상, 신호선은 0.5~0.75㎟를 사용합니다. 길이가 15m를 초과하면 전압강하를 줄이기 위해 한 단계 굵은 전선을 선택합니다.

**Q2. 압착 단자와 나사 단자 중 어느 것이 더 좋나요?**
A. 진동이 있는 환경에서는 압착 단자가 접촉 불량 방지에 유리합니다. 가정용 각방제어는 나사 단자도 충분하지만, 단자 나사를 적정 토크로 균일하게 조여야 합니다.

**Q3. 신호선에 일반 전선(VCTF)을 써도 되나요?**
A. 짧은 거리(5m 이내, 노이즈 환경 아닌 경우)는 사용 가능합니다. 10m 이상이거나 전원선 근처를 지나는 경우 쉴드 케이블(CVVS 등) 사용을 권장합니다.

**Q4. 보일러 연동선에 극성이 없다고 했는데, 왜 오결선이 생기나요?**
A. 무전압 접점(건접점) 방식의 보일러 연동은 극성이 없습니다. 단, 일부 보일러는 유전압(DC 12V 등) 입력 방식이어서 극성이 있습니다. 사전에 보일러 매뉴얼을 확인하세요.

**Q5. 결선 중 중간에 멈추고 다음날 이어서 작업해도 되나요?**
A. 가능합니다. 단, 작업 재개 전 반드시 차단기 OFF 상태를 확인하고, 어디까지 작업했는지 메모와 사진을 남겨 혼동을 방지하세요.
