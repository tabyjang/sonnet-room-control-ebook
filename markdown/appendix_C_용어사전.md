# 부록 C. 용어사전

> 이 전자책에 등장하는 주요 용어를 카테고리별로 정리했습니다.
> 처음 접하는 용어는 이 사전을 먼저 확인하세요.

---

## C-1. 시스템 구성 용어

| 용어 | 영문 | 쉬운 설명 | 관련 챕터 |
|------|------|-----------|-----------|
| **각방제어** | Zone Control | 방마다 온도를 독립적으로 설정·제어하는 난방 시스템 | Ch.1 |
| **존(Zone)** | Zone | 하나의 온도조절기가 담당하는 공간 단위. 보통 방 1개 = 존 1개 | Ch.1 |
| **열원** | Heat Source | 난방수를 데우는 에너지 공급 장치. 보일러 또는 HIU | Ch.1 |
| **분배기** | Manifold | 여러 방으로 난방수를 나눠주는 분기 장치 | Ch.1, 4 |
| **난방수** | Heating Water | 보일러 또는 HIU에서 데워져 배관을 순환하는 물 | Ch.1 |
| **루프** | Loop | 분배기에서 출발해 각 방 바닥을 돌고 돌아오는 배관 회로 | Ch.1, 4 |
| **콜-포-히트** | Call for Heat | 온도조절기가 난방 필요 신호를 제어기에 요청하는 것 | Ch.3 |
| **히스테리시스** | Hysteresis | ON/OFF 제어에서 잦은 스위칭을 막기 위한 온도 편차 폭 | Ch.3 |

## C-2. 장치·부품 용어

| 용어 | 영문 | 쉬운 설명 | 관련 챕터 |
|------|------|-----------|-----------|
| **제어기** | Controller | 온도조절기 신호를 받아 구동기를 제어하고 보일러와 연동하는 중앙 처리 장치 | Ch.3 |
| **구동기** | Actuator | 분배기 밸브 위에 장착해 전기 신호로 밸브를 열고 닫는 장치 | Ch.3 |
| **온도조절기** | Thermostat | 방 안의 온도를 측정하고 목표 온도를 설정하는 장치 | Ch.3 |
| **열동형 구동기** | Thermal Actuator | 내부 왁스 소자를 전기로 가열해 팽창시켜 밸브를 여는 구동기. 2선식 | Ch.3 |
| **모터형 구동기** | Motor Actuator | 소형 전동 모터로 밸브를 직접 회전시키는 구동기. 3선식 | Ch.3 |
| **HIU** | Heat Interface Unit | 지역난방 단지 열수를 세대 난방수로 변환하는 열교환기 장치 | Ch.4 |
| **에어벤트** | Air Vent | 배관 내 공기를 자동으로 배출하는 장치. 분배기 상단에 설치 | Ch.4 |
| **밸런싱 밸브** | Balancing Valve | 루프별 유량을 균등하게 조정하는 밸브. 분배기 환수 헤더에 설치 | Ch.4, 5 |
| **팽창탱크** | Expansion Tank | 난방수 온도 변화에 따른 부피 팽창을 흡수하는 탱크 | Ch.4 |
| **유량계** | Flow Meter | 각 루프의 난방수 유량을 눈으로 확인할 수 있는 계기 | Ch.4 |
| **NTC 서미스터** | NTC Thermistor | 온도가 올라가면 저항이 줄어드는 소자. 온도조절기 센서에 사용 | Ch.3 |

## C-3. 배선·전기 용어

| 용어 | 영문 | 쉬운 설명 | 관련 챕터 |
|------|------|-----------|-----------|
| **2선식** | 2-Wire | 구동기와 제어기 사이를 전선 2가닥으로 연결하는 방식 | Ch.2 |
| **3선식** | 3-Wire | 구동기와 제어기 사이를 전선 3가닥(COM·OPEN·CLOSE)으로 연결하는 방식 | Ch.2 |
| **L선** | Live Wire | 전원 공급선. AC 220V의 전류가 흐르는 선 | Ch.6 |
| **N선** | Neutral Wire | 중성선. 전류의 복귀 경로 | Ch.6 |
| **GND** | Ground | 접지선. 누전 시 전류를 안전하게 땅으로 흘려보내는 선 | Ch.6 |
| **COM** | Common | 3선식 구동기의 공통 기준선 | Ch.2, 6 |
| **OPEN** | Open | 3선식 구동기의 밸브 열기 신호선 | Ch.2, 6 |
| **CLOSE** | Close | 3선식 구동기의 밸브 닫기 신호선 | Ch.2, 6 |
| **무전압 접점** | Dry Contact | 자체 전원 없이 스위치 역할만 하는 접점 출력 방식 | Ch.3, 6 |
| **쉴드 케이블** | Shielded Cable | 외부 전자기 노이즈를 차단하는 금속 차폐층이 있는 전선 | Ch.6 |
| **노이즈** | Noise | 전자기 유도로 신호선에 유입되는 불필요한 전기 신호 | Ch.6 |
| **도통 점검** | Continuity Test | 멀티미터로 전선이 끊어지지 않고 연결되어 있는지 확인하는 검사 | Ch.6, 7 |
| **단락** | Short Circuit | 전원선과 접지 또는 다른 전선이 저항 없이 연결되는 오류 상태 | Ch.6 |
| **단선** | Open Circuit | 전선이 끊어져 전류가 흐르지 않는 상태 | Ch.6 |
| **전압강하** | Voltage Drop | 전선 길이·저항으로 인해 말단으로 갈수록 전압이 낮아지는 현상 | Ch.5 |
| **차단기** | Circuit Breaker | 과전류 발생 시 자동으로 회로를 차단하는 보호 장치 | Ch.6, 7 |
| **검전기** | Voltage Tester | 전선에 전압이 인가되어 있는지 확인하는 안전 도구 | Ch.7 |
| **멀티미터** | Multimeter | 전압·저항·전류를 측정하는 다기능 전기 계측기 | Ch.9, 10 |

## C-4. 난방·설비 용어

| 용어 | 영문 | 쉬운 설명 | 관련 챕터 |
|------|------|-----------|-----------|
| **지역난방** | District Heating | 외부 열공급소에서 단지 전체에 열을 공급하는 방식 | Ch.4 |
| **개별 보일러** | Individual Boiler | 세대마다 독립적으로 설치된 가스 보일러 | Ch.4 |
| **1차측** | Primary Side | 지역난방에서 열공급소~HIU 구간. 열공급 회사 관리 영역 | Ch.4 |
| **2차측** | Secondary Side | 지역난방에서 HIU 이후 세대 내 난방 배관 구간. 세대 관할 | Ch.4 |
| **공급수** | Supply Water | 보일러 또는 HIU에서 각 방으로 향하는 따뜻한 난방수 | Ch.4 |
| **환수** | Return Water | 각 방 바닥을 돌고 보일러 또는 HIU로 돌아오는 식은 난방수 | Ch.4 |
| **퍼징** | Purging | 배관 내에 갇힌 공기를 물로 밀어내 제거하는 작업 | Ch.4, 12 |
| **밸런싱** | Balancing | 루프별 유량을 균등하게 맞추는 조정 작업 | Ch.4, 11 |
| **스케일** | Scale | 배관 내벽에 쌓이는 석회질·탄산칼슘 침전물 | Ch.11, 12 |
| **동파** | Freeze Damage | 배관 내 물이 얼어 팽창하면서 배관이 파열되는 현상 | Ch.12 |
| **보온재** | Insulation Material | 배관의 열 손실과 동파를 막기 위해 감싸는 단열재 | Ch.12 |
| **온수 난방** | Radiant Floor Heating | 바닥 배관에 온수를 순환시켜 복사열로 난방하는 방식 | Ch.1 |
| **어댑터** | Adapter | 구동기와 분배기 밸브의 나사 규격 차이를 맞추는 연결 부품 | Ch.5, 7 |
| **M28, M30** | M28×1.5, M30×1.5 | 분배기 밸브 나사의 규격. 숫자는 나사 외경(mm) | Ch.5 |

## C-5. 진단·유지관리 용어

| 용어 | 영문 | 쉬운 설명 | 관련 챕터 |
|------|------|-----------|-----------|
| **트러블슈팅** | Troubleshooting | 고장 증상의 원인을 찾아 해결하는 진단·수리 과정 | Ch.9, 10 |
| **원인 매트릭스** | Cause Matrix | 증상과 원인의 관계를 표로 정리해 진단 우선순위를 보여주는 도구 | Ch.10 |
| **오프셋(보정값)** | Offset | 온도조절기 센서 오차를 보정하기 위해 입력하는 ±수치 | Ch.3 |
| **예방점검** | Preventive Maintenance | 고장이 나기 전에 주기적으로 점검하는 유지관리 방법 | Ch.12 |
| **수명 주기** | Life Cycle | 부품이 정상적으로 사용 가능한 기간 | Ch.12 |
| **계절 전환 점검** | Seasonal Inspection | 난방 시즌 시작·종료 시 수행하는 정기 점검 | Ch.12 |
| **인수인계** | Handover | 설치 완료 후 사용자에게 시스템 조작법·주의사항을 전달하는 절차 | Ch.8 |
| **SOP** | Standard Operating Procedure | 표준 작업 절차. 정해진 순서대로 수행하는 작업 지침 | Ch.7 |
| **시운전** | Commissioning | 설치 완료 후 시스템이 설계대로 동작하는지 검증하는 과정 | Ch.8 |
| **접촉 저항** | Contact Resistance | 단자 접속 부위의 저항값. 느슨한 단자는 접촉 저항이 높아짐 | Ch.10 |
| **절연 저항** | Insulation Resistance | 전선 심선과 외피(또는 다른 선) 사이의 저항. 낮으면 누전 위험 | Ch.12 |

## C-6. 약어 모음

```
┌──────┬──────────────────────────────────────┬──────────────────────┐
│ 약어 │ 원어                                 │ 의미                 │
├──────┼──────────────────────────────────────┼──────────────────────┤
│ HIU  │ Heat Interface Unit                  │ 열교환기 (지역난방)  │
│ SOP  │ Standard Operating Procedure         │ 표준 작업 절차       │
│ NTC  │ Negative Temperature Coefficient     │ 부특성 서미스터      │
│ PTC  │ Positive Temperature Coefficient     │ 정특성 서미스터      │
│ GND  │ Ground                               │ 접지                 │
│ COM  │ Common                               │ 공통선 (3선식)       │
│ N.C. │ Normally Closed                      │ 상시 닫힘            │
│ N.O. │ Normally Open                        │ 상시 열림            │
│ BLR  │ Boiler                               │ 보일러 연동 단자     │
│ CH   │ Central Heating                      │ 중앙 난방 (보일러)   │
│ AC   │ Alternating Current                  │ 교류 전류            │
│ DC   │ Direct Current                       │ 직류 전류            │
│ LED  │ Light Emitting Diode                 │ 발광 다이오드 (표시등)│
│ LCD  │ Liquid Crystal Display               │ 액정 표시 장치       │
│ IoT  │ Internet of Things                   │ 사물 인터넷          │
│ RF   │ Radio Frequency                      │ 무선 주파수          │
│ WD   │ Working Drawing                      │ 시공 도면            │
└──────┴──────────────────────────────────────┴──────────────────────┘
```
