# 부록 B. 장애 접수/조치 리포트 양식

> 이 양식은 인쇄하여 현장에서 직접 사용할 수 있습니다.
> 장애 발생 시 즉시 기록하고, A/S 완료 후 보관하세요.

---

## B-2. A/S 기사 현장 진단 기록표

> A/S 기사가 현장 방문 시 진단 과정과 측정값을 기록합니다.

```
┌─────────────────────────────────────────────────────────────┐
│              A/S 기사 현장 진단 기록표                       │
├──────────────┬──────────────────────────────────────────────┤
│  방문 일시   │        년    월    일     시    분            │
│  기사명      │                           소속:              │
│  접수 번호   │                                              │
└──────────────┴──────────────────────────────────────────────┘
```

### 전원·배선 계층 점검 기록

| 점검 항목 | 측정값 | 정상 기준 | 판정 |
|-----------|--------|-----------|------|
| 제어기 입력 전압 (L-N) | V | 210~230V | 정상 / 이상 |
| 차단기 상태 | ON / 트립 | ON | 정상 / 이상 |
| 접지 연결 상태 | 연결 / 미연결 | 연결 | 정상 / 이상 |
| 전선 피복 상태 | 육안 | 손상 없음 | 정상 / 이상 |
| 단자 나사 조임 | 육안 | 느슨함 없음 | 정상 / 이상 |

### 신호·통신 계층 점검 기록

| 점검 항목 | 존/위치 | 측정값 | 정상 기준 | 판정 |
|-----------|---------|--------|-----------|------|
| 온도조절기 신호선 도통 | Z | Ω | 수 Ω 이하 | 정상 / 이상 |
| 온도조절기 신호선 도통 | Z | Ω | 수 Ω 이하 | 정상 / 이상 |
| BLR 연동 출력 | - | V / 접점 | 난방 요청 시 도통 | 정상 / 이상 |
| 노이즈 확인 | - | 분리 배선 여부 | 분리 | 정상 / 이상 |

### 부품 계층 점검 기록

| 점검 항목 | 존/위치 | 측정값 | 정상 기준 | 판정 |
|-----------|---------|--------|-----------|------|
| 구동기 코일 저항 | Z | Ω | 50~500Ω | 정상 / 이상 |
| 구동기 코일 저항 | Z | Ω | 50~500Ω | 정상 / 이상 |
| 구동기 출력 전압 | Z | V | 정격 전압 | 정상 / 이상 |
| 구동기 출력 전압 | Z | V | 정격 전압 | 정상 / 이상 |
| 온도조절기 접점 | Z | Ω | ON:0Ω / OFF:∞ | 정상 / 이상 |
| 온도조절기 온도 오차 | Z | °C 차이 | ±2°C 이내 | 정상 / 이상 |

### 설비 계층 점검 기록 (해당 시)

| 점검 항목 | 측정값 | 정상 기준 | 판정 |
|-----------|--------|-----------|------|
| 분배기 공급 압력 | bar | 1.0~2.0 bar | 정상 / 이상 |
| HIU 2차측 공급 온도 | °C | 40~60°C | 정상 / 이상 |
| 루프별 유량 편차 | % | 20% 이내 | 정상 / 이상 |
| 분배기 누수 | 있음 / 없음 | 없음 | 정상 / 이상 |
