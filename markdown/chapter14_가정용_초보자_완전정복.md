# Chapter 14. 가정용 초보자 완전 정복

> **이 챕터는 각방제어에 대해 아무것도 모르는 가정용 독자를 위해 썼습니다.**
> 전기 지식 없어도 됩니다. "내 집 난방이 왜 이러지?"라는 질문에 스스로 답할 수 있게 됩니다.

## 14-1. 한눈에 보기

```
이 챕터를 다 읽으면 할 수 있는 것:

  ① 내 집 난방 구조를 스스로 파악한다
  ② 온도조절기를 제대로 이해하고 조작한다
  ③ 구동기가 정상인지 고장인지 판단한다
  ④ 문제 생겼을 때 AS 부르기 전에 스스로 진단한다
  ⑤ 난방비를 실제로 줄이는 설정을 적용한다
  ⑥ 봄·가을 계절 전환 시 관리를 스스로 한다
```

## 14-2. STEP 0 — 내 집 난방 구조 파악하기

> 이것부터 해야 나머지가 의미 있습니다.
> 10분이면 충분합니다.

### 내 집 난방 방식 알아내기

```
질문 1: 집 안에 가스보일러가 있나요?
(보통 주방 옆 벽, 베란다, 다용도실에 상자형 기계)

    YES → 개별난방 (가스보일러)
           내가 직접 보일러를 켜고 끌 수 있음

    NO  → 질문 2로 이동
              │
    질문 2: 현관이나 보일러실에 HIU(열교환기)가 있나요?
    (냉장고 절반 크기, 파이프 여러 개 연결된 기계)

    YES → 지역난방
           단지 전체가 공급하는 열을 받아씀
           보일러를 따로 켤 필요 없음

    NO  → 관리사무소에 문의
```

### 내 집 분배기 찾기

```
분배기란?
  → 보일러/HIU에서 온 뜨거운 물을 각 방으로 나눠주는 장치
  → 은색 또는 회색 파이프 여러 개가 나란히 달린 금속 박스

주로 있는 곳:
  아파트   → 거실 벽 하단 매립 (커버 열면 보임)
  빌라/단독 → 보일러 옆 또는 배관실

확인 방법:
  거실 바닥 가까운 벽면에 작은 점검구(뚜껑) 찾기
  → 손으로 열면 파이프 여러 개 보임 = 분배기

┌─────────────────────────────────────────────┐
│             5구 분배기 생김새                 │
│                                             │
│  [공급] ──┬──┬──┬──┬──┬──                  │
│           ①  ②  ③  ④  ⑤  ← 루프(방별)     │
│  [환수] ──┴──┴──┴──┴──┴──                  │
│         각 루프마다 유량계(눈금) 달려 있음    │
└─────────────────────────────────────────────┘
```

### 내 집 제어기·온도조절기·구동기 찾기

| 장치 | 생김새 | 주로 있는 곳 |
|------|--------|-------------|
| 제어기 | 담뱃갑~책 크기 박스, 여러 전선 연결 | 현관 벽, 거실 벽, 분배기 옆 |
| 온도조절기 | 각 방 벽면의 작은 패널 (에어컨 리모컨 크기) | 방마다 1개씩 |
| 구동기 | 분배기 루프마다 달린 원통형 장치 (음료캔 크기) | 분배기 위 |

### 내 집 시스템 현황표 (직접 채우기)

```
┌────────────────────────────────────────────────────┐
│  내 집 각방제어 현황                                  │
├─────────────────┬──────────────────────────────────┤
│  난방 방식       │  □ 지역난방   □ 가스보일러         │
│  분배기 위치     │                                   │
│  분배기 루프 수  │       구                          │
│  구동기 개수     │       개 (분배기에서 직접 세기)     │
│  제어기 위치     │                                   │
│  제어기 브랜드   │  (제어기 전면 스티커 확인)           │
│  온도조절기 수   │       개                          │
│  2선식 / 3선식  │  □ 2선식   □ 3선식   □ 모름        │
└─────────────────┴──────────────────────────────────┘

2선식 vs 3선식 판별법:
  분배기에 달린 구동기에서 나오는 전선이
  → 2가닥이면 2선식
  → 3가닥이면 3선식
```

## 14-3. 온도조절기 완전 이해

### 초보자가 가장 많이 하는 오해 7가지

**오해 1: "온도를 높게 올릴수록 빨리 따뜻해진다"**

```
사실:
  ON/OFF 방식 각방제어는 목표 온도에 도달할 때까지
  구동기를 열어두는 방식입니다.
  30°C로 설정해도 22°C 설정보다 '빨리' 따뜻해지지 않습니다.
  보일러 화력이 같기 때문입니다.

올바른 사용법:
  원하는 온도(예: 22°C)로 설정 → 기다리기
  높게 올리는 건 '더 빨리'가 아니라 '더 높이까지' 가는 것
```

**오해 2: "설정 온도가 됐는데 왜 구동기가 바로 안 닫히지?"**

```
히스테리시스(dead band) 때문입니다.

예시: 설정 22°C, 히스테리시스 1°C 설정 시

  20°C → 구동기 열림 (난방 시작)
  22°C → 아직 열려 있음 (히스테리시스 범위)
  23°C → 구동기 닫힘 (난방 중지)
  22°C → 아직 닫혀 있음
  21°C → 구동기 열림 (다시 난방 시작)

  → 이것이 정상입니다. 고장이 아닙니다.
  → 히스테리시스가 있어야 구동기가 오래 삽니다.
```

**오해 3: "온도조절기 숫자가 실제 온도랑 달라요"**

```
흔한 현상입니다. 원인과 해결법:

원인 1: 온도조절기가 직사광선, 창문 근처, 가전제품 옆에 있음
  → 위치 이동이 근본 해결책

원인 2: 센서 오차 (제품 특성)
  → 온도 보정(오프셋) 기능으로 조정

보정 방법 (일반적):
  온도조절기 설정 → 오프셋 또는 보정값
  → 실제 온도와 표시 온도 차이만큼 +/- 입력
  (예: 실제 21°C인데 조절기가 19°C 표시 → +2°C 보정)
```

**오해 4: "화면이 꺼졌어요 — 고장인가요?"**

```
확인 순서:

STEP 1: 화면 아무 곳이나 터치 또는 버튼 누르기
        → 켜지면 절전 모드 (정상)

STEP 2: 온도조절기 측면/하단 배터리 칸 확인
        → 배터리형이면 교체 (AA/AAA 1~2개)

STEP 3: 온도조절기 뒷면 배선 접촉 확인
        → 전선이 빠졌으면 다시 꽂기

STEP 4: 분전반 차단기 확인
        → 내려가 있으면 올리기
```

**오해 5: "외출할 때 온도조절기를 꺼야 난방비가 줄어든다"**

```
틀렸습니다.

완전히 끄면:
  집이 10°C 이하로 떨어짐
  귀가 후 22°C까지 올리는 에너지 = 종일 15°C 유지보다 더 큼
  특히 한겨울에는 동파 위험까지 있음

올바른 외출 설정:
  외출 모드 또는 설정 온도를 15°C로 낮추기
  → 집이 너무 식지 않도록 유지
  → 귀가 후 빠르게 목표 온도 도달
  → 종합 에너지 비용 절감
```

**오해 6: "방마다 온도를 전부 같게 맞춰야 한다"**

```
그럴 필요 없습니다.

방별 추천 설정:
  거실 (활동 공간)    → 21~22°C
  안방 (취침)         → 18~20°C (수면 중 낮은 온도가 숙면에 좋음)
  아이방              → 20~21°C
  서재 (사용 시간만)  → 20°C (있을 때만 켜기)
  빈 방 / 창고        → 10~12°C (동파 방지만)

→ 방별로 다르게 설정하는 것이
  각방제어를 제대로 쓰는 방법입니다.
```

**오해 7: "구동기 소리가 나면 고장이다"**

```
정상 소리 vs 비정상 소리:

정상:
  · 칙칙 (2선식 구동기 작동음) → 정상
  · 딸깍 (3선식 완전 열림/닫힘) → 정상
  · 윙~ (3선식 모터 구동) → 정상

비정상:
  · 계속 끊임없이 칙칙칙칙 반복 → 히스테리시스 너무 작음
  · 덜덜덜 진동 → 구동기 고정 나사 풀림
  · 끼이익 소리 → 구동기 내부 기어 마모 → 교체 검토
```

## 14-4. 분배기 유량계 — 난방이 되는지 눈으로 확인하는 법

> 분배기 유량계를 볼 줄 알면 "고장인지 아닌지"를 5초 만에 판단합니다.

```
분배기 유량계 생김새:

  ┌──────────────────────────────────────────────────┐
  │  분배기 (공급측)                                   │
  │                                                   │
  │  루프① [유량계] ← 눈금이 올라가면 물 흐르는 중     │
  │  루프② [유량계]                                   │
  │  루프③ [유량계] ← 눈금이 바닥이면 밸브 닫힌 것     │
  │  루프④ [유량계]                                   │
  │  루프⑤ [유량계]                                   │
  └──────────────────────────────────────────────────┘

유량계 보는 법:
  눈금 올라감 → 그 루프(방)에 물이 흐르고 있음 → 난방 중
  눈금 바닥   → 물이 안 흐름 → 구동기 닫혀 있음 or 고장
```

### 유량계 활용 진단표

| 상황 | 유량계 | 판단 |
|------|--------|------|
| 온도 올렸는데 방이 안 따뜻해짐 | 눈금 올라감 | 배관·보일러 쪽 문제 |
| 온도 올렸는데 방이 안 따뜻해짐 | 눈금 바닥 | 구동기 안 열린 것 → 구동기/제어기 확인 |
| 온도 안 올렸는데 방이 뜨거움 | 눈금 올라감 | 구동기가 안 닫힌 것 → 구동기 고장 의심 |
| 특정 방만 유량 없음 | 해당 루프만 바닥 | 해당 구동기 단독 문제 |

## 14-5. "지금 정상인가?" — 스스로 진단하는 흐름도

### 증상별 5분 진단

**증상 A: 방이 전혀 안 따뜻해요**

```
STEP 1. 온도조절기 설정 확인
  → 설정 온도 > 현재 온도인가?
  → NO: 설정 온도 올리기 → 해결

STEP 2. 분배기 유량계 확인
  → 해당 루프 눈금 올라가나?
  → YES: 보일러/HIU 가동 여부 확인
  → NO: 다음 단계

STEP 3. 구동기 작동 확인
  → 구동기에 손 대보기 (따뜻하면 2선식 작동 중)
  → 소리 들리나? (칙칙 or 윙)
  → 소리 없음: 구동기 전선 접촉 확인

STEP 4. 제어기 LED 확인
  → 꺼져 있음: 전원 확인 (차단기, 콘센트)
  → 켜져 있음: AS 접수
```

**증상 B: 특정 방만 안 따뜻해요**

```
STEP 1. 그 방 온도조절기 설정 확인
  → 설정이 낮으면 올리기

STEP 2. 분배기에서 그 방 루프 유량계 확인
  → 눈금 없음 → 그 방 구동기 확인

STEP 3. 구동기 전선 확인
  → 커넥터 빠졌는지 육안 확인 → 다시 꽂기

STEP 4. 구동기 교체 시도 (자가 가능)
  → 동일 규격 구동기로 교체 (2~3만원대)
```

**증상 C: 방이 너무 뜨거워요 (끄고 싶은데 안 꺼짐)**

```
STEP 1. 온도조절기 설정 온도 낮추기
  → 현재 온도보다 낮게 설정 → 구동기가 닫혀야 함

STEP 2. 그래도 계속 뜨거우면
  → 분배기에서 그 방 루프 유량계 확인
  → 눈금 있으면: 구동기가 안 닫히는 것 → 구동기 고장

STEP 3. 임시 조치
  → 분배기에서 해당 루프 수동 밸브 잠금
     (밸브 손잡이 시계 방향으로 돌리면 잠김)
  → 이후 AS 접수
```

**증상 D: 온도조절기에 E 오류가 떠요**

```
브랜드마다 다르지만 가장 흔한 패턴:

E1 또는 Er1 → 온도 센서 이상
  (온도조절기 센서 불량 or 단선)
  → 온도조절기 교체

E2~E5        → 통신 오류 (제어기-온도조절기 연결 문제)
  → 전원 껐다가 켜보기 (재부팅)
  → 해결 안 되면 배선 접촉 확인

E8 또는 E9   → 전원 이상
  → 전원 확인, 차단기 확인

오류코드 정확한 의미:
  → 온도조절기 뒷면 또는 제어기 내부 스티커 확인
  → 제조사 고객센터에 모델명 + 오류코드 문의
```

## 14-6. 난방비 실제로 줄이는 설정법

### 효과 순위별 설정 3가지

**1위 — 빈 방 온도 낮추기 (효과 가장 큼)**

```
낮에 아무도 없는 방:
  기존: 22°C 유지
  변경: 10~12°C (동파 방지 온도)

효과:
  그 방 구동기가 거의 닫혀 있음
  → 보일러가 그 방에 열 안 보냄
  → 빈 방이 2개면 난방 에너지 최대 40% 절감 가능
```

**2위 — 외출·취침 모드 적극 활용**

```
상황별 권장 설정:

┌─────────────────────────────────────────────────┐
│  상황           │  설정 온도        │  이유        │
├─────────────────┼──────────────────┼─────────────┤
│  집에 있을 때   │  21~22°C         │  활동 적정   │
│  취침 중        │  18~19°C         │  숙면 적정   │
│  낮 외출 (수시간)│  17~18°C        │  너무 식으면 복구 에너지 더 큼 │
│  장시간 외출    │  15°C            │  최적 균형점 │
│  2일 이상 부재  │  10~12°C        │  동파 방지   │
└─────────────────┴──────────────────┴─────────────┘
```

**3위 — 스케줄(예약) 기능 설정**

```
맞벌이 가정 추천 스케줄:

  06:00  안방·아이방 21°C 가동 (기상 준비)
  08:30  전체 15°C (출근·등교 후)
  17:00  거실·아이방 22°C 예약 가동 (귀가 대비)
  22:30  안방 18°C, 나머지 OFF (취침)

이것만 설정해도 난방비 20~30% 절감 체감 가능
```

### 히스테리시스 설정과 난방비 관계

```
히스테리시스가 너무 작으면:
  구동기가 자주 켜졌다 꺼짐 → 에너지 낭비 + 구동기 수명 단축

히스테리시스가 너무 크면:
  방 온도 편차가 커짐 → 불쾌함

권장값: 1.0~1.5°C

설정 위치:
  제어기 설정 메뉴 → "히스테리시스" 또는 "온도편차" 항목
  (브랜드마다 메뉴명 다름)
```

## 14-7. 계절 전환 관리 — 봄·가을에 꼭 해야 할 것

### 난방 시즌 시작 전 (10월 첫째 주 권장)

```
□ 1. 분배기 점검구 열어서 누수 여부 육안 확인
     → 파이프 연결부에 물기 있으면 시공 업체 연락

□ 2. 분배기 수동 밸브 '열림' 상태 확인
     → 여름 내내 잠가둔 경우 → 반시계 방향으로 열기

□ 3. 온도조절기 배터리 교체
     → 배터리형은 매년 교체 권장 (AA 1~2개)
     → 교체 시기: 난방 시작 전이 가장 좋음

□ 4. 시험 가동 1회
     → 모든 방 온도조절기를 현재 온도보다 5°C 높게 설정
     → 구동기 작동음 확인 (칙칙 or 윙)
     → 유량계에 흐름 확인
     → 보일러 가동 확인
     → 10분 후 분배기 누수 재확인

□ 5. 에러 없이 정상 가동 확인 후 원하는 온도로 재설정
```

### 난방 시즌 종료 후 (4월 권장)

```
□ 1. 온도조절기 끄지 말고 최저 온도로 설정
     → 완전 차단 금지 (기기 설정 초기화, 재시작 오류 원인)
     → 10°C 설정으로 유지 (전기료 거의 없음)

□ 2. 월 1회 '구동기 작동 유지' 관리
     → 여름 내내 구동기가 닫혀만 있으면 왁스 굳어서 고착

     방법:
       한 달에 한 번, 각 방 온도조절기를 30°C로 올림
       → 구동기 열림 동작 (5~10분)
       → 다시 10°C로 낮춤 → 구동기 닫힘
       → 이것만으로 고착 예방

□ 3. 분배기 누수 여부 월 1회 확인
     → 바닥에 물기 없는지 육안 점검

□ 4. 장기 외출·여행 시
     → 온도조절기 10~12°C 유지
     → 분배기 수동 밸브 열림 상태 유지 (닫으면 동파 위험)
```

### 구동기 고착이란?

```
고착이란:
  여름 내내 닫혀 있던 2선식(열동형) 구동기의 왁스가
  굳어버려서 난방 시즌에 다시 열리지 않는 현상

증상:
  가을에 난방 켰는데 그 방만 차가움
  분배기 유량계에 해당 루프 흐름 없음
  구동기에 전기 공급됨에도 열리지 않음

예방:
  여름에 월 1회 구동기 작동 (위 방법 참조)

발생 시:
  구동기 교체 (자가 가능, 2~3만원)
```

## 14-9. 실수 TOP 5 — 초보자가 가장 많이 하는 실수

| 순위 | 실수 | 결과 | 올바른 행동 |
|:---:|------|------|------------|
| 1 | 온도를 30°C로 올리면 빨리 따뜻해진다고 생각 | 기대한 효과 없음 + 과열 | 원하는 온도로만 설정 |
| 2 | 외출 시 온도조절기 완전히 끄기 | 귀가 후 더 오래 걸림, 동파 위험 | 15°C 유지 |
| 3 | 구동기 소리에 놀라 AS 전화 | 불필요한 출장비 | 정상 소리 목록 숙지 |
| 4 | 여름에 분배기 수동 밸브 잠금 | 가을 고착 발생 | 밸브는 항상 열어두기 |
| 5 | 브랜드 다른 온도조절기 구매 교체 | 통신 불가, 작동 안 됨 | 동일 브랜드 제품으로만 교체 |
